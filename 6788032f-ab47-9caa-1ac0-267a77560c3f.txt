　　魔族之中，一样有着无数争斗，并非是铁板一片。
<br/>　　不过在魔族之中，一向是以实力为尊，比起武者之间，要更加纯粹。
<br/>　　这蝎狮王，敢来讥讽楚风眠，也是因为他的实力，虽然不如龙摩罗，可也相差不远。
<br/>　　如果是一个弱者的话，楚风眠根本不用开口，一巴掌直接拍死便是。
<br/>　　“蝎狮王，不想死你就闭嘴，我看这佛寺也还没开启，不然你出来，跟我一战？”
<br/>　　楚风眠盯着蝎狮王，毫不客气的说道，这龙摩罗，在魔族之中，也就对少帝客气几分。
<br/>　　至于其他人，向来不给任何的脸面。
<br/>　　楚风眠在杀了龙摩罗之后，便是将他的记忆也全部得到，开口的气势，跟真正的龙摩罗一样，绰绰逼人。
<br/>　　蝎狮王听到楚风眠的话，脸色显现出无比冰冷的神情，他的眼神阴狠无比，却是不敢在开口。
<br/>　　他的实力，毕竟不如龙摩罗，真正一战，吃亏的也是他。
<br/>　　就在这场面僵持的时候，又是一道身影，从魔意之中走了出来。
<br/>　　这一次走出的，却是一名女子。
<br/>　　“你们两个，不要在这里争吵了，你们有着力气争吵，不如留着力气，等待这佛寺开启之后，去对付里面的那些东西。”
<br/>　　这一名女子，看起来是跟寻常的武者一样，但是她的手臂之上，却是有着一层复杂的花纹。
<br/>　　这花纹，并非是后天印上去的，而是跟这女子的身躯为一体，乃是本来就属于她的东西。
<br/>　　“魔纹，这是魔族之中的王族！”
<br/>　　楚风眠看到这花纹的一刻，也是明白了过来。
<br/>　　同时在楚风眠的脑海之中，急忙搜寻着龙摩罗的记忆，很快他就明白了眼前这女子的身份。
<br/>　　噩梦王女，梦烟罗。
<br/>　　魔族之中，曾经一共有着十大王族，便是曾经魔族之祖，十方天魔所留下的血脉后代。
<br/>　　不过眼下，真正古老的魔族凋零，十大王族，便是只剩其三了。
<br/>　　眼前这女子，便是十方天魔之一，噩梦天魔的后人，如今噩梦一族的王女，梦烟罗。
<br/>　　梦烟罗继承了十方天魔的血脉，虽然不纯粹，但是论起血脉之力，这梦烟罗只怕是比起龙摩罗，还要更加的可怕。
<br/>　　虽然这梦烟罗还未出手过，但是楚风眠也足以感受到，此女的身上，有着深不可测的力量。
<br/>　　曾经的龙摩罗，虽然狂傲，可也不敢于这梦烟罗轻易为敌。
<br/>　　“噩梦王女。”
<br/>　　楚风眠看到梦烟罗的到来，点了点头，身上的气息也是收敛了几分，算是给了她一个面子。
<br/>　　看到楚风眠收敛起气息来，梦烟罗又看向了蝎狮王，开口道。
<br/>　　“蝎狮王，这一次龙摩罗来晚了，也一定是因为一些事耽搁了，在这藏地之中，不比外面，遇到些棘手的麻烦，来晚一些，不算什么。”
<br/>　　梦烟罗的到来，显然就是为了阻止楚风眠跟蝎狮王的争斗。
<br/>　　“王女说的是，本王鲁莽了。”
<br/>　　蝎狮王也是借坡下驴，开口道。
<br/>　　“好，这一次佛寺开启，事关重大，我们先过去，路上在跟你们说清楚。”
<br/>　　梦烟罗一边开口，一边化为了一道遁光，直接向着群山之巅飞去。
<br/>　　蝎狮王跟楚风眠，也是紧随其后，紧随梦烟罗前去。
<br/>　　这群山看似不大，但是想要飞到尽头，也需要十分钟左右的时间。
<br/>　　这群山依然是在藏地之中，受到藏地佛门禁制的压制，遁光的速度都慢了不少。
<br/>　　不过楚风眠已经也是习惯了，暗自跟随者梦烟罗，同时观察着周围的一举一动。
<br/>　　他现在算是蒙混进来了，看来梦烟罗跟蝎狮王的举动，便是没有看出楚风眠的异样来。
<br/>　　已经是混入其中了，那下一步，楚风眠的目的，便是要想办法得到那一枚人皇罐碎片了。
<br/>　　楚风眠的目光扫视，最终还是落到了净土佛寺之中。
<br/>　　在这群山之中，都没有人皇罐碎片的气息，那这人皇罐碎片，唯一的可能，就只有在这净土佛寺之中了。
<br/>　　这对楚风眠来说，倒是有些棘手，他本来是想要先趁着少帝不在，先将那一枚人皇罐碎片拿到手的。
<br/>　　如此一来，哪怕是少帝识破了楚风眠的身份，楚风眠也可以全力逃走，毕竟目的已经是得到了。
<br/>　　可现在，那人皇罐碎片就在净土佛寺之中，楚风眠只能够留下来，等待这净土佛寺开启了。
<br/>　　“龙摩罗，蝎狮王，你们二人也知道，这净土佛祖，其实就是我们魔族最伟大的魔帝，血邪魔帝吧。”
<br/>　　就在楚风眠思考之际，梦烟罗突然回过头来，对着楚风眠跟蝎狮王开口道。
<br/>　　楚风眠跟蝎狮王，都是点了点头，这消息，在魔族之中知晓的人，虽然不多，但是龙摩罗，蝎狮王，都是魔族之中的高层人物了。
<br/>　　这个消息，他们早就知晓。
<br/>　　“这一次少帝发现了这净土佛寺，所以才发出消息，让你们赶来，这净土佛寺之中，传闻有着血邪魔帝的宝藏，甚至是有着血邪魔帝的传承。”

<br/>　　“净土佛寺之中，有着不少危险的麻烦，所以这一次少帝才让你们前来相助，不过少帝也不会亏待你们，只要是攻入到佛寺之中，其中的宝藏，你们可以得到的，都归你们。”

<br/>　　梦烟罗听到楚风眠的话，也并不惊讶，显然是知晓龙摩罗的性格，只是眼神之中，露出几分冰冷道。
<br/>　　“当然，龙摩罗，你若是有能耐，可以跟少帝去争，那也大可以试试。”
<br/>　　“龙摩罗，你别倒时候跟少帝争斗，死在少帝的手中啊。”
<br/>　　蝎狮王在一旁也是出言讥讽道。
<br/>　　“闭嘴。”
<br/>　　楚风眠冷哼一声道。

<br/>　　被楚风眠接连呵斥，蝎狮王眼神也越发的阴冷，不过却没有在这个时候，在跟楚风眠争斗，而是对着梦烟罗问道。
<br/>　　“在这净土佛寺之中，有什么危险，就连少帝都不好对付，要让我们出手？”
