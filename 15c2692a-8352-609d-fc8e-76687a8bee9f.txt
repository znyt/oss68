　　第135章
<br/>　　落叶城。
<br/>　　天色渐晚，街道上已无多少行人。程大雷和徐神机走在街上，二人都没有带兵器，程大雷的斧头已经让秦蛮带回去。
<br/>　　“大当家，你为什么一定要带着我？”徐神机。
<br/>　　“哎，还不是因为军师足智多谋，神机妙算，说实话我没了你可不行。”程大雷白了徐神机一眼：“另外，叫我公子，欧巴欧公子。”
<br/>　　“那好吧，公子，你去忙你的事，我也去忙我的事。”
<br/>　　“你有什么事情要忙？”
<br/>　　“去喝花酒呐。”徐神机理直气壮道：“你去找你的姑娘，我去喝我的花酒。”
<br/>　　“算了，你去吧。”程大雷摆摆手：“别误了正事，明天我们在酒楼见面。”
<br/>　　徐神机麻溜快的跑了，看他的速度，真是让千里马也汗颜。
<br/>　　看着他的背影，程大雷从怀里掏出一个钱袋，默默道：“没有钱你喝什么花酒？”
<br/>　　和徐神机分开后，程大雷便去寻苏樱，这时候天色差不多已经黑了，苏樱当在她自己的宅子内。
<br/>　　程大雷到的时候，还没靠近大门，便听见里面响起嘤嘤哭声。走在大门口推了推，门在里面锁着，程大雷直接翻墙进去了。
<br/>　　成为山贼后，翻墙方面的天赋似乎提高不少，如今让程大雷走门都有些不习惯。
<br/>　　“什么人！”
<br/>　　双脚刚刚落地，小蝶便拎着一柄菜刀出来，眼睛此刻红彤彤的。
<br/>　　“原来是你，有门不走，你翻什么墙。”小蝶说话的声音带着哭腔。
<br/>　　“你哭什么？”程大雷奇怪道：“你家小姐呢？”
<br/>　　“老爷死了。”
<br/>　　苏樱的父亲久病卧床，身子一天不如一天，苏樱信中也提到过这件事。看来他终究没熬过这个冬天。
<br/>　　“你家小姐是回家了么？”
<br/>　　“小姐被他们抓走了。”
<br/>　　“抓！”程大雷一愣。
<br/>　　“老爷是在三天前去世的，小姐想去祭拜，几次都被赶了出来。他们看上了湖底捞的生意，说除非把酒楼还回来，否则就不让小姐给老爷送终。今天下午苏家人把小姐叫走说是商量这件事，到现在小姐都没回来。”
<br/>　　这件事依程大雷的意思，不去送就不去呗，反正苏四海生前对苏樱也谈不上有多好。不过，那毕竟是苏樱的父亲，再加上这个时代的人愚忠愚孝。所以这件事对苏樱来说，可能还真是一道过不去的坎儿。
<br/>　　“走啦。我们去苏家看看。”程大雷道。
<br/>　　“你去，可你的身份，被人抓起来怎么办？”小蝶。
<br/>　　“走啦，走啦。”
<br/>　　二人快步来到苏府，程大雷立在台阶上下等候，小蝶去叩门。
<br/>　　连喊几声，才有人挑着灯笼将门打开：“吵什么吵什么，叫丧么！”
<br/>　　“苏忠，我来找小姐。”小蝶认得这个门房。
<br/>　　“找什么小姐，她天没黑就走了，我可没见到她。”
<br/>　　“不可能！我一直等着小姐，她根本没回来。”
<br/>　　“那我就不知道了，说不定她会去找什么野男人，反正她是土匪的女人，这种事一点也不奇怪。”
<br/>　　“苏忠，你嘴巴放干净些！”小蝶气得脸色苍白，指着苏忠鼻子道：“我不信，小姐一定被你们关起来了，让我进去找找。”
<br/>　　“滚开！”苏忠一把将小蝶推开：“没管教的东西，你当苏家是什么地方，是你想进就能进的么。”
<br/>　　苏忠忽然将门拉开一道缝：“我就算将门打开，你今天敢进来么！”
<br/>　　看着黑洞一般的门内，小蝶没敢说话。
<br/>　　苏忠冷笑一声，砰地一声将门重重的关上。
<br/>　　小蝶来到程大雷身边：“小姐失踪了。”
<br/>　　“我刚才已经听见了。”
<br/>　　“小姐一定是被他们关起来了，一定还在苏家。”小蝶大叫道：“对了，苏家有一个地窖，有时候一些犯错的下人会被关进里面。小姐一定也被他们扣在里面。”
<br/>　　“你能确定是苏家干的？”
<br/>　　“小姐在城里没有仇人，不是他们干的还能是谁！你不是会翻墙么，咱们翻墙进去，找到小姐看他们还有什么话说。”
<br/>　　“翻什么墙，咱们走大门。”
<br/>　　“可你的身份，如果惊动了官府怎么办？”
<br/>　　程大雷毕竟是山贼，官匪不能照面，可程大雷此刻似乎没有考虑这一点，直接将小蝶带到门口。
<br/>　　……
<br/>　　苏家布着灵堂，苏四海几个妾室以及女儿穿着麻衣，在堂前守灵。
<br/>　　但苏四海的正房夫人李如并不在此间，灵堂后面一个院子，李如靠在罗汉床上，身穿孝衣，脸上带着疲惫神色。
<br/>　　李大头坐在椅子上，表情有些凝重：“小妹，咱们把苏樱扣下来，这件事真的没关系么？”
<br/>　　“有什么关系，先在地窖内扣她几年，等风声过了，将她许配给一个下人，女人么，生了孩子也就认命了。”
<br/>　　“可这样是不是太狠了些？”
<br/>　　“狠！”李如忽然咬紧贝齿：“我十八岁就跟着苏四海，可这老混蛋一死，把家产全部留给苏樱，一分一毫都没给我，我在他眼里，永远是个妾，妾，就是奴！如果不是这老混蛋这么绝情，我何必赶尽杀绝。”
<br/>　　那温婉的脸上，浮现残酷的厉色，她看了李大头一眼，轻轻摇摇头。
<br/>　　“大哥，咱们都是苦出身，爹娘十三岁就把我卖人，我吃过的苦又怎么能对你讲。活到现在，我们早该明白，这世间的东西，你不争就不会有人给你。”
<br/>　　“可苏樱毕竟和山贼有些瓜葛，万一他们报复起来，咱们……”
<br/>　　“谣言而已。”李如无所谓的挥挥手：“苏樱即便跟过山贼，怕已被玩腻了。不然，早就把她抢回山寨了，难道世上还有讲温良恭俭让的山贼，呵呵。”
<br/>　　“好啦，你先下去吧，记得这几日帮我留心，招揽几个身手好的江湖人。有些事情，我一个女人做着毕竟不方便。”
<br/>　　“我已经在忙这件事，大概已经有些线索了。”
<br/>　　李大头道了一声，起身准备离去。正在此时，忽听得外面一声大叫，乱哄哄闹成一片。
<br/>　　“山贼打进来了！”
<br/>　　“山贼打进来了！”
<br/>　　这声音传啊传，从前院传到后院，一直传进李如和李大头耳中。
<br/>(本章完)
