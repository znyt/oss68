<br/>  青二娘被王康委派到赵国京都，布置情报网络，在山洪结束之后，离开新奉县，如今过去几日，应该已有成效……<br/>　　<br/>　　  王康坐在桌前，看着这封密信，这个消息，确实是跟他有关，<br/>　　<br/>　　  根源还是他曾在封地之争，与柳诚文笔策论，所作的那篇，教战守策！<br/>　　<br/>　　  在比试过后，监察使李济同，锦衣卫指挥使宇文奈回京赴命，连同这篇策论，也呈给赵皇。<br/>　　<br/>　　  教战守策，全文论述赵国情势，提出全民备战……<br/>　　<br/>　　  这确实是一篇名策，王康相信以赵皇的眼界定然能够看出。<br/>　　<br/>　　  只是，想要实行却很难。<br/>　　<br/>　　  全民备战可不是一句空话，简单说就是要大规模征兵，以此来激发民众进取之心，<br/>　　<br/>　　  并用此种方式，来遏制老牌贵族。<br/>　　<br/>　　  一方面能够解决赵国现今困局，一方面又能提升保皇派势力。<br/>　　<br/>　　  所以说，这篇策论是完全符合赵皇现今所求。<br/>　　<br/>　　  正好用于他实施新政。<br/>　　<br/>　　  但其本质，又涉及到了保皇派与保守派之争，所以若要推行，必然会有极大的阻力。<br/>　　<br/>　　  王康预想，赵皇想要实施此策，也该等合适时机，但没想到现在就已经提出。<br/>　　<br/>　　  据青二娘密信中所写，现在整个京都，都在议论此事，就连朝堂中，也是争吵一片。<br/>　　<br/>　　  赵皇的决心很大，但同样阻力也是不小……<br/>　　<br/>　　  而也因此，这篇教战守策公布出来，连带做这篇策论的王康，也走在人前，站在风口浪尖。<br/>　　<br/>　　  谁都知道，富阳伯王鼎昌，是赵皇提拔而起的新贵，王康作为富阳伯之子，这意味着什么，不言而喻……<br/>　　<br/>　　  可想而知，王康又一次的成为老牌贵族中的眼中钉，肉中刺！<br/>　　<br/>　　  这是王康所没有预料到的。<br/>　　<br/>　　  他摇了摇头，将信件烧毁，也没有在意。<br/>　　<br/>　　  反正他现在是虱子多了不怕咬，京都的事，暂时还与他无关，现在最重要的是发展新奉县。<br/>　　<br/>　　  只是令他奇怪的是，至从上次大战以后，永定伯爵府却没有丝毫动静……<br/>　　<br/>　　  这一点，可是大出他的预料，韩瑜应该不像是这么能忍的人，或许在筹备更大的阴谋，也说不定。<br/>　　<br/>　　  既然想不通那便不想了，兵来将挡，水来土掩，这是他一直所信奉的。<br/>　　<br/>　　  将手头的事情处理完，王康便准备前往新兵营。<br/>　　<br/>　　  在那日训话之后，所有人都安定下来，而由王康安排的特训，又开始了……<br/>　　<br/>　　  队列，体能，战术等依次展开。<br/>　　<br/>　　  标准完全是按照，特种兵来的。<br/>　　<br/>　　  而且经那日的厮杀战斗后，他们的成长也是很快。<br/>　　<br/>　　  在王康来到这里时，所有人正在整齐练着一套拳法。<br/>　　<br/>　　  而教授他们的是……李清曼！<br/>　　<br/>　　  这是王康的特地安排，李清曼出自太一教，所会的武道绝学繁多，他就想着，能否教新兵些招式什么的。<br/>　　<br/>　　  所以才有了这一幕。<br/>　　<br/>　　  据李清曼所讲，这套拳法是一门绝学的入门拳法，勤加练习，可提升气力，与敌对战，都有功效。<br/>　　<br/>　　  李清曼身穿劲装，演练着。<br/>　　<br/>　　  这套拳法大开大合，充满阳刚之气，明显是适学于男子。<br/>　　<br/>　　  李清曼竟也将之打练而出，气势非凡。<br/>　　<br/>　　  其下兵卒则是在认真跟着学习。<br/>　　<br/>　　  结束之后，李清曼来到王康身边道：“你招的这些人中，有几个好苗子，悟性挺高。”<br/>　　<br/>　　  她说着又道：“你不是一直想要修习武道，也可以跟着练习，起码能提升些气力。”<br/>　　<br/>　　  王康摇了摇头，他可是没这份毅力……<br/>　　<br/>　　  一天的训练结束，并不代表着完结，晚上还有王康给安排的专门授课。<br/>　　<br/>　　  文化培训！<br/>　　<br/>　　  据他的经验，一支能争擅战的军队，必然是有着文化的队伍……<br/>　　<br/>　　  然而，这种培训，却比正常的体能，战术训练，还难进行！<br/>　　<br/>　　  因为，这其中大多人都不识字。<br/>　　<br/>　　  这也不是个人的问题，而是所处环境决定的。<br/>　　<br/>　　  不止是这些招募之兵，在新奉县大多数人都不识字。<br/>　　<br/>　　  孩童也没有一个所学的地方。<br/>　　<br/>　　  王康早就有要办学的念头，但现在的条件还不太成熟。<br/>　　<br/>　　  事情太多，总得一项项的来。<br/>　　<br/>　　  但现在他发现，这个事情，还必须得抓紧了……因此，他决定要在新奉县掀起一场扫盲运动。<br/>　　<br/>　　  回到县衙，晚饭之后，他并没有休息，而是钻进了书房，林语嫣也跟着进来问道：“怎么？你有事情处理。”<br/>　　<br/>　　  “写点东西，”<br/>　　<br/>　　  王康准备着纸笔随意道：“准确的说是编纂教材！”<br/>　　<br/>　　  “教材？什么教材？”林语嫣好奇的问道。<br/>　　<br/>　　  “启蒙教材，”<br/>　　<br/>　　  王康说了一句，便开始执笔写了起来。<br/>　　<br/>　　  林语嫣好奇的来到王康身后看着他所写，还顺带念了出来。<br/>　　<br/>　　  “人之初，性本善，性相近，习相远，苟不教……”<br/>　　<br/>　　  这？<br/>　　<br/>　　  林语嫣读着，眼中的震惊，也是越浓郁，只是一阅，便沉迷其中。<br/>　　<br/>　　  格式上，三字一句朗朗上口，又所含韵律，通俗，顺口。<br/>　　<br/>　　  而且所写也极有内涵。<br/>　　<br/>　　  人出生之初，禀性本身都是善良的，天性也都相差不多，只是后天所处的环境不同和所受教育不同，彼此的习性才形成了巨大的差别……<br/>　　<br/>　　  简单的几字，道尽人生格理。<br/>　　<br/>　　  不止是这些，还有一些日常生活常识。<br/>　　<br/>　　  如现在写到的几句。<br/>　　<br/>　　  三才者，天地人，三光者，日月星，三纲者，君臣义，<br/>　　<br/>　　  父子亲，夫妇顺……<br/>　　<br/>　　  林语嫣一字一字诵念，愈发觉得妙用无比……<br/>　　<br/>　　  王康也不理会，自顾自的写着，有的生僻字，还做了标注，还有的则是做了释义。<br/>　　<br/>　　  时间一点点过去，他很快写完，王康放下了笔，看着整齐的字体，满意的点头。<br/>　　<br/>　　  “这通篇叫什么？”林语嫣疑惑的问道。<br/>　　<br/>　　  “你不说还忘了，”王康又拿起了笔，在首处写下了三个字。<br/>　　<br/>　　  “三字经？”<br/>　　<br/>　　  林语嫣呢喃着。<br/>　　<br/>　　  “对，就是三字经！”<br/>　　<br/>　　  王康笑着道：“这就是我编写的启蒙教材，新奉县之前贫穷，人们普遍认字不多，这可不行啊！”<br/>　　<br/>　　  三字经是传统的启蒙教材，是最浅显易懂的读本之一，用到现在最合适宜。<br/>　　<br/>　　  当然，王康所写只是部分，因为三字经中，还包含一些古代典籍，而这个时代偏差，有些没有的，他就取缔。<br/>　　<br/>　　  只留下其中的适宜部分。<br/>　　<br/>　　  就由它，作为扫盲运动的开始，在新奉县挂起一阵学习之风……<br/>　　<br/>　　  【作者题外话】：第一更。<br/><br/><br/>
