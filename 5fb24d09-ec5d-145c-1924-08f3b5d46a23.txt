　　　　“听我命令！”
<br/>　　罗昊将手中G36C快慢机拨到半自动模式，随时准备向街道上那几个整靠着皮卡车抽烟的反政府武装人员发起攻击。
<br/>　　这时，外面抽烟的反政府武装人员中突然传来电话铃声。
<br/>　　罗昊透过缺少了玻璃的窗户往外看去，看到最开始那个拿出烟，并且分发给其他武装人员的反政府武装人员从战术背心中拿出一部卫星电话，掰起折叠起来的天线，接听起电话来。
<br/>　　因为对方说得是当地的语言，所以罗昊并不能听懂他们之间到底说了些什么，只是看到那个反政府武装人员在接听完电话后，就把手中的烟头仍在地上，踩灭，踏上皮卡车，扬尘而去。
<br/>　　藏身在街道两边建筑中的罗昊、李然、慕容飞、李毅峰四个人走上街道。
<br/>　　“真是便宜他们了，让他们捡了一条命。”慕容飞看了眼那辆远去的皮卡车，看向地上的烟头，说道：“嚯，没想到居然还是万宝路，这倒是让我有些意外。”
<br/>　　“没什么意外的。”李然说道：“中东这个地方现在鱼龙混杂，雇佣兵、政府军、反政府武装、军火商、还有北约的特种部队，有太多的势力在这里活动，而且他们之间敌我不明。现在的中东比起火热又混乱的非洲大陆，有过之而无不及。走吧哥几个，我们还有很长一段路要走，而且越是深入，越容易跟活动在这附近的反政府武装遭遇。”
<br/>　　“这种在敌人眼皮子底下神不知鬼不觉的进进出出才刺激。”慕容飞咧嘴一笑，说道。
<br/>　　“实战不是游戏，挨了子弹是要死人的，不是从存储点重新读取就能重来一遍。”
<br/>　　李毅峰瞥了眼慕容飞，说道：“看把你兴奋的跟个孩子似得。”
<br/>　　“你就拉倒吧。”慕容飞不屑地撇嘴，说道：“你这些话还是留着回去跟后面来参训的菜鸟说吧，我可不是菜鸟。”
<br/>　　“有干劲是好事。”罗昊说道：“但是我们后面必须要更加警惕，后面遇到反政府武装巡逻队的频率会越来越高，稍有不慎，我们就会被包围。”
<br/>　　四个人放弃了在街道上快速前行，而是在建筑与建筑之间的小路中穿插着前行，这样一来，他们实际行走的距离要比卫星地图上所标注出来的直线距离更远，但是却可以有效的避免跟反政府武装的巡逻队接触。
<br/>　　黄昏十分，罗昊、李毅峰、慕容飞、李然四个人距离被反政府武装势力所控制的区域还有不到一公里的距离。
<br/>　　罗昊看了眼自己的手腕上手表的时间，尽管已经是晚上七点，可是天色却还没有彻底暗下来，如咸蛋黄一样的夕阳还挂在天空中，为这片被战火所燃烧过的大地镀上了一层金甲。
<br/>　　“现在这个季节，这里完全天黑差不多还需要一个小时，我们暂时先在这附近休整，等到天黑以后再进入敌占区。”
<br/>　　李然看了眼天空，向罗昊提出建议。
<br/>　　罗昊点了点头，说道：“所有人原地休息，注意警戒。”
<br/>　　四个人找到一处铁门紧闭的建筑废墟，可是刚靠近就闻到有一股恶臭从屋子中传来，而且还能看到有好几只体型硕大的绿头苍蝇从屋子中飞出来，发出烦人的“嗡嗡”声。
<br/>　　“哥几个，这间屋子里似乎有一些我们并不愿意看到的东西。”罗昊对身边其他三个人说道。
<br/>　　虽然还没有开门，但仅仅只是闻到这股恶臭味，罗昊就能猜到这间铁门紧闭的屋子里有什么东西，除了人的尸体，不会还有什么东西能在腐烂后有这么浓烈的恶臭味。而且罗昊还可以肯定，被丢弃在这间屋子里的尸体绝对不会是反政府武装势力的人。
<br/>　　慕容飞撇了撇嘴，说道：“我说哥几个，还是换个地方吧。我可不愿意坐在已经高度腐烂，甚至能看到骨头的尸体吃压缩饼干。”
<br/>　　“现在没有时间给你做选择了。”李然说道：“有人来了！”
<br/>　　“而且人还不少！”罗昊一把推开那扇没有上锁的锈铁门，率先走了进入，沉声下令道：“快，进屋！”
<br/>　　在锈铁门被打开的一刹那，那股子能直接让人把胃里食物吐干净的恶臭扑鼻而来，即便有面罩遮住了口鼻，可是这根本起不了任何作用。这个时候别说是一个面罩，就算是戴着防毒面具，估计都能有尸臭钻进鼻腔。
<br/>　　最后一个进屋的李然，立刻把铁门重新关上。屋子里有两具尸体，已经严重腐烂，尸体上的皮肤多处爆裂开，可以看到森森白骨和红色的肌肉组织，腐烂的尸体上，肥胖的白色蛆虫扭动着丑陋的身体，缓慢的蠕动着，吞食着腐肉，摄取它们所需要的养分。
<br/>　　因为腐烂的关系，此时的尸体已经看不出原来的容貌，但是通过尸体身上的服装，还有生理特征，还是能确定这两具尸体是女性。
<br/>　　至于她们生前最后的一段时刻遭遇了什么，只是用猜测，被迫进入屋子里躲藏的四个人就猜测出。战场上，女人被俘后的遭遇往往比男人更加惨烈，因为她们会经历她们最恐惧的事情。
<br/>　　街道上，三辆栽满了反政府武装人员的皮卡车和两辆在车顶架设了.50勃朗宁重机枪的沙色悍马从屋子前的街道上驶过。
<br/>　　汽车碾压街道所产生的震动，清楚的传到了罗昊、李毅峰、慕容飞、李然四个人的脚下。屋子里四个人身体贴着墙壁，隐蔽起来，不敢探头向外观察，一旦被发现，只需要两颗火箭弹，他们四个人就要跟屋子里这两具腐尸为伴了。
<br/>　　车队没有在街道上做停留，很快就从屋子前驶过。等到车队驶过之后，罗昊才打开铁门走出屋子，可就在这个时候天空又传来直升机螺旋桨切割空气所发出的“呼呼”巨响声。
<br/>　　罗昊立刻示意所有人靠街边建筑隐蔽。罗昊站在一处屋檐下，端起G36C突击步枪，透过武器上的ACOG战斗瞄准镜向天空中看去。
