　　毒手魔君，放眼整个北域也都算是一号响当当的人物了。
<br/>　　上一届武斗大会的前三位，还进入了归墟天，从归墟天之中得到了不少的好处。
<br/>　　这样的人物，只要是在给他一些时间，将来跨越生死屏障，踏入生死台阶，成为一代老祖，几乎是板上钉钉的事情。
<br/>　　可如今他千不该万不该便是招惹到了楚风眠，以至于当场陨落。
<br/>　　“楚风眠，我们要回到武王城吗？”
<br/>　　寒月漓在一旁，对着楚风眠开口道。
<br/>　　“等到过一段时间，我们便是可以启程，前往归墟天。”
<br/>　　寒月漓清楚，楚风眠贸然的带她离开，实际上便是为了引诱这毒手魔君上钩，现在既然已经完成这一项，自然是要离开。
<br/>　　只是归墟天开启的日子，还有一段时间，在这一段时间，他们二人到底要前去哪里。
<br/>　　回北邙学院，本是最好的选择，可是如今楚风眠杀了夜天君，回北邙学院也是一件危险的事情。
<br/>　　夜天君背后的夜家，在北邙学院隐藏极深，但是北邙学院的种种，从来都脱离不开这夜家的影子，楚风眠回到北邙学院，也会有着无数麻烦。
<br/>　　“不，我们回北邙学院。”
<br/>　　楚风眠思考了一下，开口道。
<br/>　　“我现在从武斗大会上，夺得榜首，有着大功，现在回到北邙学院中去，夜家的人想要出手杀我，也绝无可能，不然的话，那夜牧不至于冒着如此风险，想要从擂台之上，杀了我。”
<br/>　　楚风眠低声开口道。
<br/>　　夜牧老祖，选择在武斗大会上，在这擂台上轰杀楚风眠，本身便是极为冒险的一个行为。
<br/>　　就算他是长生之境的老祖，也不可能全然不在乎通天阁的压力。
<br/>　　要是他可以等到楚风眠回到北邙学院，在对付楚风眠的话，也不会顶着如此之多的压力，轰杀楚风眠。
<br/>　　现在楚风眠躲的了武斗大会的榜首，对北邙学院算是有着大功，北邙学院的老祖，也不可能让楚风眠现在死在夜家的手中。
<br/>　　如今回到北邙学院，看似危险，但是实际上却是最好的一步。
<br/>　　寒月漓听到楚风眠的话，点了点头，不过还是提醒道。
<br/>　　“好，不过还要小心行事，这夜牧敢在武斗大会上出手，在北邙学院内，也未必会安分。”
<br/>　　“那是自然。”
<br/>　　楚风眠开口道。
<br/>　　对待敌人，楚风眠可一直没有放松过警惕。
<br/>　　让他现在选择回到北邙学院的原因，除了这个理由之外，楚风眠现在黑玄剑阵的晋升，也是让他心中做出选择的决定之一。
<br/>　　现在这黑玄剑阵，虽然无法跟那些超越生死的老古董真正抗衡，但是拖延一段时间，还是没有任何问题的。
<br/>　　一旦是发生变故，楚风眠便是利用黑玄剑阵去拖延时间，自己也可以全身而退。
<br/>　　“我们走，现在回去。”
<br/>　　说话之间，楚风眠便是跟寒月漓，化为一道遁光，一路飞行，向着北邙学院飞去。
<br/>　　不远的路程，却耗费了两日的时间，在这两日之中，楚风眠也是在利用他血脉的力量，不断的治愈这寒月漓的伤势。
<br/>　　寒月漓被葬天打伤的伤势，也已经恢复完成。
<br/>　　两日之间，两道遁光，便是直接回到了北邙学院之中，来到北邙学院，一路都没有停下，直接是穿越虚空，进入到了北邙圣域之中。
<br/>　　在这两日时间之中，武斗大会所发生的所有事情，也在北域之中开始流传。
<br/>　　楚风眠的名字，已经是被整个北域的武者，全部知晓，铭记于心。
<br/>　　轰杀夜天君，击败冷面魔君，风无心，更是从那夜牧老祖的手中活了下来，这任何一件事，都足以称得上是惊天动地，如今全部在楚风眠这一个的身上发生了。
<br/>　　这一届年轻一代的北域第一人。
<br/>　　这个称号，无人敢反驳。
<br/>　　就连整个北邙圣域之中，都是空前的热闹了起来，北邙学院已经是数届没有夺得过武斗大会的榜首了，如今楚风眠夺魁，他们一个个也都是议论起来。
<br/>　　“咔嚓！”
<br/>　　北邙圣域之中。
<br/>　　上方天空，突然裂开，两道遁光，突然是从其中，飞入了其中。
<br/>　　“是谁？”
<br/>　　“好快的遁光！”
<br/>　　不少北邙学院弟子，纷纷抬起头看了过去，惊讶无比，这遁光的速度，实在是太快了。
<br/>　　“这种速度，莫非是楚风眠？”
<br/>　　“肯定是这楚风眠跟寒月漓二人！”
<br/>　　许多人惊讶一阵，便是反应过来，这遁光的主人，到底是谁。
<br/>　　只有楚风眠，现在才能以如此速度，大摇大摆的从外面，回来北邙圣域。
<br/>　　不过其中的不少弟子，看到楚风眠的遁光，眼神中都露出了几分诧异。
<br/>　　楚风眠现在轰杀了夜天君，居然还敢回来？
<br/>　　楚风眠跟夜牧老祖交手的事情，他们也都是听闻过了，本来在议论之中，不少人觉得楚风眠至少要在外面躲避一阵，至少是要度过了生死大劫，才会选择回来。
<br/>　　没想到现在楚风眠直接不顾一切，就回来了，好像是夜牧老祖的威胁，他丝毫不去理会。
<br/>　　北邙圣域一处山峰上，一座巨大的宫殿之中。
<br/>　　一尊白发老者，正坐在宫殿之中，闭眼修炼，这一尊白发老者，看起来苍老，但是他身上的生机，却是比起年轻人还要旺盛的多，看起来甚至是已经有着几分返老还童的模样。
<br/>　　这显然便是一尊超越生死的人物，在北邙学院中，都是真正的老古董，老祖宗。
<br/>　　突然之间。
<br/>　　宫殿之外，一道人影，从外急速过来。
<br/>　　“夜牧老祖，楚风眠回学院来了。”
<br/>　　这尊人影，正是曾经圣党核心弟子之一的曦至，这曦至在武斗大会上，也取得了前十的名次，如今提前回到了北邙学院之中来。
<br/>　　他看到楚风眠回到北邙学院，便是急速过来禀报。
<br/>　　“那小畜生，居然还敢回来！”
<br/>　　夜牧老祖紧闭的双眼，猛然睁开，一双眼睛，尽是怒火。
<br/>　　“在武王城的时候，我没有杀了此子，现在此子居然还敢回来，我定要杀了他！走！”
<br/>　　这夜牧老祖起身，直接打破空间，带着曦至，一步踏入到了空间之中去。
