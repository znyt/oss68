　　“什么，前辈，您要离开了？”
<br/>　　苏荷听到楚风眠的话，震惊道。
<br/>　　似乎也是有些没有想到，楚风眠居然会说要离开。
<br/>　　“不错。”
<br/>　　楚风眠点了点头。
<br/>　　他不可能一直留在这里，楚风眠所要面对的还有很多，七大宗门，还有北邙学院，这一切，楚风眠都无法抛弃。
<br/>　　碰到故人之后，的确是让楚风眠极为开心，不过楚风眠也依然是要离开了。
<br/>　　现在将苏荷，带到这圣地来，传授给她炼器之道，可以说已经是足以让苏荷高枕无忧了，没有任何危险，可以是好好让她去修炼炼器之道。
<br/>　　这一切，倒是已经足够了。
<br/>　　“你在这里，可以好好修炼炼器之道，也可以突破境界，苏家已经没落，但我希望苏家可以因你再度崛起。”
<br/>　　楚风眠身形一动，他的身影便是从这庭院之中，直接消失不见，只有在他的脚下，留下了一堆丹药跟灵石。
<br/>　　这些，已经是足以支撑苏荷修炼了，在这庭院之中的仓库中，其中蕴含的材料，也已经是足够苏荷修炼了。
<br/>　　他所做的，已经足够了，剩下苏荷能够走到哪里去，也只能是看她自己的造化了。
<br/>　　一直是等到楚风眠离开，苏荷才逐渐的回过神来。
<br/>　　她的心中也明白，楚风眠有着他自己要做的事，不可能一直会留在这里。
<br/>　　而她苏荷，也唯有修炼，等到她真正成为了一代炼器大师后，才有资格去追逐楚风眠的脚步。
<br/>　　“苏家之人，许久未有人来了，孩子，过来吧。”
<br/>　　那在宫殿之中的九龙鼎，缓缓的开口道，呼唤着苏荷的过去。
<br/>　　这青铜大门，随着楚风眠的离开，再度关闭，在外面，谁也不可能想得到，在这白雾之中，居然还会另有乾坤。
<br/>　　楚风眠离开了这苏家圣地之后，便是没有任何犹豫的，前往了附近最近的一处城池之中去，利用传送法阵，直接离开，准备回到北邙学院中。
<br/>　　对楚风眠来说，北邙学院，对他的用处已经不大，但是北邙学院，却是他大师兄一生心血传承所在，还有他的师尊，玄北圣者，也一样在北邙学院之中。
<br/>　　这北邙学院，楚风眠自然是一定要回去的。
<br/>　　足足耗费了十日的时间。
<br/>　　楚风眠这才通过传送法阵，回到了北邙学院之中，一下子前往了北邙圣域，属于他的悟剑峰之中去。
<br/>　　这悟剑峰上周围的剑气，在觉察到楚风眠的气息后，便是纷纷推开，引出一条道路，让楚风眠走了进去。
<br/>　　“是谁？悟剑峰的剑气，居然打开了？”
<br/>　　悟剑峰上发生的变化，自然是瞒不过北邙圣域中无数弟子的目光，这些北邙圣域弟子，不少人都对这悟剑峰，一个个心痒难耐。
<br/>　　悟剑峰，在北邙圣域之中的无数山峰之中，都算得上是灵气最为浓郁的山峰了，在上面修炼，简直是一日千里。
<br/>　　曾经这悟剑峰被夜天君得到，他们这些人不敢逾越窥探，不过后来圣党被楚风眠毁灭，却是令不少人的心中，都有着想法。
<br/>　　毕竟当初夜天君的威势，在整个北邙学院，人人尽知，无人敢去挑衅他的威严。
<br/>　　而楚风眠如今虽然实力强大，更是在北域百绝榜的位列第一名，但是在无数弟子的心中，依然无法比起当年夜天君的威慑。
<br/>　　不过悟剑峰周围的剑气，却是实打实的放在那里，一些北邙圣域的弟子，试图想要接近的一刻，便是被上面的剑气直接毁灭了身躯。
<br/>　　若不是北邙圣域的一尊超越生死的老古董出手相救，只怕是差一点就要陨落。
<br/>　　“楚风眠回来了！”
<br/>　　看到悟剑峰上的变化，不少北邙圣域弟子的心中都清楚，这只有一个可能。
<br/>　　便是楚风眠回来了。
<br/>　　唯有楚风眠归来，才会让悟剑峰发生如此变化。
<br/>　　“这楚风眠，居然还敢回来，本来我都以为他要死在归墟天中了，没想到他居然回来了。”
<br/>　　一尊曾经圣党弟子，看着楚风眠归来，目光闪烁，眼神冰冷到了极点。
<br/>　　“可恶，此人的实力比起曾经，甚至是还要更加强大了，不好对付。”
<br/>　　“跟他在一起的那寒月漓，似乎是没有回来，可能是死在了归墟天中了。”
<br/>　　也有一些北邙圣域的弟子注意到。
<br/>　　这一次，楚风眠是自己一个人回来的。
<br/>　　平时楚风眠跟寒月漓二人，一直以来都是形影不离，在一起的，可这一次却是楚风眠一个人回来的。
<br/>　　“没关系，谁回来了，这次都是在自寻死路，此子居然敢回来，看来是差点忘记当初的事了！”
<br/>　　一尊面容狰狞的武者，正是那狞厉开口道。
<br/>　　“圣主可是夜家的天才，此人想必还不知道吧，在这北邙学院中，谁才是一手遮天的人物，得罪了夜家还敢回来，此人是真的活腻了！”
<br/>　　“一个小子，得到了些机遇，就真的目中无人了，可惜在夜家的眼中，他不过是如同蝼蚁一样的家伙罢了，一根手指都可以捏死，等着吧，此人距离死期不远了！”
<br/>　　狞厉冷笑连连，身躯一动，直接离开了。
<br/>　　楚风眠的归来，在整个北域圣域内，几乎是一日时间，便是所有人都知道。
<br/>　　楚风眠参加武斗大会，得到北域百绝榜第一人，斩杀夜天君，更是从归墟天中得到了极大的机缘，这一切，简直是让楚风眠意气风发。
<br/>　　不过有不少武者，眼看着悟剑峰的时候，都是冷笑。
<br/>　　楚风眠得罪了夜家的人，上一次虽然是被玄北圣者保下了，但是玄北圣者不可能保护的了楚风眠一辈子。
<br/>　　这一次，夜家必然是要直接杀了楚风眠。
<br/>　　不少北邙圣域弟子的目光，已经是都看向了悟剑峰，眼神中尽是贪婪。
<br/>　　只要是等到楚风眠一死，这悟剑峰便是会变成无主之物，倒时候他们谁都可以争抢。
<br/>　　“师尊，居然不在北邙圣域之中？”
<br/>　　楚风眠回到了悟剑峰中，灵识横扫整个北邙圣域，却是没有感觉到，玄北圣者的气息。
<br/>　　这让楚风眠的眉头紧锁起来。
