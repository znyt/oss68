　　“陛下，臣有些话不知当讲不当讲！”苏定方依旧十分犹豫，琢磨了半天才憋出这样一句让人听完恨不得摔杯子的话。
<br/>　　李承乾也是深深吸了一口气，好不容易才压下心头的那股子郁闷，撑起一个笑脸说道：“定方，我们君臣之间应该有十余年的关系了吧？你觉得我这个人是那种动不动就迁怒的人么？”
<br/>　　“陛下，这不是迁怒不迁怒的事情，实在是，实在是臣不知道应不应该说。”苏定方苦笑说道。
<br/>　　“你就放心大胆的说，朕可以保证，绝不多想，如何？！”李承乾为了满足自己被吊起来的胃口，向苏定方保证道。
<br/>　　苏定方再次犹豫了一下，最后狠狠一握拳，像是下定了某种决心：“陛下，前段时间有人找到微臣，与臣说起一件事情，大意是指陛下想要鸟尽弓藏。”
<br/>　　“鸟尽弓藏？”李承乾先是一愣，没有想明白是怎么一回事儿，待看到苏定方纠结的表情时才恍然说道：“你的意思，有人告诉你朕想要拿换防当借口，免了你们的权力？罢你们的官职？”
<br/>　　苏定方点了点头：“陛下，臣并没有相信此人的话，臣知道陛下完全没有必要这样做，相信其他几位将军也都是这样想的。可是……”
<br/>　　“可是什么？”李承乾问道。
<br/>　　话说到这个份上，苏定方索性也放开了，毫不犹豫的说道：“陛下，这分明就是有人想要破坏您与臣等之间的关系，臣觉得这应该不是个人的行为。”
<br/>　　这当然不是个人行为！李承乾想到前段时间黑子他们回报帝国内部正有一股不明势力在搞事情，立刻将苏定方说的事情和那股不明势力联系了起来。
<br/>　　但是这些话他没有对苏定方说，只是低头想了想，半晌之后才说道：“定方啊，这件事你注意一些，如果有几会的话，和仁贵他们讨论一下，就跟他们说，如果再有人找到你们说这种事，务必要稳住他们，不要让他们起疑心。”
<br/>　　“陛下，您的意思是……放长线钓大鱼？”苏定方到底是智勇双全之辈，一听李承乾的安排，立刻明白了他的意思。
<br/>　　“对，朕就是要看看，到底是谁在打大唐的主意，如果有可能的话，朕要把它连根拔起。”承乾微微眯起眼睛，恨声说道。
<br/>　　这一辈子，他最讨厌的就是那些背后搞小动作的人，那些人不顾大局，只为自己的私利，甚至不惜挑起战争，也要满足自己的欲望。
<br/>　　明明一个两个都有着不凡的能力，可是这种能力却没有被他们用到正地方，只是想着一些莫名其妙的东西，被自己的欲望控制着，然后还要找各种各样的理由，来掩饰自己的错误。
<br/>　　这种人在李承乾看来全都该死，如果不能将他们全部绳之于法，将他们的恶行公之于众，就对不起自己这个大唐皇帝的身份。
<br/>　　和老头子相比，李承乾并不认为自己是一个明君，可至少也不是一个昏君，大唐旬报是他对外的喉舌，同样也是他留给某些人与自己沟通的渠道。
<br/>　　如果有人对大唐的某些政策有意见，完全可以通过旬报来提出自己的观点，大家一起来讨论，然后再做出正确的决定。
<br/>　　可是这份心思显然并没有被人所理解，他们固执的认为自己是对的，在暗处鼓捣一些见不得人的事情，若是惹出什么麻烦，这些人便找个地方一躲，装出事不关己的样子，由大唐来替他们这些人擦屁股，由百姓来替他们这些人埋单。
<br/>　　所以现在的李承乾对那些敢于对大唐发起挑战的人没有任何的怜悯之心，只要有机会，抓到一个杀一个，抓到两个杀两个，不给他们留下任何侥幸的机会。
<br/>　　苏定方从李承乾语气中听出了浓浓的杀机，心中也是微微一紧，虽然这件事情与他没有什么直接的联系，但是很显然当水落石出的时候，会有很多人会人头落地。
<br/>　　……
<br/>　　而就在李承乾给苏定方布置任务的时候，长安城中的那些异族们也都收到了一条消息：三日后，大唐的贵族们将去咸阳城参加一次狩猎活动。
<br/>　　甚至那些异族们还听说，这次活动的的召集人是一个大人物，是皇帝陛下的亲信，有好多大唐的贵族都会去参加。
<br/>　　有了这样的消息，那些异族们自然是不甘人后，一个两个的各自找到自己的门路，打算去咸阳城见识一番。
<br/>　　不过这些人显然是醉翁之意不在酒，去咸阳的目的也不是那么单纯。
<br/>　　要知道，咸阳可是大唐科学院的地盘，那里有着大唐许多最顶级的机密，谁知道去了之后会不会有机会得知一两样？或者有机会的话，搞到几个专业的人才也是一件不错的事情。
<br/>　　托尼克自然也收到了这份消息，不过他得到的消息要比其他人要多一些，通过一些渠道他知道了这次活动的召集人叫李禾，也知道参加这次活动的人几乎包括了各大世家和士族。
<br/>　　另外还有一点就是，他还得到通知，知道这次的活动背后还有另外的一层深意，至于具体是什么，只可意会不可言传。
<br/>　　对于这样的一个活动，托尼克自然不会错过，就算是不知道那些隐秘的事情，他也是要过去踩踩盘子的，更不要说他还提前知道了一些东西。
<br/>　　但让他有些无法接受的另外一件事情，因为他这次如果想去咸阳就必须带上一个人，一个长的十分帅气，但却有一双阴鹫双眼的男人——李毅恒。
<br/>　　他不知道与自己合作的那个家伙为什么会让自己带上这个人，但如果他不答应，那么就会与这次的活动失之交臂。所以托尼克无奈的答应了前来送信人的要求，同时也将那个年轻人安顿在了自己的队伍之中，两日之后坐上了通往咸阳的火车。
<br/>　　一路无话，那些异族随着那些大唐的贵族来到了咸阳，仅仅半天功夫，就把这座平日里安详宁静的城市变的喧嚣异常，就好像是另外的一个长安一般。
