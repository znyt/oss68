　　星玄的来历，一直是神秘无比。
<br/>　　唯一楚风眠对他的了解，便是这战龙决。
<br/>　　战龙决，绝不是九域可以创造出的武技，起码在楚风眠的认知中，九域不可能有人可以创造的出如此可怕的武技来。
<br/>　　“没什么，这东西是极为珍惜之物，你能够得到，也算是你的机缘了。”
<br/>　　听到楚风眠的话，星玄随意是岔开话题道。
<br/>　　“我知道你有很多想问的，但是至少你也要将战龙决的第一重修炼到大成，我才会告诉你。”
<br/>　　说完，星玄的声音便是直接消失。
<br/>　　任由楚风眠如何开口，他都不在出现。
<br/>　　“这星玄的来历，只怕是来自域外星空。”
<br/>　　楚风眠抬起头，看向天际，这天际之外，便就是域外星空。
<br/>　　域外星空到底有多大，只怕是没有任何一个人可以知晓，这星玄来自与域外星空，也并非是不可能。
<br/>　　“算了，没有实力，就算是知晓也没有任何的用处。”
<br/>　　楚风眠摇了摇头。
<br/>　　没有足够的实力，他问再多也没有任何用，等到他有了实力，星玄自然会告诉他的。
<br/>　　楚风眠坐在床上，便是继续缓缓修炼起来，利用天晶的力量，将身体精华到了完美的地步。
<br/>　　两日时间，刹那之间便是过去。
<br/>　　“少主，北邙学院的考核快要开始了。”
<br/>　　门外，拓跋鼎敲门，提醒道。
<br/>　　“好。”
<br/>　　楚风眠睁开双眼，缓缓起身，便是推开门离去，寒月漓已经休息好。
<br/>　　如今三人，便是直接向着北邙学院的方向走去。
<br/>　　这考核的地方，并非是在北邙学院，而是在一处巨大无比的宫殿之中，北邙宫。
<br/>　　这一座宫殿，足足是比起许多城池都要庞大，足以容纳百万人。
<br/>　　楚风眠一眼望去，如今这一次来参加北邙学院考核的武者，都有着百之多。
<br/>　　人山人海，极为壮观。
<br/>　　“居然如此多人。”
<br/>　　寒月漓在一旁，看到这一幕，是深深的震撼住了。
<br/>　　如此之多的青年才俊，不知道是来自与多少国度，整个北域的天才，如今都到来的。
<br/>　　“这倒是极为正常的，若是没有这么多人，北邙学院也不会如此昌盛。”
<br/>　　楚风眠听到寒月漓的话，笑着说道。
<br/>　　他的心中倒是丝毫不意外，前世楚风眠曾经见到过七大宗门开宗收徒之日的样子，比起这还要恐怖许多。
<br/>　　毕竟七大宗门这等庞然大物，开宗收徒，前去的武者可不单单是一域之人，整个九域的天才，都会前往。
<br/>　　比起这声势还要浩大许多。
<br/>　　如此之多参加考核的青年才俊，经过筛选，才会被选入到北邙学院中。
<br/>　　每一个进入北邙学院的弟子，都是千挑万选的，至少是百中挑一，都是一方天才的人物。
<br/>　　没有任何一人，会是泛泛之辈。
<br/>　　“这在场的人，其中九成九都会失败，只有百分之一的人，才能加入到北邙学院中去。”
<br/>　　拓跋鼎看到这一幕，也是暗自感叹一声。
<br/>　　他曾经也参加过这考核，自然懂得其中的残酷。
<br/>　　能够真正进入到北邙学院的，只怕是百不足一，而这些人，只是最为普通的外门弟子，想要真正拥有地位，晋升核心弟子，又是百不足一。
<br/>　　只怕是一万个人中，才会有着一人可以晋升为内门弟子，绝对是残酷考核。
<br/>　　至于剩下的人，则全部都是垫脚石一般的存在。
<br/>　　这边是残酷的养蛊。
<br/>　　大宗门的方式，便都是养蛊，拥有无数弟子，让他们自相残杀，争夺，脱颖而出的人，才会收到重视。
<br/>　　而失败的人，则只能默默无闻。
<br/>　　虽然残酷，但是在九域这个以武为尊的地方，便正是如此。
<br/>　　“这考核，我是不能参加的，所以只能在外面旁观。”
<br/>　　拓跋鼎在一旁，对着楚风眠开口道。
<br/>　　他是北邙学院的弟子，自然不能在参加一次，只能是离开了宫殿之中。
<br/>　　毕竟这考核，可是北邙学院最为重要的事之一，不允许出现任何问题。
<br/>　　一尊年轻男子，步入到这宫殿中。
<br/>　　这年轻男子，长相极为俊美，气宇轩扬，一看便是大家族的弟子。
<br/>　　看到他的出现，楚风眠身边周围不少人都是议论起来。
<br/>　　“你们看，东州燕家的少主燕古也来了。”
<br/>　　“好强大的气息，虽然只是神力境七重的境界，但是这一股气息，只怕是足以达到巅峰圣者的地步了。”
<br/>　　“我们居然跟这样的天才一届，这哪里竞争的过。”
<br/>　　“不用担心，此人的资质，进入学院，肯定会被一些长老，直接拉入到内门中的，所以不用担心。”
<br/>　　“哦？此人的实力，倒是不差。”
<br/>　　楚风眠眼神一动，看向那燕古，悄然之间动用灵识，将这燕古探察的一边。
<br/>　　这燕古的境界，便是神力境七重，但是如今他身上的气息，却是足以比拟巅峰圣者。
<br/>　　这绝对是一尊赫赫有名的天才。
<br/>　　但是楚风眠也注意到，这燕古表现出来的实力，其实还有着极大的隐藏，不足燕古真正实力的十分之一。
<br/>　　这燕古的真正实力，只怕是战斗指数也逼近十，接近那夺花尊者的地步。
<br/>　　这种实力，在这一届新人中，极为显眼，赫赫有名。
<br/>　　除了这燕古，楚风眠也看见，在人群中，也有着不少隐藏实力的强者。
<br/>　　应该说在场的强者，大多都隐藏了自己的实力，毕竟所有考核的人，实际上互相之间，都是对手。
<br/>　　所以并没有人会主动暴露出自己的实力来。
<br/>　　正午。
<br/>　　这宫殿中，已经是满满的站满了人。
<br/>　　“砰！”
<br/>　　一声巨响，宫殿的大门关闭，一尊高大的武者，突然站了起来。
<br/>　　这一尊武者，足足是有着生死境五重的境界，战斗指数达到两百，是楚风眠绝对无法抗衡的存在。
<br/>　　“这就是主持这一届考核的苏凌长老。”
<br/>　　楚风眠看着眼前这尊生死境五重，通过拓跋鼎的关系，自然清楚此人是谁。
<br/>　　拓跋鼎已经是将关于考核的事，全部告诉楚风眠了，所以楚风眠也算是提前知道了许多事。
<br/>　　“让一尊生死境五重的强者来担任这次的主持，北邙学院倒是大手笔。”
<br/>　　楚风眠看着那苏凌长老，暗自想到。
