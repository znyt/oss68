　　第249章
<br/>　　绿色的眸子，深灰色的睫毛轻轻眨动，黑纱下的面孔莹润如玉，白皙如象牙。
<br/>　　蓝青婷不是一个丑八怪，她黑纱下的面孔并非着了火伤后的狰狞可怖，而是完美得如同魔鬼。
<br/>　　如此，也难怪男人开口的第一句话就是惊叹。
<br/>　　蓝青婷脸红了红，虽然身后杀人的马匪渐近，她还是红了红脸。下意识想说的是：你也挺好看。
<br/>　　男人长得不高，脸颊有些婴儿肥，咕噜噜的眼睛滚动，宛若黑星。
<br/>　　“你快走，马匪…是马匪…”
<br/>　　蓝青婷失声大喊，因紧张而呼吸急促，怕他还不知道马匪的可怕，此刻脸上竟然没有什么反应，这么好看的少年，实在不该死在马匪刀下。
<br/>　　“你快走啊，混蛋！”蓝青婷忍不住骂出声。
<br/>　　马蹄声疾，十几个马匪将两人包围，蓝青婷心脏砰砰乱跳，抓起腰间的匕首向喉咙扎去。
<br/>　　相比死亡，落在这些马匪手中会更加可怕。
<br/>　　突然，一只干净的手掌抓住她手腕，将匕首拍落。
<br/>　　蓝青婷下意识抬起头，见少年垂着头，满脸笑意的看着她。蓝青婷还没见过，世间竟有笑得这么好看的少年。
<br/>　　“大当家。”
<br/>　　蓝青婷这才发现，马匪不知何时已经从马上跃下，他们围成一圈，口中高呼。
<br/>　　他们跪拜的自然不会是自己，而是——面前的少年。
<br/>　　蓝青婷从心灵深处打了个寒颤，抬起头望着面前的少年，一股深沉的寒意瞬时间吞没了她。
<br/>　　边城十三家马匪之首——鬼见愁。
<br/>　　怪不得他见到自己一直笑呢，那是，一个饕餮客看见食物的笑。
<br/>　　蓝青婷好像陷入沼泽中，使不上半分力气。边城十三家马匪，以鬼见愁为首，鬼见愁以残忍著称。但蓝蜻蜓难以相信，令所有人闻风丧胆的鬼见愁，竟然是一个娃娃脸少年。
<br/>　　“大当家，肥羊快到了。”
<br/>　　“到了么，那就让弟兄们忙活起来。”
<br/>　　“这次护送的可是鱼龙卫……咱们要不要再考虑一下。”
<br/>　　“鱼龙卫又如何？”少年挑眉问。
<br/>　　周围鸦雀无声。
<br/>　　……
<br/>　　夕照关，帝国西北之地最后一道重关。离开此关，就是浩瀚草原，也就是戎族地界了。
<br/>　　当然，戎族对于此地也没有实际控制权，自然他们也不会控制，这些生活在草原上的家伙，除了抢东西和放马就不知道还有第三件事。
<br/>　　于是，夕照关外二百里，成了三不管地带，戎族是根本不管，帝国是想管没力气管，于是这片地带，成了马匪，奴隶贩子，恶棍暴徒的乐园，几乎每天都能见到抛尸荒郊的尸体，没几多时间变成白骨，这地方的野狗也比其他地方更肥些。
<br/>　　夕照关守将雄卯今天病了，生病的原因不是风寒，不是气候，而是……帝国明玉公主要从夕照关经过。
<br/>　　夕照关自然要做好招待工作，除了明玉本身的侍卫外，夕照关还要排出守军，护着明玉公主出关五百里。
<br/>　　这些事情自然是要做的，但雄卯不想亲自送。
<br/>　　身为夕照关守将，他和戎族的接触自然比京城的老爷们多。而他们经常接触的方式，自然是弓箭，滚石，火药。
<br/>　　他对戎族恨之入骨，这种送女人给戎族的事情，对他来说是奇耻大辱。可他人微言轻，也改变不了什么，索性，眼不见心不烦，接待护送的事情都让副将去做，自己窝在屋子里装病。
<br/>　　好在，他们终于要走了，雄卯也可以走出屋子，到外面透透气。
<br/>　　“大人。”副将肖竹跑进屋子：“明玉公主马上要出关，咱们派多少人随行，要不，派五千？”
<br/>　　“五千？”雄卯咧咧嘴：“有你这么不会过日子的么，给他们一千人就可以了。象征性的送一送，送完赶紧回来。”
<br/>　　“大人，现在关外可是不太平，我听说有贼人要劫和亲队伍。”肖竹道：“只有一千人，是不是忒少了些，明玉公主的侍卫才五百。”
<br/>　　“五百？”雄卯冷哼一声：“那可是五百鱼龙卫！”
<br/>　　“鱼龙卫……”肖竹顿了顿，道：“大人，这鱼龙卫老是听人说，到底有什么了不起的？”
<br/>　　鱼龙卫，这三个字帝国听说过的人很多，但真正见过他们的却寥寥无几。因为这支队伍，保护的不是朝廷，也不是长安，而是皇宫中的某个人。
<br/>　　明帝。
<br/>　　他们是由明帝直接指挥的队伍，是明帝杀人的刀，也是明帝防身的盾。
<br/>　　鱼龙卫内部情况不详，组织架构不详，只听传言说，他们由明帝秘密培养，分为鱼卫和龙卫两支。鱼卫是从各地找来的孤儿，由高人传授武艺，择地秘密训练。鱼卫中优秀者可以晋升为龙卫，龙卫放到军中，可抵一员猛将，放到江湖能成一方悍匪。
<br/>　　而明帝又招揽江湖上的奇人异事，比如一掌击落程大雷的青叶佛，便是龙卫之一。
<br/>　　简单来说，鱼龙卫就是一支明帝不计代价训练的特种兵。
<br/>　　肖竹将和亲队伍送出城，他也带了一千守军随行。这支队伍将一支跟着明玉公主的车队，直到将对方送到戎族的王城。
<br/>　　和亲队伍的侍卫负责人，是一个叫宫符的男人，双方一接触，肖竹就感受到了对方的专业。
<br/>　　肖竹把关外的情况与对方说了，言外之意便是这里情况很复杂，一路上怕是不会很顺利。
<br/>　　宫符没有看轻地方军的意思，认真倾听肖竹的话，这让肖竹有一种自己很受尊重的感觉。宫符将这些情报记在本子上，又与其他几人商量，诸人都表现得很谨慎，并没有任何丝毫轻敌的意思，但也没有流露出任何畏惧。
<br/>　　“救命，救命！”
<br/>　　队伍正往前行，前方出现呼救的声音，在道边一棵大槐树上，吊着一个人。
<br/>　　漆黑长发，绿色眸子，关键是身上没有任何衣物。那如绸缎般的身体浮现在众人眼前，众人没有其他感觉，只有愤怒。
<br/>　　是怎样的禽兽，舍得如此糟蹋这样一个倾国倾城的女子。
<br/>　　“来人，快将她救下来。”肖竹大喊。
<br/>　　“慢。”宫符拦下了他，谨慎的盯着树上的女人。
<br/>(本章完)
