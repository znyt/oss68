　　第76章
<br/>　　院门打开，一众人涌进来。杜茂正在树下饮酒，微微皱起眉头。
<br/>　　“疤子，你将他们赶走就是，抓他们都过来干什么。”
<br/>　　疤子往前站了一步，道：“大哥，这两人是卖酒的，他们的酒一百文一碗，能够连喝十碗就不要钱。”
<br/>　　杜茂扫了那老者和年青人一眼，二人脸上的表情都很拘谨。他微微笑笑：“这倒有些意思，什么样的好酒，敢夸这样的口。”
<br/>　　一片笑声响彻院子。
<br/>　　“疤子，看你的样子，也是喝了吧，怎么，没撑下去。”有人问。
<br/>　　疤子脸上涨红：“什么话，我是碰到好酒，想让大哥尝尝，这样的酒可灌不醉我。”
<br/>　　“哈，你小子别吹牛了，你的酒量谁不知道，不行就说不行。”
<br/>　　有人跃了出来，拍着胸脯道：“让俺来试试。”
<br/>　　这人长得像头大黑熊一样，虎背熊腰，下巴乱渣渣络腮胡子。
<br/>　　“这酒可不算什么，连这也能吓住你。”这人端着酒碗，不忘嘲笑疤子一声，说罢将碗中酒一饮而尽。
<br/>　　一股辣意往上涌，说实话，这种酒的口感并不好，可喝到肚中就如翻江倒海一般。只是一碗酒下肚，他眼也花了，头也晕了，身子也开始打趔趄。
<br/>　　“怎么，你也不行了吧，我刚才可连喝三大碗。”疤子赶忙发出嘲笑，又冲那老者递个眼神，不让他拆穿自己。
<br/>　　“谁说我不行，我只是刚才吃得太饱，撑着了。”『大黑熊』道了一声，端着碗一伸：“再来一碗。”
<br/>　　喝到第三碗时他便不行了，身子摇摇晃晃，哇呀呀吐个不停。
<br/>　　“这酒真这么烈！”
<br/>　　有人忍不住道一声，这大汉的酒量他们是了解的，平时喝酒如喝水一般，怎么今天三碗酒就给放倒了。
<br/>　　“来，给我也来一碗！”
<br/>　　紧跟着又有几个人跳出来，将碗递到那老者面前。
<br/>　　“诸位好汉，我们是小本生意，刚才几碗酒就算我送你们的，不要钱了，求你们放我们走吧。”那老者道。
<br/>　　“怕什么，以为我们给不起钱么。”杜茂取出一碇银子，放在桌子上：“今天无论有没有人能喝十碗酒，这银子都是你的。”
<br/>　　如此一说，众人都来了精神。
<br/>　　“我这辈子从没有喝醉过，今天我就连喝十碗给乡亲父老看看。”
<br/>　　“来，老头，把碗给我舀满，少舀一点，我要你脑袋。”
<br/>　　说是这样说，可到最后院子里的人轮过来一圈，莫说十碗，能喝过三碗的都没有几个。都迷迷糊糊倒在地上，吐得吐躺的躺。
<br/>　　“大哥，我们不行了。”
<br/>　　“这酒真是烈，以前从没有喝到过。”
<br/>　　这时代的酒，大部分都是酿造酒，黄酒果酒之类的，先莫说口感，度数肯定是上不去的，能到十几度就顶天了。差不多相当于程大雷前生的啤酒，否则怎有那么多江湖大侠成斤成斤的喝酒。
<br/>　　如果是啤酒那样的度数，只要让吐，换程大雷也能喝它几斤。
<br/>　　而蛤蟆寨按照程大雷的思路制作蒸馏酒，总不是什么技术难题，渐渐被他们摸索出来。这样的酒其实很难喝，但就有一个好处，够猛够劲，程大雷尝了尝，大概在四十几度左右。
<br/>　　而如今碗的容量也大些，一碗酒大概七八两左右。能够连喝十碗的猛人，程大雷反正没见过。
<br/>　　而且，喝惯啤酒的人，他喝不了白的呀。
<br/>　　院子内醉倒一片，大家都将目光放在杜茂身上。
<br/>　　“把酒与我斟满。”杜茂坐在那里纹丝不动。
<br/>　　“大哥海量，让他们见识见识。”
<br/>　　老汉将酒给他斟满，好心劝道：“好汉，我这酒是烈了些，您悠着点。”
<br/>　　“什么话，我大哥千杯不倒，当年打虎全靠烈酒助兴，你这些酒还想惯倒我大哥。”疤子只喝了一碗酒，如今还精神。
<br/>　　“嗳……山野村夫什么都不懂，你冲他大吵大嚷做什么。”
<br/>　　杜茂用手一推，阻止疤子说下去，紧跟着将碗中酒饮尽。
<br/>　　酒刚入腹，就像刀子刮过喉咙相仿，杜茂立刻就知道坏喽，今天要丢人。
<br/>　　“快，快给我大哥倒满，我大哥酒兴刚起。”
<br/>　　杜茂狠狠的瞪了疤子一眼，疤子忙点头，一副我懂你的表情。
<br/>　　还没回过神，面前的酒碗已经倒满，杜茂深吸一口气，如老牛饮水般一饮而尽。
<br/>　　杜茂连喝三碗，就已经撑不住了。他本来坐着，此刻不得不站起来，用手撑着大树。
<br/>　　“好汉，不行就别喝了，今天的钱我不敢要了。”
<br/>　　“来，倒满。”
<br/>　　三碗酒入肚，杜茂也迷糊了，老者这句话恰好激起他心中的豪横之气。
<br/>　　又是满满一大碗酒，杯中一滴残酒无有。
<br/>　　人群齐声叫好，以后杜茂每喝一碗酒，人群就叫一声好，最后杜茂一连喝了七大碗酒。
<br/>　　就连那老者心中也忍不住赞了一声，此人真是一条好汉，想我们那大当家，一碗酒就干躺了。
<br/>　　“好汉，还要喝嘛！”老汉问道。
<br/>　　“倒……倒满……”
<br/>　　“我劝你还是不要喝了，你平日不烧香不敬佛，酒喝多了，半夜容易撞鬼。”
<br/>　　“倒……”杜茂忽然一激灵，猛地抬起头，只见那老者目不转睛的盯着他，脸上那还有半点恭敬温顺的样子，双目之中赫然绽放虎一样的凶芒。
<br/>　　“你……你们是……”
<br/>　　“动手！”
<br/>　　老者突然把手中的酒碗摔在地上，随着哗啦声响。推车旁边的年青人跃到车上，从酒桶中捞出几把钢刀，丢给七八个围观进来的闲汉。
<br/>　　这些『闲汉』各个昂起头，脸上杀机流露。
<br/>　　真有杜茂附近的街坊，闹不懂是怎么会事，此刻一脸错愕。
<br/>　　疤子呆住，他诧异的看到模样丑陋的驼子突然直起身，接住抛过来的钢刀
<br/>　　“你是……”
<br/>　　“蛤蟆寨做事，无关人等滚开。”
<br/>　　人的命树的影，蛤蟆寨凶名在外，论名声并不比杜茂弱。一听蛤蟆寨的恶贼到了，人群慌张一片，有人干脆吓倒在地，身下流出黄白之物。
<br/>　　杜茂一脚将桌子踹翻，杯碗乱飞，他两只大拳挡在身前，口中雷霆一喝。
<br/>　　“来！”
<br/>　　
<br/>(本章完)
