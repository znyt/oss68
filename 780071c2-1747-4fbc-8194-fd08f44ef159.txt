　　紫云洞，位于萧家宅邸最深处，常年有家族强者坐镇，外人休想进入。
<br/>　　哪怕是萧家子弟，平日也无法靠近。
<br/>　　在距离紫云洞还有一段距离时，所有族人，乃至长老和执事都停下了脚步，只有获得修炼名额额子弟才能继续前进。
<br/>　　萧逸、萧若寒、萧子木、萧壮等十人，在族人们带着希冀和充满希望的目光之下，一步一步走着。
<br/>　　他们，代表着萧家年轻一代最出色的子弟；他们，将是未来扛起萧家大梁的武者。
<br/>　　所有族人都暗暗祝福着，希望他们能在紫云洞内能有所收获。
<br/>　　除了一人，五长老。
<br/>　　那掩饰得极深的恶毒目光，正紧紧地盯着萧逸的后背。
<br/>　　“小废物，就让你得意一番又如何！只要我的计划没被打乱，我儿若寒从紫云洞出来之时，就是彻底将你从萧家抹去之际。”
<br/>　　五长老早有一连窜的恶毒计划。
<br/>　　另一边，与萧逸和其他子弟一同前进的萧若寒，也是斜眼看了萧逸一眼，心中暗暗得意，“哼，今日让你一鸣惊人又如何。我踏入凡境七重已久，马上就能突破凡境八重，在紫云洞修炼十日，很有机会突破到后天境，到时候，你的死期便到了。”
<br/>　　紫云洞三年开启一次，每一次，进去修炼的家族子弟都会修为大增，连跳几重也不是不可能。
<br/>　　萧逸敏锐地捕捉到萧若寒的眼神，更感觉到了背后如芒在刺的目光，但他没有在意，毫不在乎地笑了笑。
<br/>　　“这对阴险的父子，又在想什么坏主意吧。”萧逸暗道。
<br/>　　若是以前的萧逸，肯定早就被他们收拾惨了。可现在的萧逸，只当他们是跳梁小丑罢了。
<br/>　　“嗯？”萧逸忽然顿了顿脚步，目光定格在远处。
<br/>　　那里，有一个巨大的雕像，通体红色，栩栩如生。
<br/>　　单看外表，这似乎是一种不知名的妖兽，有些像狮子，但它更巨大，头顶有一只骇人的尖角，背长双翅。
<br/>　　整个萧家上下，没人知道这尊妖兽雕像的名字。只知道，这是百年前萧家老祖宗放在这里的。
<br/>　　自从萧家在紫云城立族，它就存在。
<br/>　　每个萧家子弟，在觉醒武魂前，都会来这里观摩它。
<br/>　　武者觉醒的武魂，来源于这个世界的一切事物。
<br/>　　武者年幼时，一般都会通过观摩各种事物，眼刻于心，心传于神。武者对这样物体越是有感觉，印象越深，越能觉醒出这种武魂。
<br/>　　比如，常年去观摩一根草，便觉醒出草武魂。
<br/>　　常年观摩一把剑，便觉醒出剑武魂。
<br/>　　自然的，很多大家族，或者门派、宗门等等，都会准备一样比较厉害的物品，给予年幼的子弟常年去观摩。
<br/>　　萧家也一样，据说，萧家最宝贵的东西，就是这尊雕像了。
<br/>　　不过，越是厉害，越是高级的事物，越难觉醒。
<br/>　　比如说一把绝世神剑，天赋不佳者，很有可能只觉醒出一把比较锋利的剑；甚至更差的，会觉醒出最最普通的铁剑。
<br/>　　当然，也会有天赋极高者，觉醒出比较厉害的剑武魂。
<br/>　　萧家也一样，从来没有任何一人能觉醒出这尊妖兽雕像的武魂。
<br/>　　而且很显然，这尊雕像的妖兽，是一只火属性的妖兽。
<br/>　　所以，萧家中几乎所有人觉醒的武魂都跟火属性有关。
<br/>　　萧若寒的火云蟒，萧石的火狼，箫星扬的蛮火牛等等。
<br/>　　而当年的萧逸资质极其低下，便觉醒出了最垃圾的控火兽。
<br/>　　萧逸撇了撇嘴，暗道，“若是我早个几年穿越到这里，或许能觉醒出这雕像刻画的妖兽的武魂。真想看看，这只妖兽到底是什么东西，又有多强大。”
<br/>　　恰在他有这个想法之时，身体忽然猛地一震，一股前所未有的危机感油然而生。
<br/>　　自己仿佛被一只绝世凶兽盯上了一般，全身发麻，不能动弹。
<br/>　　他仿佛看到了，那尊妖兽雕像铜铃般的摄人目光，正紧紧地盯着他，意欲将他的心神吞噬。
<br/>　　他感到无力，感到恐慌；他发现，在这股感觉之下，自然竟毫无反抗之力。
<br/>　　忽然，他体内又是一阵躁动，冰鸾剑武魂猛地光芒大作，一股充满了锋利、无坚不摧的气息笼罩了他的身体。
<br/>　　在冰鸾剑的这股气息出现后，那股危机感瞬间消退，身体也恢复了正常。
<br/>　　“怎么回事，错觉吗？”萧逸咬了咬牙，他发现，自己后背的衣衫，竟已经被汗水打湿。
<br/>　　萧逸没来得及想清楚问题，被一道粗犷的声音打断了思路。
<br/>　　“萧逸，愣着干嘛，跟上阿。”
<br/>　　“额，哦，好的。”萧逸愣了愣，连忙跟上。
<br/>　　刚才叫他的是萧壮。
<br/>　　“萧逸，前几天的事，谢了。”萧壮走到萧逸身边，低语道。
<br/>　　“小事罢了，不必谢。”萧逸耸耸肩。
<br/>　　萧逸前不久才知道，那天被萧杰和萧石欺负的年轻子弟，竟然就是萧壮的弟弟。
<br/>　　萧壮，人如其名，长得魁梧高大，像头牛似的，摇了摇头，道，“才不是小事，日后又机会的话，我一定会还你这个人情。”
<br/>　　“随你。”萧逸淡淡地应答了一声。
<br/>　　几分钟后，众人终于来到了紫云洞前，而这时，二长老早已在这里等候众人。
<br/>　　所谓的紫云洞，其实不过是以妖兽雕像为中心，而后划出一块方圆数百米的地方。
<br/>　　一般年幼的家族子弟，就算是观摩，也只能在千米之外。
<br/>　　只有三年一次开启紫云洞时，有名额的子弟才能靠近这数百米之内。
<br/>　　“众子弟听令，上前一步。”二长老没有任何废话，直接威严地喝了一句。
<br/>　　十位子弟同时上前一步。
<br/>　　当这一步踏出之时，萧逸立马露出了惊讶之色。
<br/>　　因为，周遭的气息竟然完全变了。明显可以感觉到，身旁的灵气浓郁到一个可怕的地步。
<br/>　　周围的灵气，竟然比外面浓郁了十倍都不止。甚至不用自己去主动吸收，周围的灵气已经自动涌入体内。
<br/>　　“果然是个修炼的绝佳宝地。”萧逸惊讶地赞叹一声。
<br/>　　而且，这是最精纯的天地灵气，没有任何杂质，也不用有任何顾虑，比吞服丹药的效果好太多太多了。
<br/>　　萧逸转身看了看后面，二长老就站在他们一步之后。
<br/>　　明明只是一步之遥，周围的环境竟然变化如此之大。
<br/>　　萧逸猜测，这里必然有一种强大的力量，封锁了数百米内的浓郁灵气，让这些灵气不会泄露出去，哪怕是泄露一步都不可能。
<br/>　　这时，一步之外的二长老严肃地吩咐道，“听好了，萧家雕像，存在着极强的威压，越是靠近，压力越大。五十步之内，是你们承受的最大限度。五十步之外，连后天境都难以承受。”
<br/>　　“切记，任何人不可跨出五十步外，否则，将有生命之危。”
<br/>　　“谨遵二长老吩咐。”十位子弟恭敬地应答一声。
<br/>　　二长老点点头，在严肃地吩咐后，自顾地再退后几步，盘膝坐下，开始修炼。
<br/>　　真不愧是武痴，不会浪费任何一点儿修炼时间。
<br/>　　众人开始继续往前走，越是靠近雕像，威压虽然越大，但蕴含的灵气也越大，自然地，修炼起来也更快。
<br/>　　当然，只要不超过五十步距离，就不会有危险。
<br/>　　按照规定，萧逸走到了最前方，恰恰是五十步便停了下来。
<br/>　　五步之后，是萧若寒。
<br/>　　再之后，就是按家族比武的名次来决定。
<br/>　　五十步的范围，按照每人间隔五步，十人分别开始了修炼。
<br/>　　五步的距离，正好是彼此修炼不会相互影响，也不会抢了别人身旁的灵气，影响别人的修炼。
<br/>　　萧若寒看着面前萧逸的背影，恨得咬牙切齿，萧逸的位置是灵气最浓郁，修炼最快的位置，他自然眼红。
<br/>　　萧逸自然感受到他的目光，但懒得鸟他。现在最重要的，是赶紧修炼，增强修为。
<br/>　　在体内双武魂的吸收下，周围的灵气铺天盖地涌来。特别是冰鸾剑那恐怖的吸收速度，萧逸立马感觉到，自己的修为正以一种骇人的速度提升着。
<br/>(本章完)
