叶翎是炙手可热的大明星，走到哪里都会被媒体记者偷拍，所以今天她穿了一件很低调的私服。
<br/>
<br/>        叶翎身上是一件白色宽大的T桖，外面罩了一件黑色衬衫，跨了一个小香风的流苏小包包，这原本是很简单的穿着，但是穿在叶翎身上就是与众不同，俏媚充满元气，那裸露的两条白腿十分纤长，无比的养眼。
<br/>
<br/>        怕人认出来，叶翎戴了一顶鸭舌帽，茶色的卷发慵懒披散肩头，巴掌大的小脸上还有一副宽大的墨镜，遮的很低调严实，但她娇艳的菱唇唇红齿白，在机场走了一路就“刷”的吸引了所有人的目光。
<br/>
<br/>        夏夕绾挥动小手，心情从阴转晴，“翎翎，这里。”
<br/>
<br/>        叶翎抬眸，迅速走了过来，她伸手捏了捏夏夕绾绝丽的小脸，“啧绾绾，才几日不见，你又惊艳到我了。”
<br/>
<br/>        夏夕绾迅速被逗乐，“彼此彼此啊。”
<br/>
<br/>        叶翎的经纪人金花推着行李箱，她看到很多人拿出手机来拍了，这座四九城的南绾北翎合体，杀伤力太大，她不得不提醒打断这两个美人的商业互捧，“翎翎，夏小姐，太招摇了，我们先上车再聊。”
<br/>
<br/>        “好。”
<br/>
<br/>        夏夕绾和叶翎出了机场大厅，已经有高级保姆车来接了，金花将行李箱放了上去。
<br/>
<br/>        “绾绾，我们上车。”叶翎拉着夏夕绾要上车。
<br/>
<br/>        这时身后传来了陆寒霆低沉磁性的嗓音，“去哪里，我送你们。”
<br/>
<br/>        叶翎转身，伸出小手拿下了脸上挂着的宽大墨镜，露出了明媚似娇阳的小脸，她一双妙目上下打量了陆寒霆一眼，很像是娘家人打量上门的姑爷，然后她勾起菱唇，“陆总，久仰大名。”
<br/>
<br/>        说着叶翎看向金花，“没想到我刚回海城就能沾到我家绾绾的光由陆总专车接送，花姐，这里有陆总当司机，你先将我的行李箱送到醉玉欢。”
<br/>
<br/>        金花点头，“好。”
<br/>
<br/>        陆寒霆绅士的拉开了劳斯莱斯幻影的后车门，“请。”
<br/>
<br/>        夏夕绾上了车，这时叶翎迅速贴过来，暧昧的眨了眨眼，小声道，“绾绾，眼光不错，这位陆总还真是百闻不如一见，成熟禁欲型的。”
<br/>
<br/>        其实夏夕绾不想再上这辆车的，毕竟刚才一个吻将她和陆寒霆都闹得不愉快，现在乍然听到叶翎这颇有深意的“成熟禁欲型”，她雪白的耳垂当即一红，“翎翎，你胡说些什么呢？”
<br/>
<br/>        这时陆寒霆返身上了车，闺蜜之间面红耳斥的话题也就在叶翎挑起的柳叶眉里终结了。
<br/>
<br/>        …..
<br/>
<br/>        劳斯莱斯幻影豪车平稳的向醉玉欢出发，夏夕绾看向身边的叶翎，“翎翎，你大概回来多长时间？”
<br/>
<br/>        叶翎这两年的行程很满，除了影视拍摄还有很多商务活动包括出席奢侈大牌每季度的时尚T秀。
