　　绝望中忽然看到了希望，顿时让叶尘精神大震。
<br/>　　当即一连施展出几次缩地成寸，两人瞬间跨越了数千米的距离。
<br/>　　紧接着，
<br/>　　那道亮光的源头，也终于出现在了叶尘的面前。
<br/>　　然而，
<br/>　　等叶尘定睛一看，顿时傻眼了。
<br/>　　这里的确是洞穴的尽头，可是却压根没有出路！
<br/>　　至于之前那道亮光的发源处，竟然是一颗三米来高的古树！
<br/>　　这颗古树，通体玄光流转，一看就知道不是凡品。
<br/>　　而在那古树之上，还有五颗亮晶晶的果子，在幽暗的洞穴之中，犹如黑夜中明灯，将周围照耀的犹如白昼一般。
<br/>　　“这、这难道是……传说中的扶摇仙果？”
<br/>　　等叶尘认出眼前这个果树的来历，顿时两眼瞪得老大，简直不敢相信自己的眼睛！
<br/>　　要知道，即便是在修真界，扶摇仙果也是最顶级的仙物！
<br/>　　传说此物能生死人，活白骨，哪怕是受了再重的伤，只要还有一丝生机，服用一颗扶摇仙果，也能立即恢复如初。
<br/>　　倘若叶尘之前得到了此物，哪里还需要去找什么神龙之血和七叶回仙草？
<br/>　　只要让两人服用一枚扶摇仙果，便能立即复活，而且还能功力大增！
<br/>　　就在叶尘正惊叹之时，
<br/>　　“吼吼吼！！”
<br/>　　后方再次传来了，五彩神蛤的咆哮之声，这家伙显然已经愤怒到了极点，肚子竟然比之前大了将近一倍！
<br/>　　到了此时，叶尘也终于明白，这畜生为何如此愤怒了，
<br/>　　“这扶摇仙树，要上百年才能结一次果，五彩神蛤显然是打算，用此物来提升自己的修为，难怪会如此紧张！”
<br/>　　想到这里，叶尘身体纵身一跃，便跳到了仙树之上，高高举起手中的紫琼剑，
<br/>　　“五彩神蛤！你若是再敢上前一步，我便毁了这颗仙树！”
<br/>　　根据叶尘的推测，这五彩神蛤少数也有几百岁了，灵智应该不输人类，纵然不懂自己的的语言，应该也能大致猜到他的用意。
<br/>　　果不其然，
<br/>　　那五彩神蛤冲到跟前，正要和叶尘决一死战，看到这一幕，顿时戛然而止，身体定在了当场，不敢再上前一步。
<br/>　　“呱呱！”
<br/>　　五彩神蛤先是叫了两声之后，继而口吐人言，道：
<br/>　　“人类小子！你若是敢毁坏我的仙树，我一定将你碎尸万段！”
<br/>　　这五彩神蛤说的，赫然是修真界的通用语言。
<br/>　　叶尘淡淡一笑，也用相同的语言道：
<br/>　　“好说！我也不想你做过多的纠缠，你先退到千米之外，否则我就一剑毁了你这棵扶摇仙树！”
<br/>　　说着，叶尘做挥剑欲斩之状。
<br/>　　五彩神蛤顿时急了，
<br/>　　“别别！我照你说的做便是，仙树毁不得！”
<br/>　　五彩神蛤显然也是被叶尘给吓住了，当即连连倒退，很快便退到了千米之外。
<br/>　　叶尘见逼退了五彩神蛤，心头微微一松，这才开始低头探查苏蔓的情况。
<br/>　　“蔓姐，你现在感觉怎么样？”
<br/>　　苏蔓此时毒力已经彻底发作，根本听不到叶尘在说什么，身上的衣服，也早已经被她自己撕扯了稀巴烂，如今竟然又开始撕扯叶尘的衣服……
<br/>　　叶尘不由得轻叹了一口气，知道事不宜迟，当即祭出吞天鼎，将两人笼罩其中。
<br/>　　远处的五彩神蛤看到这一幕，顿时往前走了数十米，可是它一时间也不清楚叶尘的意图，踌躇了片刻之后，终究还是没敢上前。
<br/>　　叶尘带着苏蔓，进入吞天鼎的内部空间之后，当即将苏蔓的周身大穴封住，令她动弹不得，然后开始输送真元进入苏蔓的体内，希望可以化解她体内的毒素。
<br/>　　然而很快，叶尘发现那公孙茂给苏蔓吃的，绝非一般的催情媚药，而是极为高级的灵丹！
<br/>　　若是之前在刚服下时，叶尘及时为苏蔓运功驱毒，说不定很轻松就可以驱除。
<br/>　　可是如今，那媚药之毒已经流遍苏蔓的全身上下，此时想要化解，可就千难万难了。
<br/>　　“除非……”
<br/>　　叶尘望了一眼此时，几乎已经全裸的苏蔓，堪称完美的火辣身材，心头顿时升起一道燥热的感觉。
<br/>　　媚药之毒，若没有解药，无法运功化解，那便只有一个办法，通过男女合欢，阴阳交合，方才解决。
<br/>　　而此时，苏蔓虽然全都动弹不得，可是却正媚眼如丝，死死盯着叶尘，美眸中透着一股强烈的欲望之色，口中低声喃喃道：
<br/>　　“小尘，救救我~~我好难受~~”
<br/>　　说实话，有那么一瞬间，叶尘险些就没能把持住。
<br/>　　可是在这道邪恶的念头升起的同时，叶尘脑海之中，又忽然闪现出曦月仙子的身影，浑身顿时一个激灵，便又立即将那道邪念给打消了。
<br/>　　“不行！蔓姐虽然和我没有血缘关系，可也毕竟是我的姐姐啊！”
<br/>　　“而且我心中只有曦月一人，又怎能做出这种事情？”
<br/>　　就在叶尘不知道该如何是好的时候，忽然两眼一亮，顿时一拍脑袋，
<br/>　　“对了！我怎么把这茬给忘了，扶摇仙果不但可以起死回生，增强功力，还能解百毒，区区媚药，应该不再话下！”
<br/>　　“只不过，我一旦动了扶摇仙果，必定彻底会激怒五彩神蛤，到时候后果可就难测了！”
<br/>　　一念及此，叶尘顿时又有些纠结。
<br/>　　然而，
<br/>　　眼看着苏蔓全身上下，已经开始变得炙红，倘若再不解救，只怕就要活活爆体而亡。
<br/>　　叶尘最终一咬牙，
<br/>　　“管不了这么多了，拼了！”
<br/>　　打定主意之后，叶尘当即又收起了吞天鼎，重新回到扶摇仙树之上。
<br/>　　“既然已经选择和这臭蛤蟆撕破脸皮，那这五颗扶摇仙果，我就全收了！”
<br/>　　叶尘一发狠，当即随手一招，仙树枝头的那五颗闪亮如星的仙果，便落入了叶尘的手中。
<br/>　　而远在千米之外的五彩神蛤，看到这一幕，先是微微一滞，继而肚子又变大了一倍有余！
<br/>　　紧接着，
<br/>　　便发出一道惊天动地的怒吼声，
<br/>　　“人类小子！你找死！”
<br/>　　它足足等了一百多年，才等到仙果成熟，竟然就这么被叶尘窃取了，这让它如何不怒？
<br/>　　轰隆隆！
<br/>　　整个万毒窟，都一阵乱颤，犹如随时都要崩塌一般。
<br/>　　而那五彩神蛤，怒吼之后，更是杀气腾腾的朝叶尘扑了过来！
