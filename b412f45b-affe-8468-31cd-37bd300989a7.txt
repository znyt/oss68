　　全宗主的脸色阴沉了下来，原来这天虚宗的琥圣，带领一般天虚宗的长老来此，是因为苏莫杀了天虚宗两名真传弟子。
<br/>　　“当然，就在苍渊星魔渊，苏莫斩杀了我宗武仁和禄冲两名真传弟子！”
<br/>　　琥圣点了点头，继续道：“就连我宗天才弟子扬天成，都差点死在苏莫的手中。”
<br/>　　“不错苏莫灭绝人性，十恶不赦，绝不可饶恕！”
<br/>　　琥圣的身边，扬天成满脸冷冽的说眼，眸中杀机森然，他对于苏莫真是恨之入骨了。
<br/>　　这一次他们来道宗，必须要拿下苏莫。
<br/>　　全宗主闻言，稍稍沉默了下来，随即沉声道：“琥圣，单凭你们的一面之词，本宗主也无法得知具体情况！”
<br/>　　“一面之词？”
<br/>　　琥圣闻言冷笑一声，继续道：“此事不仅扬天成可以作证，还有很多人亲眼目睹，只要出去稍一打听，便能得知情况！”
<br/>　　“全宗主，你可以将苏莫传来，我们当面对质！”扬天成朗声说道，此事证据确凿，很多人亲眼所见，谅苏莫也无法狡辩。
<br/>　　全宗主点了点头，随即立刻命人前去传唤苏莫，不过，最终苏莫并没有来，因为其不在宗内。
<br/>　　“琥圣，苏莫并不在宗内，不如这样吧，你们先行回去，待苏莫回到宗内，本宗主再仔细审问！”全宗主微笑着说道，他知道琥圣等人来者不善，苏莫现在不在宗内倒是好事。
<br/>　　苏莫可是道宗的天才，不论有没有错，全宗主自然不希望苏莫遭到天虚宗之人的毒手。
<br/>　　但是，作为道宗的宗主，他明面上必须要保持公正严明的态度。
<br/>　　“全宗主，我宗的两名真传弟子，不能白死，今日你必须给个说法！”琥圣阴着脸说道，他怎么可能被一句话打发了？
<br/>　　苏莫是天才，超级妖孽的天才，这等妖孽不是自己人，就一定要将其折损。
<br/>　　“全宗主，你们道宗的吕罡，当时也在场，只需将他传来一问，便可得知情况！”扬天成说道。
<br/>　　全宗主闻言点头，随即便命人前去传唤吕罡。
<br/>　　没过多久，吕罡便来到了宗门大殿之中。
<br/>　　瞥见殿中的扬天成，以及一众天虚宗的强者，吕罡心中暗凛，立刻知道了所来何事。
<br/>　　“见过宗主！”吕罡向全宗主恭敬行礼。
<br/>　　“吕罡，听天虚宗之人所言，苏莫杀了两名天虚宗的真传弟子，此时是否为真？”全宗主开门见山的问道。
<br/>　　“是！”吕罡深吸口气，微微点了点头，他没有隐瞒，因为此事根本不可能瞒过。
<br/>　　在魔渊之时，有太多的人看到了，想瞒也瞒不住。
<br/>　　天虚宗众人，闻听吕罡之言，均是面带冷笑了起来，既然吕罡承认了，那接下来就好办了，他们也好向道宗施压了。
<br/>　　“吕罡，将当时的情况，详细道来！”
<br/>　　全宗主沉声说道，言罢，他眸中闪过一道精光，声音幽幽的补充道：“记住——要详细！”
<br/>　　吕罡见此，心中微怔，宗主神情有异，这是什么意思？好像在暗示什么？
<br/>　　吕罡不傻，他心念一转，便知晓了宗主的意思，这是要保苏莫啊！
<br/>　　旋即，吕罡沉吟了少许，道：“宗主，当时苏莫来到魔渊，和天虚宗的真传弟子武仁争夺魔渊的控制权，然后成功的击败了武仁和一众天虚宗的弟子……！”
<br/>　　“等等！”
<br/>　　吕罡的话还未说完，全宗主便抬手打断了其话语，沉声问道：“你说苏莫打败了武仁和一众天虚宗弟子，那苏莫是什么修为？”
<br/>　　全宗主好奇了起来，前些日子的会武之事，传的沸沸扬扬，苏莫好像是武帝境八重修为吧？
<br/>　　这么低的修为，能打败天虚宗真传弟子？
<br/>　　难道苏莫已经突破到了武尊之境，不会这么快吧！
<br/>　　“禀宗主，苏莫是武帝境九重境界！”吕罡道。
<br/>　　“哦？”全宗主闻言，顿时眼眸一亮，武帝境九重修为就能击败天虚宗真传弟子，苏莫的战力居然强到了如此地步？
<br/>　　“苏莫用了多少招，才得以击败武仁？”全宗主好奇的问道，同时心中暗想，苏莫越是妖孽，他就越是要保住苏莫了。
<br/>　　“全宗主！”
<br/>　　这时，琥圣阴着脸开口了，提醒道：“我们现在是询问苏莫的恶行，不是询问苏莫的战力？”
<br/>　　“呵呵，说的也是！”
<br/>　　全宗主闻言微微一笑，摆摆手道：“吕罡，你继续吧！”
<br/>　　吕罡闻言点头，继续道：“苏莫夺得魔渊的控制权后，武仁和一众天虚宗弟子便离开了，但过了几日之后，武仁带着扬天成和禄冲再次回到了魔渊，企图杀死苏莫，但是他们的实力不如苏莫，禄冲和武仁被苏莫反杀……！”
<br/>　　“混账！简直是一派胡言！”
<br/>　　吕罡的话还未说完，再次被打断了，琥圣暴喝一声，长身而起，面带怒气，身上强大的圣威，直接压迫在了吕罡的身上。
<br/>　　呃~
<br/>　　吕罡口中闷哼一声，脸色惨白，全身剧震，身形摇摇晃晃，险些跌倒在地。
<br/>　　“吕罡，你真是信口开河，我们明明是想要夺回魔渊的控制权，苏莫却是心狠手辣，直接下狠手将武仁和禄冲杀死，若不是我逃的快，现在也已经死了！”扬天成怒声大喝，他当然要将罪责全部按在苏莫的身上。
<br/>　　全宗主见此，面色一冷，一挥袖袍，琥圣的圣威便烟消云散，他冷声提醒道：“琥圣，这里是道宗，可不是你天虚宗！”
<br/>　　琥圣的圣威被击溃，吕罡顿时神情一松，全身已经大汉淋漓。
<br/>　　琥圣虽然只是动用了一丝威压，也不是吕罡所能承受的，现在他已经受了一些内伤。
<br/>　　“全宗主，你道宗弟子简直是一派胡言，颠倒黑白！”琥圣目光冷视着全宗主，冷声说道。
<br/>　　“琥圣，你天虚宗弟子所言就是事实？我道宗弟子所言就是颠倒黑白，这是什么道理？”
<br/>　　全宗主冷笑一声，随即又看向吕罡，道：“吕罡，你所言是否为真？”
<br/>　　“禀宗主，弟子所言千真万确，不敢有任何虚言！”吕罡郑重的说道，他知道宗主肯定是要保苏莫，所以，即便惹得天虚宗之人大怒，他也要坚持己见。
<br/>　　所以，他的言语一定要对于苏莫有好处，不然定会惹得宗主不喜。
