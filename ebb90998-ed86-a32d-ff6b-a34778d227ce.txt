　　卫生间地方太小，叶少阳无处可退，也没法器在手，就站着不动，跟他们硬刚，好在他现在实力超凡，虽然这些邪物实力也都可以，但根本不是叶少阳对手，一个个刚出来就被干掉了，一共有十几只，最后一个被杀之后，虚空裂缝也就合上了。
<br/>　　叶少阳总算喘了口气，回头对橙子说道：“没事了。”
<br/>　　“啊！”橙子捂住了双眼。
<br/>　　叶少阳急忙回头看去，却什么都没有，这时橙子羞涩地说道：“老大你快穿衣服！”
<br/>　　叶少阳低头一看，靠，自己刚才打的太激动，浴巾什么时候掉的都不知道，赶紧把衣服抓过来，三下五除二穿上，橙子这才站起来。
<br/>　　四宝在外头咚咚咚敲门，问发生什么事了，叶少阳把门打开，四宝和瓜瓜看到一地的污血——那些邪物死了之后，有肉身的，也很久就化掉了，只留下污血，也在不断地缩小。
<br/>　　“你没事吧？”叶少阳伸手抹掉橙子嘴角的血，关切地问道。
<br/>　　橙子摇摇头，伸手理了理头发，把淋浴打开，对着身上直冲下去，叶少阳看到这一幕，招呼四宝和瓜瓜到外面去。
<br/>　　三人面面相觑，都不知道发生了什么。
<br/>　　很快，橙子穿了浴袍，打开门出来，精神好了很多。他是鲛人，不管受了什么伤——只要不是致命伤，碰到水伤口就会很快好起来。
<br/>　　“那些人，是太阴山的。”面对三人的疑问，橙子简单讲述起来：太阴山出兵征讨阴司，已经横扫了沿途的那些割据势力，来到阴司附近。
<br/>　　橙子受崔天子的命令，去南海给中山王徐达送一封急件，结果回来的路上，被一队赶到的太阴山的兵士围住，这帮人虽然单个实力都不是橙子的对手，但是人多，橙子且战且退，无法突围，自己又受伤了，危机之下，只好给叶少阳发信号，确定了他的位置后，打开虚空，穿越过来。
<br/>　　那些太阴山士兵也立刻跟过来……
<br/>　　叶少阳三人听罢，震惊得一句话也说不出。
<br/>　　“太阴山的军队……居然开进到酆都城附近了？”叶少阳不敢想象这种事会发生。
<br/>　　橙子点头，“太阴山这次进攻跟以往不同，这次是大举进犯，想来他们是做了充足的准备，而且也等不及了。现在大半个鬼域都被他们占领了，酆都城就要被围住了。”
<br/>　　“我的天，还真的发生这种事了……”四宝吓的脸都白了。
<br/>　　“阴司那么多兵呢？”叶少阳问。
<br/>　　橙子道：“在前线与太阴山的兵马大战了一场，败退下来，听说都退到了阴司在北方铸造的那些城里，这些城池也被包围了，目前只有岳家军还在西线战斗，其余都转攻为守了，然后太阴山兵分三路，有两路进攻酆都城，将城池围住了。”
<br/>　　兵临首都城下。叶少阳虽然不懂军事，但也知道这意味着什么，一时间心惊肉跳，一句话说不出来。
<br/>　　突然窗户上出现了一道裂缝，四人立刻警惕，结果钻出来的是萧逸云。
<br/>　　“橙子，你没事吧！”
<br/>　　萧逸云扑上去，把橙子抱在怀里，看了又看。
<br/>　　之后又跟叶少阳三人打了招呼，讲述自己来这里的原因：橙子离开阴司不久，他就得到情报，有一支太阴山的军队往南海去了，立刻要去营救，但南边的山脉被太阴山封锁，闯不过去，后来——在橙子到了人间之后，通过玉牌给他传达讯息，让他感知到在人间的位置，这便立刻赶来了。
<br/>　　被问到阴司现在是什么情况，萧逸云眼中闪过一丝不安，说道：“我觉得，你们还是自己去看看比较好。”
<br/>　　“老大不方便去阴司。”瓜瓜接了一句。
<br/>　　萧逸云苦笑道：“你就放心吧，现在的阴司，绝对没人有工夫管你了，正好我跟橙子要回去，一起吧。”
<br/>　　叶少阳也有此意，于是开坛做法，打开虚空裂缝，五个人一起来到阴司。
<br/>　　降落之后，叶少阳环顾左右，发现自己竟然落在了酆都城的里面，面前是高高的城墙，四边还有跟人间一样的房舍。
<br/>　　这是……枉死城？
<br/>　　萧逸云察觉到了他跟四宝的疑惑，让他们回头，两人回头看去，在他们身后不远的地方，有一根插在地上的长杆，古铜色的金属质地，高高立起，有五层楼那么高，长杆顶端，有大概十几个圆圈，像是小孩子滚的铁环，一圈套着一圈，不断朝着对立的方向旋转着，看上去像是个魔术用品。
<br/>　　在这些金属环的中间，是一抹清澈的黑暗，看上去像是一个巨大的黑洞，就在他们仰头观看的时候，黑洞里突然飞射出几道人影，借着惯性，轻盈地落在地上。
<br/>　　居然是慈心师太和两个小尼姑。
<br/>　　慈心师太起初闭着眼睛，睁眼之后，一眼看到他们，也是当场愣住。
<br/>　　“师太也是听说了阴司的情况，前来参观的吧。”萧逸云上前施礼。
<br/>　　虽然知道萧逸云跟叶少阳是一伙的，但他地位摆着，慈心师太也不敢怠慢，还了礼，说道：“我等正在龙虎山参加大典，突闻阴司剧变，前来探一探情况。为何……为何会落在酆都城中？”
<br/>　　叶少阳也有同样的疑问。
<br/>　　萧逸云解释道：“现在城外已为太阴山占据，诸位如果落在城外，恐怕就直接被太阴山戝众围攻了，不久之前，你们看到这是地藏菩萨的贴身法器涅槃法杖，特意插在这里，被他老人家加持了法阵，通达三界，能印度虚空，凡是从人间过来的生灵，都会为这法杖牵引，落在这里。”
<br/>　　众人恍然。叶少阳转头看去，这才发现附近还有不少当代人间打扮的生灵，猜测都是从人间来的法师，一个个也都神色惊骇，在一队阴兵的带领下，朝酆都城的南门走过去。
<br/>　　以涅槃法杖为中心，在百米开外的空地边缘，站着一队兵士，将整个空地围起来，叶少阳正在打量他们，中间有几人飞身而来，是几个银甲鬼武士，叶少阳一眼看过去，还有几分面熟。
