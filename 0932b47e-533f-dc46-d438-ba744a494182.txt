　　对于龙腾之言，龙城嗤之以鼻。
<br/>　　不仅龙腾不行，龙洌、龙烨、龙双双三人亦是不信，他们认为龙腾在说笑。
<br/>　　龙腾微微笑了笑，并未反驳龙洌之言，事实如何，要不了多久就能出结果了。
<br/>　　迷宫之中，苏莫已经在修炼，仿佛不再参与迷宫中的宝物争夺。
<br/>　　也没有奴仆来到他所在之地，他倒是非常的清静。
<br/>　　而浩大的迷宫城之中，五百多件宝物，也基本上都被人找到了。
<br/>　　这些宝物，有神果、神草、宝器，以及一些特殊的异宝。
<br/>　　每一件宝物都不同凡响，虽然这些宝物对龙族之人无用，但是对人族的武者，或者很多异族而言，均是顶级的至宝。
<br/>　　血腥的争夺，在迷宫城深处不断的上演，不断的有人陨落，不断的有人摆淘汰。
<br/>　　没过多久，二十余万人的迷宫城，剩下的奴仆，已经不足十分之一。
<br/>　　淘汰的人很多，陨落的人同样极多，龙族高层对于这些奴仆，没有什么仁慈之心。
<br/>　　就算是一些陨落奴仆的主人，都不会心疼，毕竟，陨落代表着实力不济，实力不济的奴仆，要之何用。
<br/>　　司空炎很强大，几乎是所向披靡，淘汰了一个又一个奴仆。
<br/>　　不过，司空炎一般都不下杀手，他明白不能太过狠辣，毕竟自己和苏莫身份敏感。
<br/>　　但是，由于宝物有限，司空炎在半个时辰之内，走过了上千条通道，淘汰了五六百人，也只得到一件宝物。
<br/>　　整个迷宫城，均是在外界之人的关注之下，经过一段时间的血腥争夺，一部分厉害的奴仆，也已经凸显了出来。
<br/>　　毫无疑问，司空炎、龙洌的两位女主冰儿和凝雪，龙烨的奴仆石崐，以及龙双双的女奴，都是一等一的强者，基本上没有遇到实力相当的对手。
<br/>　　龙城的奴仆巫铨，虽然败给了司空炎，但是实力还是毋庸置疑的，轻松击败了很多人。
<br/>　　除了这六人之外，还有十几位奴仆，也表现出了超强的实力，非常的耀眼。
<br/>　　至于苏莫，已经被人遗忘在了角落，毕竟，他一直在安静的修炼。
<br/>　　外围高墙之下，苏莫双眸紧闭，如同老僧入定。
<br/>　　这一刻，他突然睁开了眼眸，因为他的身后，出现了一个虚幻之门。
<br/>　　这门户在高墙之上，如同要到一个白色的影子，在缓缓的移动。
<br/>　　“终于出现了！”
<br/>　　苏莫微微的叹了口气，随即长身而起，他要守住出口，就要跟着出口不断的移动。
<br/>　　很快，出口的门户，就离开了这处通道，移向了左侧。
<br/>　　苏莫立刻跟了过去，由于有高墙足够，他必须还要绕一个圈子。
<br/>　　就这样，接下来的时间之中，苏莫就跟着出口的门户移动，有高墙阻挡他就绕过去。
<br/>　　但有时通道就顺着高墙，他便不用绕圈子，顺着通道前进便可。
<br/>　　苏莫这异常的一幕，无可避免的引起了外界所有人的注意。
<br/>　　“那个人是龙腾的奴仆吧？他要做什么？”
<br/>　　“遇到了出口又不出来，在搞什么鬼？”
<br/>　　“此人实力很强吧！”
<br/>　　迷宫城之外，很多龙族之人，都看到了苏莫的情况，纷纷诧异不已。
<br/>　　龙洌、龙城、龙烨和龙双双四人，心中亦是疑惑了起来，毕竟这一幕实在是太奇怪了。
<br/>　　“龙腾，你这个奴仆，该不会是想要守在出口抢夺宝物吧？”龙城想到龙腾之前的话语，惊疑不定的问道。
<br/>　　之前龙腾曾言，这个奴仆会成为最大的赢家，那么此人一直在跟着出口，无外乎是这个意图了。
<br/>　　“他应该是要这么做！”龙腾背负着双手，微微的点了点头。
<br/>　　“哈哈哈！！”龙城闻言，顿时朗声大笑了起来，仿佛是听到了最好笑的笑话一般。
<br/>　　“龙腾，你在开玩笑吧？”龙洌面带愕然之色，他倒是从未往这方面想。
<br/>　　毕竟，此法无异于是和所有奴仆为敌，简直是自寻死路。
<br/>　　龙烨和龙双双也是看着龙腾，心中惊疑不定，若真是这样的话，那龙腾这个奴仆真是太狂妄了。
<br/>　　“我从不开玩笑！”龙腾沉声说道，他也没有隐瞒，因为事实很快就会证明。
<br/>　　“不知所谓！”
<br/>　　龙城闻言，嗤笑一声，满脸戏虐的说道：“龙腾，你这个奴仆难道比那个奴仆炎空还厉害？”
<br/>　　龙城感到有些好笑，这也太猖狂了，就算是以他的实力，也不太敢这么做。
<br/>　　“呵呵，事在人为，一切就看结果吧！”龙腾笑着说道，他也懒得多言，事实会让所有人明白。
<br/>　　半空之中，那两位龙族的中年执事，都不禁多看了苏莫两眼。
<br/>　　不过，他们并没有什么表示，不管用什么手段，只要能抢到宝物，就是在规则之内。
<br/>　　迷宫城之中。
<br/>　　苏莫跟着出口的门户转悠，牢牢的锁定着出口，仅仅是片刻的功夫，便有人来了。
<br/>　　只见一个身材高壮的青年，风驰电逝而来，速度快到极致。
<br/>　　此人是妖族，身高达到九尺一闪，一身体毛极重，脸庞上长满了络腮胡子，身上妖气浓郁。
<br/>　　只不过，此人身上的气息，有些虚浮，明显是受了不轻的伤势。
<br/>　　“找到了！”青年看到出口的门户，顿时大喜过望，他一直在被人追击，找了很久终于找到了出口。
<br/>　　嗖！
<br/>　　青年顿时加速，向出口的门户冲去，但是，光芒一闪，一道身影却是拦在了出口之前。
<br/>　　“宝物交出来，你就可以出去！”苏莫面上带着一丝微笑，声音淡淡的说道。
<br/>　　“找死！”青年怒喝一声，身躯瞬间化为了一只青色的雄狮，足有小象一般大小。
<br/>　　雄狮的速度不仅不减，然而猛然大增，直接向着苏莫冲了过来。
<br/>　　“吼！”
<br/>　　一声狮吼，震动四野，只见从雄狮的空中，发出了一股浩大的音波，粉碎虚空，排山倒海一般击向苏莫。
<br/>　　这一击音波攻击，极为强大，若是一般的同阶武者，身躯和神魂，都会被音波瞬间震碎。
<br/>　　而就算是天才武者，实力不够强大的话，都会被音波重创，轻者神魂受损，重者神魂湮灭。
<br/>　　而且，这种攻击根本无法躲避，只能硬挡。
<br/>　　不过，这位妖族的奴仆，估计做梦都没有想到，他眼前之人，是一位怎样可怕的存在。
