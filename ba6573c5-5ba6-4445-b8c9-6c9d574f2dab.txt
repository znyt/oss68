        &lt;h3&gt;第三卷第二百九十一章  凌辰到西城&lt;/h3&gt;
<br/>
<br/>        凌辰在杭州呆几天，父亲被他给忽悠回去了，老头子虽然糊涂了一次，还是很明事理的，在老头子的心目中，儿子还是很重要。
<br/>
<br/>        他和董亚子在杭州的工作进行得也不是很顺利，因为南方城市对酒业不是很重视，他们有地方产品，凌辰和董亚子考察了几天杭州的市场，便决定要去别的地方，凌辰想去北京，董亚子将眼光放在了西城。
<br/>
<br/>        西城是大西北，那里的人们思想豪放，无酒不成席，无酒不欢，相对来说，市场良莠不齐，而他们的产品也会受到一定程度的桎梏。
<br/>
<br/>        凌辰想去北京的原因是因为宁雅在北京。
<br/>
<br/>        凌辰和董亚子产生了分歧。
<br/>
<br/>        “凌辰，我知道你去北京是因为宁雅，北京这个地方，对于我们的这个行业来说，很少有人敢作假，如果你一心想去北京我也不拦着你，你走你的，我走我的。”董亚子一针见血的指出凌辰的动机。
<br/>
<br/>        “我们能不能先去北京，然后去西城？”其实凌辰也能明白这个道理，只是他很想见见宁雅。
<br/>
<br/>        “不行，马上离三一五愈来愈近了，而且刘诚之还需要我们配合他，你不能只想着情情爱爱，不顾大局。”董亚子严厉拒绝了。
<br/>
<br/>        “我哪有整天情情爱爱。西城就西城，我现在马上就定机票，飞西城。”凌辰明白事情的重要性，虽然心里不舍，工作还是很重要，没有工作他有什么资格站在他的父亲面前说离家出走？没有工作他那什么来对宁雅说：“你的一切决定我都会支持你！”
<br/>
<br/>        “男人真可怕，一旦动起情来还真能六亲不认。”董亚子摇了摇头，要不是她意志力坚定，就会着了凌辰的道，跟着凌辰到处瞎跑。
<br/>
<br/>        “你不要想着我是瞎跑，这两天我拿到的数据比你多，我们一走，杭州就会有行动，不信你看着。”凌辰看着董亚子那一脸不可思议的样子说。
<br/>
<br/>        “什么行动？”董亚子不信。
<br/>
<br/>        “你等着看就是！”凌辰冷冷的说。
<br/>
<br/>        董亚子心系刘诚之，对凌辰说的话也没有在意，她需要给刘诚之创造机会，也不知道刘诚之在甸城的情况如何，刘诚之不主动联系她，她也不能主动联系刘诚之，分离的痛苦撕扯着董亚子的心，焦虑让董亚子有些暴躁。
<br/>
<br/>        杭州的行程没有多大的效果，董亚子需要锲机，需要证据。
<br/>
<br/>        凌辰看着董亚子焦虑的模样对董亚子说：“亚子，诚之比你我想象的都聪明，他发的邮件你仔细品一品，你细细品，便知道意思，关心则乱，你的心已经乱了，不利于判断，也不利于我们的工作。”
<br/>
<br/>        “细品？”董亚子反问。
<br/>
<br/>        “是啊，你好好看看，有兽有鱼，不想回来，而且邮件是发给董总的，不是发给我的。”凌辰微笑的看着董亚子。
<br/>
<br/>        “我明白了，然后你将邮件转发给何少华，让何少华以为你觉得他挖了你的墙角，转而更加会利用或者是拉拢诚之，这样诚之在甸城将会站稳脚跟。”董亚子略一沉思便明白了其中的道理。
<br/>
<br/>        “聪明的亚子只是被爱情蒙蔽了心智，如凌辰一样被爱情冲昏了头脑，是应该彼此原谅的。”凌辰乐呵呵的说，爱情的甜蜜让凌辰不再是哪个冷冰冰的凌辰，而是充满了温情。
<br/>
<br/>        “嘶，油嘴滑舌，好了，原谅你被爱情一时冲昏了头脑。”董亚子心里一松，只要刘诚之安全就好。
<br/>
<br/>        凌辰和董亚子飞到了西城，在他们阴差阳错的选择中，他们无形中参与了一场前所未有的斗争中，一场邪于正，善于恶的较量中，他们在这一次的西城之旅中和陈子昂绑在了一起。
<br/>
<br/>        凌辰和董亚子到达西城，在下飞机的时候，凌辰给宁雅打了个电话。
<br/>
<br/>        宁雅刚刚从某工商部门出来，他们的日期提前了，要去报备，一系列的手续都得提前。
<br/>
<br/>        她看着凌辰的电话，有一丝恍惚，这些天真的是太忙了，自己的事情，公司的事情，每次回家都很晚，洗洗倒头就睡，都让她忘了凌辰。
<br/>
<br/>        “凌辰，你在哪里？”
<br/>
<br/>        “我在西城，原本要先去北京的，但是董亚子认为西城的事情比较重要，而且她威胁我，如果我选择去北京，那我的总代就不要拿了，我想了想，如果我不能拿到总代，我也就没有资格站在你的面前，给你想给你的一切。”凌辰将不能去北京的锅都摔给了董亚子，听得董亚子在旁边只翻白眼，她见识到了凌辰为了美人无耻的地步，难怪古代的男人都为了美人而不要江山。
<br/>
<br/>        董亚子想骂人有没有？
<br/>
<br/>        “我这几天也很忙，你要来北京我估计都没有时间陪你，你好好工作吧，我们来日方长，对了陈子昂也在西城，你们可以联系一下，见见面。”宁雅边走便打电话。
<br/>
<br/>        “我就不联系了吧，我和她也没有太多的关系，要不是你，我也不认识她是谁。”凌辰在电话里说。
<br/>
<br/>        “你去见一见，她的父亲做手术，你就算是替我去的，然后给我打电话汇报一下情况。”宁雅有点不满凌辰的态度，现在的陈子昂就是宁雅的天，要不是工作的原因，宁雅是会飞到西城去陪陈子昂的。
<br/>
<br/>        “好，老婆大人吩咐，我一定办到！”凌辰听了宁雅的吩咐，立马将自己的身份顺杆爬，升了一级。
<br/>
<br/>        “嘚瑟！”宁雅在电话里轻笑一声。
<br/>
<br/>        “那我现在就给陈子昂打电话？”凌辰的心情也很好。
<br/>
<br/>        “行，你打吧，我这里还有事情，先挂了，拜拜。对了，注意安全啊。”宁雅挂了电话。
<br/>
<br/>        “小雅说陈子昂在西城，她的父亲生病了，让我替她去问候陈子昂。你怎么办？”凌辰看着身边的董亚子。
<br/>
<br/>        “我也一起去啊，难不成就你能去问候，我就不能去问候了？”董亚子嫌弃的看着凌辰。恋爱中的男人智商果然低，凌辰都忘记了董亚子其实也是认识陈子昂的，不但认识而且对陈子昂充满了好感。
<br/>
<br/>        凌辰给陈子昂打电话的时候，陈子昂刚好从陈子悦的家里出来。
<br/>
<br/>        “陈总，听小雅说你在西城，伯伯做手术了，她让我来替她看看伯伯。”凌辰对陈子昂说。
<br/>
<br/>        “谢谢凌总，心意到了就行，我父亲才做完手术，不适宜探望，晚上我们一起坐一坐，我请你吃饭。”陈子昂心里一直惦记这凌辰出手相助的事情，凌辰来到西城，她理应请凌辰吃一顿饭。
<br/>
<br/>        “吃饭？”凌辰看了一眼董亚子，董亚子兴奋的冲凌辰点点头。
<br/>
<br/>        “你们住的地方定好了吗？”陈子昂想着凌辰出差不可能是一个人，便用了“你们”两个字。
<br/>
<br/>        “我们已经定好了，你不用管了。”
<br/>
<br/>        “那好你一会将住的地方发给我，我等会让人去接你们。”
<br/>
<br/>        人生有很多的不可预知，在冥冥之中，他们都在漩涡的边缘挣扎着，当风卷起浪潮，他们都跌进了漩涡里，只有共进退，才能不被吞噬。
