　　魏征说的这些李承乾明白，李二也明白，但一个皇帝的想法和一个臣子的想法，从出发点上来说，是完全不同的。
<br/>　　就像找媳妇这样的事情，放在李恪身上和放在李承乾身上有不同的结局一样。
<br/>　　李恪看好谁了，可以直接纳回来作小妾，但是同样的事情放在李承乾身上就不行，不管他看好谁了，他都要经过长孙和李二的同意，否则那就是忤逆。
<br/>　　高句丽的事情也是一样，在魏征看来，这是一点小事，忍忍也就算了，大丈夫报仇十年不晚。
<br/>　　但在李二看来这就是对方不给自己面子，如果这一次忍了，那其他国家如果也这样怎么办？大唐的脸面还要不要？他这个皇帝的脸面还要不要？
<br/>　　李承乾很清楚老头子的为人与想法，而且站在他的立场上，李二既是他老子，又是他的君主，所以无论从哪一方面讲，他都应该站在老头子的一侧，否则就是不孝、不忠！
<br/>　　所以在魏征说完他的观点之后，李承乾便接过了话头：“魏伯伯言重了，父皇的意思只是想让高句丽受到教训而以，并不是要屠其国、灭其种。”
<br/>　　“唔，不错，朕就是这个意思！”正在喝茶的老李连连点头，放下手中杯子。
<br/>　　李二现在只想出兵，至于要打到什么程度，那要打打再说，如果高句丽怂的一逼，灭其国也并非是什么不可能的事情。
<br/>　　“太子殿下！”对李家父子这种打马虎眼的作法，魏老头打心眼里腻味，觉得还是前两天那个小胖子可爱些。
<br/>　　一直坐而不语的杜如晦突然开口说道：“陛下，太子殿下，战阵之上兵凶战危，战与不战怕是非我大唐能够决定的。”
<br/>　　老杜的发言让李承乾颇为意外，不由将注意力移了过去。
<br/>　　只听杜如晦接着说道：“陛下，高句丽乃是苦寒之地，冬长而夏短，冬日里天气异常寒冷，我大唐军士并不适应，是以不耐久战；而到了夏天，该国又会成为一片泽国，大型攻城军械又运不进去。是以，若要东征，还要作万全的准备才是。”
<br/>　　李二咂了咂嘴，将目光看向李承乾。
<br/>　　刚刚这些对话他已经和杜如晦等人讨论过无数次，早就没了新意，现在说来，无非就是说给李承乾听的。
<br/>　　果然，李承乾没有辜负李二的期望，并没有被杜如晦所说的困难吓倒，而是将话题引到了兵卒的战斗力上：“杜伯伯可知高句丽兵卒战力如何？”
<br/>　　杜如晦见李承乾竟然还不听劝，不由加重了语气：“高句丽民风彪悍，可谓全民皆兵。”
<br/>　　李承乾满不在乎的嘿嘿一笑：“杜伯伯言过其实了，一千只羊再厉害也打不过一头强壮的老虎，全民皆兵也不等于战力无敌，否则他们也不会每年被掳走那么多边民，而无计可施。”
<br/>　　魏征老头见李承乾也和李二一个调调，而且更加难缠，不由一声长叹抢过话头：“太子殿下，隋末杨广三征高句丽而不成，前车之鉴就在眼前，殿下因何执迷不悟啊！”
<br/>　　老魏话音刚落，房间中陡然一静，所有人目光都看向桌案之后的李二。
<br/>　　老魏头儿这是犯忌了啊，拿杨广来作例子，说李承乾执迷不悟，可小李同志分明就是李二的代言人，说他执迷不悟，无异于说李二执迷不悟，这可就等于在拿老李和杨广作比较了。
<br/>　　李承乾敢感觉到了气氛的诡异，看着老头子越来越阴沉脸色，似乎处在马上就要爆发的连缘，估计下一秒很可能就会把老魏头儿给弄出去砍了。
<br/>　　心念电转之间从椅子上起身，立于房间之中，将众人的目吸引到自己身上，然后朗声说道：“父皇，诸位叔伯，既然在担心举世攻唐，为何就没有想过来个举世攻‘高’？想那高句丽与百济、新罗乃是世仇，若大唐发兵，百济与新罗必然从其背部发起猛攻。”
<br/>　　“而我们只要对联系契丹部族以及北方的黑水部族，给他们一些好处，利益趋使之下，这两部必然响应，到时候五方联动四面围之，高句丽纵然举国皆兵又能如何？”
<br/>　　没办法了，不管有没有道理，李承乾胡乱编一些理由，想把老头子的注意力分散一下，别让老李一怒之下砸了自己的‘镜子’。
<br/>　　老李书房里坐的都是些人精，哪个不是老狐狸一样的人物，李承乾的话音刚落，立刻房玄龄就接了上来：“殿下如何知道百济、新罗必然出兵？若是他们不出兵又如何？”
<br/>　　李承乾洒然一笑，傲然说道：“出不出兵就要看大唐的决心如何了，若我大唐提四十万大军陈兵边境，只怕不等我们先动手，百济、新罗两国就要先动了。”
<br/>　　吹牛逼呗，谁不会啊，可行不可行先不说，听着提气，能把老李的注意力引开就行。
<br/>　　“行了行了，别在这儿演戏给朕看，朕还没那么小气，不至于连句话都听不进去！”李二无奈的拿着桌上的玉石笔架敲了敲，打断李承乾等人的吹牛抬杠行为。
<br/>　　瞬间，书房中响起一片吁气之声，老杜和长孙无忌等人也在举袖擦拭着自己额角冒出来的冷汗，不管如何，魏老头儿这条命应该是保住了。
<br/>　　“臣，谢过陛下不责之恩！”李二表示不怪罪，但老魏不能就这么心安理得的受了，谢个恩施个礼也是题中应有之意。
<br/>　　“免了！”既然不想杀老魏，李二也就好人作到底，让老魏坐回原位。这种耿直的家伙，杀了只会成全他的名声，但对李二除了能泄愤没有一点用处，弄不好还要背一个暴君的恶名，智者所不为也。
<br/>　　只是免了老魏的责罚之后，老李却把目光看向了李承乾，啧了一声，又扫了一圈长孙无忌等人：“刚刚太子说的计策朕认为不错，你等如何看？”
<br/>　　嗯？！不错？李承乾一愣，诧异的看向老李，刚刚那些都是临时编出来吹牛逼败火用的，怎么在老头子嘴里竟然变成一个不错的计策了？不会是说反了吧？
