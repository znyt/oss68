　　家族大比的规则很简单，依据修为强弱上场。
<br/>　　率先上场的肯定是凡境一重的子弟，裁判会主动点名，点名后，子弟便上场接受检测，而后与同等级的其他子弟战斗。
<br/>　　直到一重的子弟比罢，一重中最强者应战二重。若该弟子败，则进入下一轮，也就是凡境二重子弟间的比武。
<br/>　　而后依次类推。
<br/>　　在裁判高亢的声音下，一个个家族子弟被点名上场。
<br/>　　数个小时后，已经进行到凡境五重子弟间的比斗。
<br/>　　凡境五重以下的子弟虽然占多数，但毕竟他们等级低，战斗很快便结束了。
<br/>　　不过，在这些比赛中，倒是有数名凡境四重的子弟，竟然越级打败了凡境五重，战斗经常至极，让台下的不少族人纷纷叫好。
<br/>　　至于台下的萧逸，看着数十场比赛过去，差点没困得睡着。
<br/>　　实在是对于他这个曾经的杀手界王者来说，这些小屁孩的战斗毫无营养，一招一式中尽是破绽，你来我往地随便打一场，像小孩子过家家似的，太无聊了。
<br/>　　虽然说这个世界的武者拥有完整的修炼体系，真气也很强，但战斗经验着实是差，招式也太过简陋，着实是入不了眼。
<br/>　　正当他真的差不多要睡着时，比武场上，忽然传来了一声巨响。
<br/>　　而在这巨响之下，一个家族子弟直接飞落比武台，口吐鲜血，晕厥了过去。
<br/>　　“嗯？”萧逸皱了皱眉头，看向台上。
<br/>　　“好大的力气，竟然一拳将同境界的武者打败。萧星扬，橙阶武魂，蛮火牛，果然厉害。”台下的族人纷纷赞叹一声。
<br/>　　萧星扬，九长老之子，在萧家也算小有名气的天才，今年不过15岁，已是凡境五重武者。
<br/>　　在凡境五重的子弟中，他当属最强。
<br/>　　“此战，萧星扬胜。”裁判宣布了结果。
<br/>　　萧星扬得意地举起了双手，环视了比武台四周一眼，仿佛在享受族人们炽热的目光。
<br/>　　但不知为何，单他的目光扫到萧逸身上时，却忽然满脸不屑，眼中尽是挑衅之意，更是不着痕迹地伸出了尾指，对着萧逸摆了摆。
<br/>　　“这家伙，对我有敌意。”萧逸有些疑惑。
<br/>　　.....
<br/>　　不一会儿，下一轮比武开始。这次，是属于凡境六重间的战斗。
<br/>　　凡境六重，代表着萧家如今最出色的一批子弟，同时也是族人们最上心、最看好的子弟。
<br/>　　对比之前的战斗，接下来的战斗无疑更受瞩目，也更为重要。
<br/>　　毕竟，今年的家族大比，恰逢赶上了三年一次的开启紫云洞。
<br/>　　凡境六重的子弟们，虽说肯定都能进入紫云洞，但他们的名次，却也决定了他们在紫云洞中的位置。
<br/>　　据说，紫云洞是一个修炼的宝地，位置越是深入，修炼速度也会越快。
<br/>　　所以，接下来的战斗，这些子弟们必定会拼了命地使出浑身解数，力求得到更好的排名。自然的，他们的比武，也必将更加精彩激烈。
<br/>　　“萧子木、萧壮，上台接受检测。”裁判点名道。
<br/>　　两个年龄约莫17岁的年轻人跳上比武台。
<br/>　　裁判打量了他们一眼，满意地点了点头，道，“嗯，不错，都是凡境六重，特别是萧子木你，应该要不了半年就能突破到凡境七重，很好。”
<br/>　　接受完检测，两人开始了比武。
<br/>　　一瞬间，两人都施展了武魂。
<br/>　　萧子木的武魂是橙阶火旋棍，萧壮的武魂是橙阶赤眼红牛。
<br/>　　两人的战斗果真比之前经常了太多，萧子木一棍之下，带着火焰旋转的棍圈，顷刻间将比武台打得火花四溅。
<br/>　　萧壮更是一拳轰出，那被炽热火焰包裹的拳头，极具冲击力和爆发力，一下子将地面震得粉碎。
<br/>　　老实说，在萧逸眼中，他们的战斗仍旧是不入流的。起码，如果自己在同一个境界，施展同样的武魂，爆发出的战力，绝对数倍于他们。
<br/>　　论战斗经验和战斗技艺，萧逸太强了。
<br/>　　不过，毕竟这两人打得那么激烈，极具观赏性，萧逸也权当看表演了。
<br/>　　正当他看得津津有味时，肩膀却被人重重地拍了一下。
<br/>　　“嗯？”萧逸条件反射，一个跨步，以极为精妙的步伐脱离了那人的手。
<br/>　　什么人？以我的敏锐直觉，竟然察觉不到他走到我身后？
<br/>　　当萧逸转过身时，才发现，来人是三长老，顿时松了口气。
<br/>　　这就是先天境武者吗，竟然能做到无声无息地收敛气息，步伐轻盈，却又稳如泰山，这几乎比得上当年自己在地球时的巅峰实力了。
<br/>　　传言，先天武者拥有极恐怖的战力，一拳一脚下，足以开山裂石。
<br/>　　“逸儿，你过来。”三长老低声说了一句，将萧逸拉到角落。
<br/>　　“怎么了？”萧逸问道。
<br/>　　“逸儿，凡境六重这个等级的战斗，你也看到了。”萧重语重心长地说道，“以你的实力，上去和他们比武，跟送死没什么区别。”
<br/>　　“三长老的意思是？”萧逸问道。
<br/>　　“放弃这次的比赛。”萧重沉声道。
<br/>　　萧重叹了口气，眉宇间尽是对萧逸的浓浓关爱，道，“逸儿你放心，就算你这次不参加比赛，五长老他们也奈何你不得。少家主的头衔仍旧保留，紫云洞的修炼名额也依旧给你。”
<br/>　　“哦？”萧逸很是疑惑。
<br/>　　萧重笑了笑，慈爱地说道，“就像你之前所说的，这本来就是你应得的东西。”
<br/>　　“可是....”
<br/>　　“放心。”萧重打断道，“四长老、六长老表示愿意支持你，加上我，我们三个长老的决定，单凭五长老他们还无法改变。”
<br/>　　“三长老，我不是这个意思。”萧逸苦笑了一声，他当然明白三长老的用苦良心，就是怕自己上场比武会有危险嘛。
<br/>　　而且，四、六长老愿意支持自己，恐怕三长老也付出了不少的代价。难怪这半个月来都没有见到三长老来找自己，原来他在暗中准备着这件事。
<br/>　　但，萧逸自有把握应付这次的比武。
<br/>　　“三长老，我和萧若寒还有赌约，这次的比武，我非上不可。”萧逸认真地说道。
<br/>　　萧重立马否定道，“不行，赌约就让他过去吧，我不可能眼睁睁看着你受伤冒险。”
<br/>　　萧重继续说道，“据我所知，萧若寒已经向好几个家族子弟施压了，你一旦上场，那些子弟会对你下死手的。”
<br/>　　“死手？”萧逸冷冷一笑，道，“正好，那么我也不用对他们客气了。”
<br/>　　萧重气道，“还对别人不客气，别人不收拾你就不错了。听话逸儿，这次比武你不能上。”
<br/>　　“放心吧三长老。”萧逸自信一笑，道，“萧家子弟，自当遇事而刚，遇危而不惧。好男儿，岂能畏畏缩缩，言而无信，贪生怕死呢。当年我儿时，父亲也是这样教导我的吧。”
<br/>　　“额，家主他...”萧重一时哑口无言。
<br/>　　萧逸淡淡地看了比武台一眼，认真地说道，“男人间的约定，是很重要的吧。如果我怯战不出，会很没用，很让人瞧不起的吧。”
<br/>　　三长老萧重愣在了原地，很多年以前，萧家曾经经历过一次危机，那时，萧逸的父亲，也就是家主，也是这般毅然，没有任何退缩，孤身一人，以八尺之躯，抵挡了住所有敌人。
<br/>　　直到比武台上一轮轮的比武结束，萧重还愣着，回忆着当年跟随家主的峥嵘日子。
<br/>　　而这时，萧逸已经主动向比武台走去，一个箭步跳了上去。
<br/>　　“萧逸？”裁判愣了愣，有些惊讶。
<br/>　　印象中，萧逸从来没有参加过家族大比。因为萧逸那万年垫底的凡境一重实力，还有废武魂，根本不是任何一个弟子的对手。
<br/>　　不过他很快想起来，半月前，萧逸与萧若寒的打赌。
<br/>　　而萧逸的出现，下方的族人们，当即变得喧闹起来。
<br/>　　“萧逸竟然真的敢上场。”
<br/>　　“他是要应战上一轮凡境六重的弟子吧，这跟找死有什么区别？”
<br/>　　“那个废物，区区一个凡境一重，还妄图和家族内顶尖的子弟战斗，真是不自量力。”
<br/>　　“按我说，直接废了他的少家主之位好了，至于比武什么的，完全是浪费时间吧。”
<br/>　　族人们的谈话间，几乎所有都是轻蔑与嘲讽，更多人甚至是觉得在浪费时间，连观看战斗的欲望都没有。
<br/>　　萧逸，一个废物，萧家的耻辱，这几乎成了整个萧家所有人的共识。
<br/>　　“小废物果然脑门一热自己来送死了。”五长老等人阴鸷地笑着。
<br/>　　“萧逸，跟我打赌，今天我要彻底废了你。”萧若寒心头对萧逸的杀意极为浓郁。
<br/>　　“萧逸，下去吧，我们没有兴趣与你对战。”已经分出名次的凡境六重子弟们不屑地说着，眉宇中尽是厌恶之色。
<br/>　　仿佛，对于他们这些顶尖弟子来说，和萧逸比武，哪怕赢了，也是一件非常耻辱的事。
<br/>　　弱者，永远是强者眼中的垃圾，口中的笑柄。
<br/>　　一声声嘲笑，一声声不耐烦，一声声让他下台，萧逸充耳不闻。孤身站于比武台上，冷傲的双眸，扫视了一眼族人们，最后定格在凡境六重的子弟们上。
<br/>　　“你们，谁敢上台一战！”萧逸一声大喝，战意冲天。仿佛一个孤傲的王者，不屑地看着那一幅幅丑陋的嘴脸。
<br/>(本章完)
