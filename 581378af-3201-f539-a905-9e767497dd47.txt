他陆寒霆浸淫商场这么多年，十几岁就开始玩转商界，对敌人从未心慈手软，最擅长的就是进攻，他从来没有遇到等着敌人来挑衅，而他束手被打的境地。
<br/>
<br/>        在她盈亮明媚的目光里，陆寒霆搂着她莹润的肩头将她紧紧的按在自己的怀里，然后轻声道，“恩，不介意，你去吧。”
<br/>
<br/>        他想，他终究还是不能对她做出些什么，比如折了她的羽翼将她禁锢在自己的身边。
<br/>
<br/>        他可以为了她站在原地，什么都不做，他也可以为了她忍着敌人的一再挑衅，一退再退。
<br/>
<br/>        “陆先生，谢谢你啊。”夏夕绾甜甜的道谢。
<br/>
<br/>        陆寒霆将薄唇勾出了一道魅惑的弧线，“陆太太，你不是真心感谢我，如果你是真心的，不应该只是用嘴巴说说。”
<br/>
<br/>        “…”夏夕绾迅速推开他，还沾了一点泡泡擦到了他的俊颊上，转身就跑。
<br/>
<br/>        陆寒霆扣住她的纤腰将她扯了回来，俯身就往她的红唇上吻去，“陆太太，好久没收拾你了是吧，又欠了。”
<br/>
<br/>        夏夕绾在他怀里咯咯的笑，脆铃一样，还躲着他的亲吻，“不要，我手还没有洗，还有泡泡…”
<br/>
<br/>        “待会儿再洗，反正都是要洗手的。”
<br/>
<br/>        夏夕绾被他吻住了红唇，两个人从沐浴间一路吻到了卧室，脑袋晕乎乎的时候就被他推倒在了柔软的大床上…
<br/>
<br/>        ……
<br/>
<br/>        两个人闹了好长时间，夏夕绾很快就睡着了，迷迷糊糊的时候她感觉有人又在亲她，纤长的羽捷颤了颤，她一双惺忪的睡眸就睁开了。
<br/>
<br/>        现在已经很晚了，不知道几点了，卧室里打着昏黄柔和的灯光，她身上很重，被压着…
<br/>
<br/>        夏夕绾抬手遮了一下眼，然后柔白的手指穿梭进了男人碎短的发里，微微用力将他给扯了上来，“你怎么还不睡啊？”
<br/>
<br/>        陆寒霆看着女孩初醒的模样，那双眸子懵懵的清澈干净的转，清纯到不行，但又被他染上了几分女人青涩的软媚，让他移不开目光。
<br/>
<br/>        他嗓音低哑的呢喃道，“把你吵醒了？你那份牛鞭汤让我睡不着。”
<br/>
<br/>        夏夕绾这才感觉到他身上的温度已经发烫了，她好像…洗手没多久吧…
<br/>
<br/>        “要不，我给你扎一针吧？”
<br/>
<br/>        陆寒霆被气笑了，两手撑在她的身侧居高临下的看着她，“绾绾，我可是你老公，你除了学医外，没学会两性吗，我要的不是针，而是你。”
<br/>
<br/>        说完，陆寒霆掀开被子下床，去沐浴间里冲冷水澡了。
<br/>
<br/>        夏夕绾躺在床上闭了一下眼，睡意已经被他搞得全无了，她坐起身，肩头的黑色衬衫滑落下来，里面的肌肤被种的全是草莓。
<br/>
<br/>        夏夕绾捂住了自己的领口，伸手摸了一下自己发红的小脸，她也不知道自己发什么神经给他买牛鞭，现在已经凌晨一点了，两个人貌似要没完没了的折腾，最后买单的还是她！
