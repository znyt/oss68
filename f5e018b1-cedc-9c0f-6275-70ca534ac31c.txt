        上车后，傅夜沉问清她的住处时，俊眉微微地皱起。
<br/>
<br/>        只见坐在他副驾驶座上的这个女孩，脸色有些苍白，额头上有细密的汗珠渗出。
<br/>
<br/>        傅夜沉并未多想，而是直接发动了车子。
<br/>
<br/>        车子往她的住处飞驰而去，十五分钟过后，停在窄小的巷子里。
<br/>
<br/>        “谢谢！”落以柔轻声道谢，连忙下车，脚踝的痛楚袭上心坎，纤弱的身子跟跄地震了一下。
<br/>
<br/>        车门关上后，她深吸一口气，脚踝的痛楚愈来愈剧烈，小心地退后数步，她颔首表示谢意，目送他的车子滑行而去。
<br/>
<br/>        “呃……”
<br/>
<br/>        落以柔忍不住地蹲下身去，吃痛地轻喘着，低头才发觉自己脚踝处肿得十分惊人，难怪会痛成这般。
<br/>
<br/>        当她紧合着湿润的眸，咬着牙打算要站起身子时，身侧传来紧急的刹车声，然后是沉重的脚步声……
<br/>
<br/>        “你在做什么？”傅夜沉那霸道的口吻隐含着疑惑不解。
<br/>
<br/>        落以柔不自然地半拱着背，诧异地睁开星眸，望入一双冷凛的幽深黑瞳中，纳闷他何以去而复返？
<br/>
<br/>        “我问你在做什么？”
<br/>
<br/>        傅夜沉双手抱胸，半侧着身子倚靠在车门边，淡漠的态度似乎在问一件无关紧要的事。
<br/>
<br/>        “我没事……”
<br/>
<br/>        她的语气虚弱，清澈的眸光逃避似的躲开他逼人的视线，然后倏地站直身子，冷不防传来的痛楚让她颤抖了一下，秀眉蹙起。
<br/>
<br/>        锐利的鹰眼微眯，路灯黯淡的无人街道上，黑瞳分外清亮，像紫外线般地扫视着她，目光从上往下挪动。
<br/>
<br/>        她的左脚有些不自然的弯曲……
<br/>
<br/>        方才他无意识地瞥上后视镜，发现她吃痛蹲在街道上的影子，于是掉转车头回来一探究竟……
<br/>
<br/>        “你住哪栋？我送你上去。”傅夜沉接着说道，一双健臂从容的半扶着她。
<br/>
<br/>        “我……”落以柔迎上他那双漆黑的眼眸，脸颊微微泛红地支吾了一声，“就在、在前面那栋的第一个单元。”
<br/>
<br/>        傅夜沉半扶着落以柔走上陈旧的楼房，阴暗的光线、沉闷的空气让俊眉不自觉地皱起，从容地扶她到三楼的加盖铁皮屋，走至左侧第一间的门口。
<br/>
<br/>        低沉如警告的话语从上传来：“钥匙给我。”
<br/>
<br/>        落以柔乖乖地掏出皮包里的钥匙，二话不说就直接递给了傅夜沉。
<br/>
<br/>        傅夜沉拿过钥匙，打开门后，将她推入房内，随着她的脚步也踏进那小小的斗室。
<br/>
<br/>        目光一扫，已把小小斗室内的一切看透。
<br/>
<br/>        地上铺着苹果绿的塑料地板，两层的单人床，明显可以看出她睡在下铺，上铺堆满一箱箱的物品。书桌上摆满整齐有序的书籍，还有衣橱、一张小小的和室桌、小浴室、阳台改建的厨房。
<br/>
<br/>        “去那边坐着，把鞋脱了。”傅夜沉又命令道。
<br/>
<br/>        面对这男人的强势，落以柔百分百服从，乖乖地落座，脱了鞋。
<br/>
<br/>        傅夜沉走到她跟前，蹲下身去，动手替她推拿受伤的脚踝。
<br/>
<br/>        在他推拿之际，落以柔痛得直喘气，脸庞沁出汗珠，双手紧扳着椅背，泛出青白的色泽。
<br/>
<br/>        “咯吱”一声，当傅夜沉推拿的手劲蓦地加重的那一刻，落以柔痛得眼泪直流。
<br/>
<br/>        “好了，已经复位了。明天，你可以去药店买瓶药油，再擦一擦，就没事了。”傅夜沉一边说着，一边站起身来。
<br/>
<br/>        同样是脚踝扭伤，他对待这个女孩的态度与对待筱颖的态度，截然不同。
<br/>
<br/>        那次在傅家大庄园里，筱颖扭伤了脚，他心痛不已，替筱颖推拿复位后，还不忘温柔地拭去筱颖的眼泪哄筱颖开心。
<br/>
<br/>        所以，傅夜沉一清二楚，筱颖在他心底，有着其他人无法取代的位置。
<br/>
<br/>        “大哥，谢谢你。”落以柔搓揉着双手，不自在地低喃，“大哥，时间太晚了，你看起来似乎很疲惫，你要不要在这儿休息一会儿再走？”
<br/>
<br/>        其实，起初她坐他的车的时候，就发现他开车时已经是哈欠连连。
<br/>
<br/>        傅夜沉下意识地低头，目光瞥视腕上的表，竟然半夜一点多了。
<br/>
<br/>        他今天确实很疲惫，从这里开车回私宅还要开一个多小时。
<br/>
<br/>        疲劳驾驶，对他很不利。
<br/>
<br/>        “我坐一会儿，你先睡！”傅夜沉往一旁的小沙发上坐去，有些头疼地揉了揉太阳穴。
<br/>
<br/>        落以柔点了点头，垂下脸，不敢吭声，走至衣橱取出睡衣，步履艰难地往小浴室前进……
<br/>
<br/>        翌日，生理时钟仍在七点整准时敲醒了落以柔。
<br/>
<br/>        落以柔从床上爬起来后，才发现那男人就这样一手优雅地支撑着脑袋倚着她的小沙发睡了一晚。
