　　叶少阳道：“这怎么说来着？”
<br/>　　“很简单的道理，如果这座墓是元朝以前的，那自然跟赵禥没关系，如果是明朝或者大清的墓，那也不可能……只有在元朝时期，赵家还算是前朝正统，那些南宋的遗臣才会想到复国，到了大明，蒙古人已经被汉人赶走了，还要宋朝的皇室血脉有什么用？”
<br/>　　宋晓宇也点了一盏油灯，照着那个缺口说道：“这个洞是哪来的？”
<br/>　　妙心环顾着洞穴四周，说道，“我怀疑外面这个洞穴，是山体变化产生的，之后可能有老鼠或穿山甲一类的东西，感觉到里面凉爽，在这里打洞，将墓道打穿……”
<br/>　　毛小方一听，忍不住插嘴说道：“穿山甲能打出这么大一个洞？”
<br/>　　妙心白了他一眼道：“听我说完，古墓之下，阴阳二气分隔，只要不接触阳气，就算是有邪物，也是在沉睡之中，不会惊蛰而变。但这里有了个洞，虽然阴气外泄，但阳气也会进入，里面的僵尸一类的邪物，感受到阳气的缩在，苏醒之后，会一直寻找源头，很可能是找到这里之后，想要吸取更多的阳气，所以把这个洞给扒得更大了。”
<br/>　　虽然是猜测，但是在场几个人都觉得这分析得合情合理。
<br/>　　妙心从卢晓清手中拿过三色莲花灯，蹲在那个墓道上的小洞旁边，从包里拿出毛笔，蘸了某种药粉，在洞穴最下方的墓砖上涂抹了几下，原本青灰色的墓砖，在火光的照耀下立刻泛起了一层黑色，然后挥发散去。
<br/>　　“这是尸气，没错了，不久之前，肯定有僵尸从这里爬出来过。”
<br/>　　叶少阳跟毛小方对视了一眼，都是暗暗吸了口气，虽说以他们的身份，对僵尸之类的没什么怕的，不过这里毕竟是古墓，是僵尸的地盘，谁知道下面有多少僵尸？是什么等级的僵尸？因此现在找到了下面确定有僵尸的证据，两人还是有点紧张的。
<br/>　　“好，有僵尸就好！”陈晓宇摩拳擦掌，“我们几个今天要大开杀戒，就怕僵尸太少。”
<br/>　　“放心，不管有多少僵尸，我全让给你来对付。”叶少阳说道，心里暗想，但愿到时候你还能保持这种不怕死的劲头。
<br/>　　妙心没让大家先进去，而是摸出一张黄裱纸，折成纸鹤的形状，用朱砂笔点睛之后，松开手，对着纸鹤吹了口气，念道：“地行无果，阴阳明证，见阳而沉，见阴而返，去吧。”
<br/>　　纸鹤扑腾了一下翅膀，顺着洞口飞了进去。
<br/>　　叶少阳明白她这一手作法的动机，伸手在洞**试了试，说道：“空穴不来风，这里有风，那就说明下面是有氧气的，应该没问题。”
<br/>　　说完之后，叶少阳转头看去，发现几个人都在用奇怪的眼神看着自己，正不知道怎么回事，妙心说道：“你刚说什么，阳气？这里怎么会有阳气？”
<br/>　　“呃……”叶少阳总算知道怎么回事了，拍了拍脑门，说道：“我说的不是阳气，是……氧气。人就是因为吸收了空气中的氧气，才能存活。”
<br/>　　这个时代，现代科学还没普及，至少在华夏，还没有“氧气”的概念，叶少阳只好给他们普及了一下。
<br/>　　梧桐道：“你这是哪里听来的？我怎么没听说过。”
<br/>　　“这……”叶少阳不知道怎么解释。
<br/>　　“是他自己编的呗，故作惊人之语。”陈晓宇不失时机地对叶少阳讽刺道。
<br/>　　叶少阳解释不清楚，干脆就不解释。陈晓宇还当他是被自己逼问的无话可说，更加得意，讽刺了叶少阳几句。
<br/>　　等了有一炷香的时间，那只被妙心作法驱使的纸鹤又从洞穴里飞了出来，但是扑腾了几下翅膀，立刻就萎缩下去，浑身变黑，变成了一堆枯纸……
<br/>　　妙心震惊，道：“下面有阴巢，而且阴气极重，不是好兆头。”
<br/>　　陈晓宇不以为意地说道：“怕什么，兵来将挡，就算是有尸王在里头，凭我们几个也没什么好怕的。”
<br/>　　妙心盯着化作一团烂纸的纸鹤残骸说道：“话虽如此，还是要小心为上，你们切记，跟在我的后面，少阳子，你来掌灯吧。”
<br/>　　叶少阳知道她潜意识里觉得自己最没用，所以让自己干这个活，笑了笑，从卢晓清手接过了三色莲花灯。毛小方望着灯，跟叶少阳打趣了一句：“少阳子，这时候要是有你说的‘手电’，肯定比这灯好用多了吧。”
<br/>　　叶少阳道：“那当然。”
<br/>　　“什么电？法器么？”妙心问道。
<br/>　　“咳咳，不是不是，咱们继续。”叶少阳干咳了两声，用三色莲花灯给她照亮。
<br/>　　妙心摸出三炷香，点燃之后，让叶少阳捏着，告诉他时刻注意，如果香灭了，一定要告诉她，然后大伙就要立刻退出来。
<br/>　　叶少阳立刻就知道了这么做的原因，跟灵异无关，其实是最简单的科学原理：所谓人点烛，鬼吹灯，在下墓的时候，盗墓贼都要点一支蜡烛放在墓穴里，一旦蜡烛灭了，以为是鬼魂作祟，就要立刻退走。
<br/>　　其实蜡烛灭了，只是因为古墓里缺乏氧气……不过那是真到了墓穴深处，不用走动的时候，才方便用蜡烛来检测氧气，在墓道里行进，一般都是越来越深，很可能走着走着就没氧气了，但拿着蜡烛不方便走动，所以用香，香火如果灭了，自然也是因为缺氧。过去的人虽然不知道火燃烧的原理，但是在实践之中也总结出了这样的原则。
<br/>　　至于三色莲花灯，因为用的不是灯油，就算没有空气，这要有阴气，一样可以燃烧，所以不具备检测的作用。
<br/>　　妙心拿出一只罗盘，拨弄了几下，第一个顺着洞穴爬了进去，然后叶少阳跟上，用三色莲花灯照过去，是一条长长的墓道，修得很整齐，两边都是一眼看不到头。
<br/>　　等人都进去之后，大伙开始研究往哪边走。妙心提议往风吹来的方向走，风只能从低处吹向高处，下面，自然就是墓穴深处。
