　　火鬼王毕竟是鬼王，身为一方霸主，怎么可能是心慈手软之辈。
<br/>　　别看她刚才看似一副柔弱，一旦杀起人来，比碾死蚂蚁更简单，从话语中就能听出她的手段是何其狠辣。
<br/>　　然而，火鬼王的话语声还未落下，地狱冥火中传出的清朗声音，令她狰狞的表情凝固在了脸上。
<br/>　　“若是对别的修士或许是棘手的东西，但区区冥火，对我可丝毫不管用。”
<br/>　　话语刚落，冥火似乎受到命令一样，以极快的速度汇聚，最后汇聚在了慕清霄的手心中。
<br/>　　冥火跳烁间，似乎是畏惧。
<br/>　　慕清霄本人，则站在原地，连衣角都没有烧损一丝。
<br/>　　见慕清霄平安无事，并且将她释放出的地狱冥火抓在手中，如同玩具一样的画面，险些将火鬼王的眼珠子都给瞪出来。
<br/>　　“你，你不惧冥火！”
<br/>　　瞬间，火鬼王脸色难看到了极点，甚至有些惨白。
<br/>　　整个鬼界中，不惧怕地狱冥火的只有两种，一种是比她更强的火鬼，第二种就是比鬼王更强的修士。
<br/>　　第一种完全没有可能，她可不曾听说过，鬼界中有比她更强的火鬼，至于第二种……
<br/>　　想到此处，火鬼王才发现，后者的气息她竟完全看不透，如同迷雾一样，这令她的心瞬间凉了一大截。
<br/>　　“鬼界高层不可能插手鬼王间的争斗，哪怕阎罗鬼王也没有这种待遇，你究竟是谁？”
<br/>　　“十息到了，既然如此，那就只能送你上路了。”
<br/>　　慕清霄话语刚落，火鬼王顿时觉得不妙，转身就化作一团赤色火焰，想要逃离此处。
<br/>　　然而，刚飞出没几步，她便感觉自身仿佛陷入了一座沼泽中，整个空间都变得粘稠无比，令她寸步难行。
<br/>　　凤眼中闪过一抹惊惧，火鬼王失声道：“你……你不是鬼族！”
<br/>　　“咻咻……”
<br/>　　火鬼王刚想开口求饶，一抹寒光在她的视线中渐渐清晰，魂剑仿佛无视空间，霎时间就来到了她的颈部。
<br/>　　寒光四溢，魂剑似乎要撕裂空间。
<br/>　　“嗤！”
<br/>　　剑刃断物的声音回荡在偌大的溶洞当中，一颗硕大的头颅高高抛飞而起，赤红色鲜血携着炙热的高温喷发而出。
<br/>　　洒落过后，地面都被灼烧出一个个坑洞。
<br/>　　魂剑之所以称之为魂剑，那是因为，魂剑斩的可不止肉身，更是连灵魂都一起斩杀。
<br/>　　魂剑掠过，最终折回阴阳玉中。
<br/>　　但身为一方霸主的火鬼王却身死道消，身躯倒地后，化作一滩岩浆，随后就有一颗半个拳头大小，通体赤红的圆珠漂浮而起。
<br/>　　失去了宿主，火灵珠似乎有些惊恐的在空中飘飞，下一秒就要逃去远方。
<br/>　　但周围空间已被封锁，慕清霄随手一挥，火灵珠就仿佛受到强大的吸力，无处可逃，最后直接掠向他的掌心。
<br/>　　“叮，掠夺火灵珠任务完成，宿主信息数据化……”
<br/>　　宿主：慕清霄（龙骑士）
<br/>　　血脉：未知
<br/>　　资质：未知
<br/>　　心法：《御女心经》
<br/>　　神通：《黑凤化身》《空间门》
<br/>　　剑意：无上剑意
<br/>　　武器：魂剑
<br/>　　战技：《空明剑》《太虚剑》《空明幻虚剑》

<br/>　　脑海中响起久违的系统提示声，慕清霄嘴角牵起一抹笑意，随后目光落在手中的赤红色圆珠上。
<br/>　　火灵珠，蕴含着无穷法力的火系灵珠。
<br/>　　将火灵珠装进储存空间，慕清霄并不打算现在炼化，这已经是第三颗灵珠了，总共五颗灵珠，还剩下最后两颗。
<br/>　　水灵珠与风灵珠。
<br/>　　正当慕清霄准备离开之际，目光一凝，视线落在早已化作一滩岩浆的火鬼王身上。
<br/>　　只见他随手一挥，一团诡异的青色火焰就从地下漂浮而起，正是火鬼王刚才释放出的火焰，地狱冥火。
<br/>　　伴随着强大的牵引力，地狱冥火慢慢飘飞到了慕清霄的面前。
<br/>　　既然火鬼王死后，地狱冥火没有消失，可见这火焰以前恐怕也是无主之物，只是碰巧被火鬼王炼化了而已。
<br/>　　地狱冥火与斗破位面的异火相同，似乎更加的强大诡异。
<br/>　　望着眼前的地狱冥火，凭空一团黑色火焰升腾而起，最终将地狱冥火包裹住，慕清霄轻轻一吸，地狱冥火就被咽入腹中。
<br/>　　“地狱冥火，在将岩浆中的精纯能量吸收，不知能不能助我突破一重修为。”
<br/>　　对于慕清霄来说，地狱冥火真是意外之喜，将其炼化，神火威力估计会增强一些，最重要的是还是他的修为。
<br/>　　他现在是大乘期二重境，大乘期过后就是渡劫期，渡劫期过后才是地仙。
<br/>　　他现在施展神通《黑凤化身》，虽然不可能是地仙的对手，但配合神通《空间门》逃跑却不成问题。
<br/>　　唯有迈入渡劫期，配合上神通，才有与地仙强者抗衡的实力。
<br/>　　所以，实力自然是越强越好。
<br/>　　他的修炼速度已经不算慢，修为越到后面，每一重都如同隔了一重山，若是普通修士，几十年突破一重也是屡见不鲜。
<br/>　　而他从合体二重迈入大乘二重，跨越了一整个境界，仅仅只用了寥寥数年的时间，如此修炼速度，只能用骇人听闻来形容。

<br/>　　火鬼王死亡的消息传出去越早，他暴露的可能性就越大，将空间结界留下，鬼王级别都不能破开，谁也不清楚里面的情况。
<br/>　　等有人发现火鬼王已经死亡，或许那个时候，慕清霄已经早早的离开了鬼界。
<br/>　　离开鬼王宫后，慕清霄顺着原路返回，几秒后就出现在了极乐世界的入口。
<br/>　　此时，入口处正有两名鬼兵把守，慕清霄挥手划开虚空，钻入空间门中，不多时就重新回到赤红的岩浆世界中。
<br/>　　感受着周围包围而来的粘稠岩浆，以及其中蕴含的精纯能量，慕清霄结印布下结界，就地盘坐岩浆中，五心向天，运转起心法。
