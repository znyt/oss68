　　第667章
<br/>　　今天天气很好，日头高悬头顶，天气是一日比一日热了，炎风扑面。
<br/>　　程大雷一人踱步在蛤蟆城的长街上，身边一个随行的人也无有。以他目前的身份，出入皆有软轿，身边十几个亲随护卫是理所应当的事。
<br/>　　直到如今，程大雷还没有养成这样的毛病。倒不是说程大雷不喜欢这些，而是他完全不知道可以这样。
<br/>　　他手下有精兵三万，谋士无数，关鱼、秦蛮这样的世之虎将对其忠心耿耿，程大雷一声令下，便有无数人愿意去为他赴死。如今虽然只占了朔方和蛤蟆城两座城，但进一步，取凉州不是不可能。再进一步，贪婪得想取天下也并非不可能。
<br/>　　早已不是从前。
<br/>　　当他走在长街上时，众人纷纷避让，那些远远张望的眼神，除了敬重，怕更多的还是恐惧。
<br/>　　如今的他，府中珍宝无算，都是世之奇珍，如果他想要，不知有多少女人愿意投怀送抱，而他挥一挥手，也可以让无数人人头落地。
<br/>　　当然，程大雷并非一个贪婪的人，野心这种东西他也很欠缺。但，曾经不要，是没有能力要，如今当已可以要的时候，有几人舍得说不要。
<br/>　　正如林少羽当初打破长安城时，也没有取天下的心，心里还寻思着把皇位让给程大雷。然而，当他真正做上那高高在上的龙椅时，他还舍得让么。
<br/>　　程大雷踱步而行，也不着急赶路，只是脑袋里胡思乱想着。
<br/>　　如果有朝一日，自己真的可以登上那帝位，如刘悲、关鱼这种人必然是要委以重任的，毕竟是辛辛苦苦打出来的，而他们的亲人儿女也不可能被亏待。程大雷是这种心思，就没有权力要求刘悲和关鱼不是这种心思。到时候子子孙孙，代代世世无非又是一次帝国的重演。
<br/>　　想到这里，程大雷忽然有一阵后怕，只感觉自己在苍茫天地之间，是如此渺小。那高高在上的位置上，坐的是姓李还是姓程，真的有区别么。
<br/>　　天地冷眼旁观，看世人争夺，偶尔发出冷笑。
<br/>　　回过神来时，程大雷来到军策府，刘悲、徐神机、吴用都在场，正召集手下谋士开会。
<br/>　　见程大雷进来，诸人纷纷起身，程大雷摆摆手，令他们都坐下。
<br/>　　“吵得怎样了？”程大雷坐下后，将双脚搁在桌面上。
<br/>　　吴用道：“昨天争了一天，现在还在继续吵，关键还是要大当家拿个主意。”
<br/>　　诸人在一起讨论的还是战术问题，真正的战略问题还是要程大雷来决定，也只能由程大雷决定。当程大雷拿出战略后，他们才好讨论具体作战。
<br/>　　地图在桌子中间摊开，程大雷看着凉州的五座城，以及上面的山脉水源。
<br/>　　形势对己方其实不利，兵力没人家多，粮草也没人家富，而且喜字队刚刚得到的情报，宋伯康已向京州请求援助。援助必然是粮草、兵器、马匹或者直接是精兵强将。
<br/>　　一旦京州的支援过来，本来不利的形势就会变得更加艰难。
<br/>　　程大雷如今唯一拿出来可以说的便是兵员素质，如果是野战的话，蛤蟆城三万精兵能破敌十万。
<br/>　　但这次要打的明显是攻城战或者是防御战，如此比的就不是兵员素质，而是粮草。
<br/>　　朔方城刚有些气势，但还没有站稳脚跟，蛤蟆城虽然有些粮草，但如果对方围而不攻，也足以将程大雷耗尽。
<br/>　　现在军策府讨论来讨论去，要么就是坚壁清野，要么就是主动出击，都不能说是错误的策略，但同时，也未必是能够获胜的策略。
<br/>　　程大雷手指敲着桌面，一直没有做声，其实他心中已有个大概思路，但却没有直接说出来。
<br/>　　“大当家，大当家……”徐神机连唤几声：“等您的法子呢？”
<br/>　　程大雷点点头，抬起手落在地图上，从武威、浩甲、金城以及宋伯康几处重要营寨前划过，最后落到一地。
<br/>　　“我们直接打这里如何。”
<br/>　　众人一惊，屋子里变得安静下来，因为程大雷手指落的地方是凉州城。
<br/>　　凉州城是凉州的主城，兵最强、粮最多，墙最厚，如此，也是最不好打。
<br/>　　蛤蟆城倒是不缺胆大的人，但程大雷昨日还说着要保守些，保守些，怎么睡了一觉，今天就突然像换了一个人。
<br/>　　连程大雷都没感觉到的是，他已在发生潜移默化的改变。当他品尝过权力的甜美后，怎么可能不动心，总是会变得越来越贪婪。
<br/>　　“我们和宋伯康之间必定有一战，凉州容不下我们两个，此战必是生死之战。既然是生死之战，那就直接奔最后的决战，两军对垒，一剑定胜负。”
<br/>　　饶是吴用这种胆大心狠之人，也被程大雷镇住。程大雷这是在棋局的开始，便推开车马炮，直接与对方将帅对决。
<br/>　　“如此做，也并不是不可行，但我们攻打凉州城，武威、金城、浩甲城不会坐视不理……但应该也不是没有办法。”吴用的野心也被点燃。
<br/>　　程大雷点点头，将脚从桌子上放下来，直起身道：“就是这个战略，你们讨论吧，尽快拿出个主意来。”
<br/>　　“是。”军策府的谋士痛快答应。
<br/>　　幸运的是，程大雷是从山贼起家，有这个企业文化在，他手下的人都是不怕招惹事，就怕没有事。
<br/>　　程大雷敢有这样的胆量，他们就有胆子去谋划，执行力是不用催促的。
<br/>　　程大雷离开了军策府，临行前，嘱咐刘悲通知赵子龙，令人将苏樱与樊梨花接回来。
<br/>　　这是风雨来临前的宁静，程大雷此心安稳，今天的蛤蟆城虽然平静，但真正的战争来临后，会是怎样的结果，程大雷完全无法预测。
<br/>　　赢了，自己就是凉州之主，帝国十三州，自己占了一席之地。输了，侥幸的话，带着身边几个人另起炉灶重开火，不幸的话，自己有大概率能活着，毕竟绝世身手，逃跑还是没问题的。但徐神机、苏樱这些人怕就没办法活下去了。
<br/>　　程大雷拍了拍额头：原来真的输不起呐。
<br/>(本章完)
