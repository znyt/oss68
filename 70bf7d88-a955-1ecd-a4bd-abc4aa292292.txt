　　只见，
<br/>　　叶尘的手掌，按在唐威的胸口之上，自他的掌心之中，顿时发出一道柔和的白色光芒，犹如牛乳般，缓缓流入唐威的体内。

<br/>　　更甚至，就算没有那些检测仪器，众人也可以清晰的感觉到，唐威的生命元气在以一种奇迹般的速度，急剧的恢复着，简直比神话故事里的灵丹妙药还要神奇！
<br/>　　“这……我不是在做梦吧？”
<br/>　　“这是什么妖术？”
<br/>　　“这人是从哪里冒出来的？”
<br/>　　……
<br/>　　众人望着眼前这个突然出现的神秘少年，以及如此诡异的举动，一个个又惊又骇，满脸的不可思议，一时间都忘记了阻拦。
<br/>　　后方追进了的一众保安也全部看傻，李虎和陈东两人，同样也是第一次看到叶尘施展出如此手段，不由得两眼瞪得老大。
<br/>　　“尘哥也太牛了吧！难道是神仙下凡不成？”
<br/>　　李虎看到原本命悬一线的唐威，脸庞已经开始红润起来，顿时激动的不得了，望向叶尘的目光，也变得越发崇敬、崇拜。
<br/>　　“咳咳！”
<br/>　　在众人无比震撼的目光中，原本被医院判定，几乎已经没救的唐威，一阵咳嗽过后，竟然缓缓睁开了双眼。
<br/>　　不但各种生命指标恢复正常，甚至身上、体内的伤势，也在以肉眼可见的速度，在飞速的愈合着！
<br/>　　要知道，这前后，也不过才区区几分钟的时间而已！
<br/>　　“尘哥，你、你怎么来了？我记得我被龙盟的人打伤了，我没死？”
<br/>　　唐威恢复了神志之后，双手在床上微微一撑，竟然直接坐了起来，再次把周围的那些医生看的目瞪口呆。
<br/>　　唐威自然不知道，自己在短短几分钟之内，已经从鬼门关走了一圈。
<br/>　　叶尘见唐威的伤势已经好了七七八八，这才收起手掌，微微一笑道：
<br/>　　“有我在，你想死哪有那么容易？”
<br/>　　说到这里，叶尘的眼眸深处又微微闪过一抹寒光，
<br/>　　“至于那龙盟，竟然敢对你下如此狠手，咱们这就去找他们好好算算这笔账！”
<br/>　　唐威听到这话，脸色顿时微微一变，
<br/>　　“尘哥，龙盟不知道从哪里请来一位化劲宗师，我在他手下连一招都撑不过，这件事还是算了吧！”
<br/>　　听到这话，叶尘顿时笑了，
<br/>　　“区区一个化劲宗师而已，我弹指便能灭之，你不必担心！”
<br/>　　说完这话，叶尘又随手在唐威身上轻轻一拍，
<br/>　　轰！
<br/>　　唐威顿时感觉一股暖流进入自己的身体之中，瞬间便流遍他的五脏六腑，紧接着，全身上下都传来一股无比舒适的美妙感觉，身上的伤势已经好了七七八八，浑身充满了力量。
<br/>　　唐威顿时又惊又喜，双手在床上一撑，直接便从床上跳了下来，
<br/>　　“我的伤……竟然全好了？这怎么可能？”
<br/>　　不但唐威满脸的惊骇，就连一旁的其他人，也如同活见鬼了一般。
<br/>　　他们又哪里知道，叶尘刚刚给唐威服用的，乃是以扶摇仙果炼制的九转天香丹，即便是对于叶尘这个级别的修真强者，也是不可多得的灵丹神药，更何况唐威只是比普通人略强一些而已。
<br/>　　如今的唐威不但伤势已经好了七八成，功力也更大胜从前，用不了多久，便可以直接突破壁障，达到化劲宗师之境。
<br/>　　一直等到叶尘四人堂而皇之的离开手术室，周围的那些医生、护士、保安等人，依旧个个一脸懵逼，甚至都开始怀疑人生了。
<br/>　　一个刚刚还处于垂死状态、几乎都要被放弃治疗的重伤病人，竟然在转眼之间，变得活蹦乱跳！
<br/>　　此事若非亲眼所见，他们打死也不敢相信。
<br/>　　……
<br/>　　天海大学，龙盟武道社，
<br/>　　“何前辈，这次多亏了您出手，听说那姓唐的，中了您一掌之后，已经被送进了医院，快要撑不住了！”
<br/>　　龙盟武道社社长钱龙，对着一名气势很强的中年人，一脸赔笑道。
<br/>　　原来，这中年人名叫何坤山，是龙盟武道社高价聘请来的化劲宗师，之前便是此人，一掌险些要了唐威的性命。
<br/>　　何坤山倚在一张太师椅上，满脸傲然之色，
<br/>　　“若不是怕被人说我以大欺小，我早就一掌结果了那小子，又岂会让他多活一天？”
<br/>　　钱龙连忙继续赔笑道：
<br/>　　“那是那是！以何前辈的修为，便是在化劲宗师里面，也难觅对手，更何况那唐威只有区区半步化劲宗师而已……”

<br/>　　“那唐威自然不算什么，怕就怕他背后的那个人，可不太好惹啊！”

<br/>　　他听说何坤山将唐威打成了重伤，便立即赶了过来，总觉得此事有些不妙。

<br/>　　听到这话，钱龙和何坤山，全都霍然站起身来，一脸的惊愕之色，
<br/>　　“阿豪，你没有看错吧？真的是唐威？”

<br/>　　“难道是因为他？不，不可能，即便他背景通天，也不可能有起死回生的能力啊？”
