　　　　这虚堇跟虚家家主的对话，是在布满禁制的宫殿中谈论。

<br/>　　“果然，这虚堇的目的没有那么单纯，不准备给我北邙学院考核资格的名额？”
<br/>　　楚风眠冷笑连连。
<br/>　　这想不想给，可不是他们说的算了，既然是将楚风眠请来了，请神容易送神难，楚风眠自然要得到目的在走。
<br/>　　“楚风眠，这虚堇似乎并没有打什么好主意，看那样子，她是不想要给我们北邙学院的考核资格名额了。”

<br/>　　“无妨，我早就清楚，这虚堇不会乖乖将北邙学院的考核资格名额交出来的，不过既然是来了，自然不能空手而归。”
<br/>　　楚风眠笑着回答道，言语中平静无比。
<br/>　　想要算计楚风眠，那倒是要看看，到底是谁算计谁来。
<br/>　　“楚公子，这是我们虚家最好的客房，还请各位休息一下，到晚上家主会布置晚宴，倒是我在来邀请各位。”
<br/>　　那管家华瑞，将楚风眠一行人带到了一处宫殿中，开口道。
<br/>　　这管家华瑞的实力，却是也极高，有着神力境五重的境界，放在一些小国度中已经足以算得上是一尊称王称霸的存在了。
<br/>　　在这虚家，却是只是一个关键，不过也是极为极高的一尊管家。
<br/>　　华瑞平静开口道。
<br/>　　“诸位有什么需要的，尽管吩咐，诸位是堇小姐的贵客，我们自然会照顾好一切的。”
<br/>　　“哦？楚某还真有些需要的东西，需要劳烦华瑞管家了。”
<br/>　　在那华瑞刚刚打算离开的一刻，楚风眠却是突然开口了。
<br/>　　这让华瑞的脸色，微微的抽动了一下，眼神中尽是不屑。
<br/>　　果然是偏远之地的土包子，居然就连这点便宜都想要占。
<br/>　　楚风眠一行人，本就是来自一个偏僻的国度，在华瑞看来，根本就是一些小地方出来的人，本身便是就极为不屑。
<br/>　　他刚刚的话，不过是客气，但是如今楚风眠居然是毫不在意，直接开口。
<br/>　　更令他的心中极为不屑。
<br/>　　不过华瑞的表情，倒是一副极为平静的样子，转过身来，向着楚风眠开口道。
<br/>　　“不知道楚公子需要些什么？”
<br/>　　“我需要一批圣者精血，或者妖神精血，华瑞管家不知道能否去帮楚某来购买一些？”
<br/>　　楚风眠平静的开口道。
<br/>　　“当然，这钱不用华瑞管家来出，我自己出便是。”
<br/>　　楚风眠开口的瞬间，大手一挥，便是足足两百亿九鼎丹，一下子飞了出来。
<br/>　　这两百亿九鼎丹，直接便是化为了一条灵药长河。
<br/>　　那华瑞的表情，本来还是极为不屑的，不耐烦的样子。

<br/>　　这可不是两亿，二十亿，足足两百亿九鼎丹。
<br/>　　他们虚家虽然拿的出两百亿九鼎丹这等财富来，但是也不可能挥手之间一下子拿出来吧，听楚风眠的样子，这却是极为轻松的样子。

<br/>　　这等财富，虚家家主都不敢一口气拿出来这么多。
<br/>　　“这些土包子，哪里来的这么多的财富。”
<br/>　　华瑞的脸色微微抽动，还是很快化为正常。
<br/>　　“我们虚家，在这斗玄城中也有着不少生意伙伴，这生意直接跟我们虚家做便是，还可以给诸位一个优惠的价格。”

<br/>　　说完，华瑞便是直接转身离去。
<br/>　　房间中只剩下楚风眠三人。
<br/>　　“少主，我看着虚家人都没有什么好心，我们留在这里，只怕是会遇到危险。”
<br/>　　拓跋鼎在一旁，暗自开口道。
<br/>　　“一个北邙学院的考核资格名额，价值极高，我怕虚家的人不会这样乖乖交出来。”

<br/>　　虚家家主皱了皱眉的说道。
