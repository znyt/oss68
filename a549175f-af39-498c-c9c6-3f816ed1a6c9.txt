　　P病毒的致死率很高，只要感染上，在没有抗病毒疫苗的情况下，基本就是必死的结果。
<br/>　　而美国人从头至尾都是认为西伦科特手中那两支改进型的P病毒到了中方手中，所以，即便罗昊现在拿着一个高音电喇叭大声叫喊没有研究出最新的P病毒抗病毒疫苗，美国人也不会信。
<br/>　　“拥有”改进型P病毒抗病毒疫苗，而且“得到”了布克尔特手中那支改进型P病毒，罗昊这边就彻底掌握了主动权，这还不唬的外面那些忙着爆破破门的美国人两眼发直，三魂丢了两魂半简直就是没天理啊！
<br/>　　“可是我们要用什么东西来伪装成P病毒？”司徒空对罗昊问道：“我们不能真的在瓶子里灌满自来水糊弄他们吧？这也太低端了，根本就欺骗不了他们，而且这里也没有自来水啊。”
<br/>　　罗昊走到地下室的冰箱前，把冰箱门打开。
<br/>　　这个地下室是布克尔特这间安全屋最后的藏身之地，所以在这里布克尔特还是准备了比较齐全的东西，冰箱中不仅有冷藏的血浆，还有各种食物和饮料，布克尔特甚至还准备了氧气瓶以应对长时间被困的情况，只是刚才他根本没想到罗昊能猜出地下室那扇钢板门的密码，所以来不及使用他准备好的氧气瓶，导致地下室中出现缺氧的情况。
<br/>　　罗昊从冰箱中拿了一瓶葡萄汁出来，又拿了一袋血浆出来，把玻璃瓶中的葡萄汁倒得只剩下最后一点，然后把血浆兑入到玻璃瓶中的葡萄汁中，深紫色的葡萄汁颜色立刻发生了变化，变得紫色中带着一缕红色，不仔细分辨真的看不出跟P病毒的颜色差别。再加上东西也是从冰箱中拿出来的，也符合P病毒低温储存的要求。
<br/>　　“你还真别说，这颜色还真跟P病毒的颜色很像。”司徒空对着罗昊竖起大拇指，说道：“狂龙，真有你的，这么好的办法也能想出来！”
<br/>　　“你在胡说八道些什么！”陈应天瞪了司徒空一眼，说道：“这就是P病毒，可以通过皮肤接触、呼吸道和直接注射感染的改进型P病毒！改进型P病毒就是这个颜色！”
<br/>　　“罗刹说得不错。”
<br/>　　罗昊把勾兑好的“P病毒”玻璃瓶的盖子拧紧，说道：“除了布克尔特、西伦科特和‘暗月’以外，只有我们见过改进型P病毒，只要我们咬死这就是布克尔特手中的P病毒，外面那些人能说个‘不’字？”
<br/>　　“不过为了确保万一，大家把身上手雷的引信全都换成碰炸引信，以备需要我们强行突围。”
<br/>　　罗昊最后还是对战术中队的队员提醒了一句。
<br/>　　战术中队这是参考了外军特种部队作战方式，身上会额外携带碰炸引信和绊发引信两种引信，在需要的时候，只要更换手雷上的引信就能让手雷用更多的用处。
<br/>　　比如在山地作战的时候，使用延时引信的手雷显然是难以应对占据了地理优势的敌人，因为手雷往高处扔，可能还没等到手雷爆炸就又重新滚落下来，不说无法对敌人造成杀伤，甚至可能对己方部队造成杀伤；但是更换成碰炸引信，情况就不一样了，只要触碰到就会发生爆炸，可以有效对藏在山地岩石背后的敌人造成杀伤和火力压制。
<br/>　　如果换上绊发引信，就能把手雷当做反步兵地雷来使用，用来制作各种令人防不胜防的诡雷。
<br/>　　拧掉原来手雷上的延时引信，重新换上碰炸引信，并不需要很长时间，很快所有战术中队的队员都完成了对手雷引信的更换。
<br/>　　嘭！嘭！嘭！嘭！
<br/>　　地下室的脱氧钢板门传来几声塑胶炸药爆炸的声音，这扇一度将战术中队阻挡住的脱氧钢板门被塑胶炸药炸开了一个大洞，两枚闪光震撼弹被扔了进来。
<br/>　　特种部队那套破门突袭的套路，龙隐部队都清楚，所以地下室中的战术中队早就有所准备，这两枚闪光震撼弹并没有对战术中队造成很大的影响。
<br/>　　Royce带着CiA行动队员穿过被炸开大洞的脱氧钢板门，进入到地下室中，双方举枪相对，地下室中的气氛瞬间变得紧张起来。
<br/>　　看到躺在地上的布克尔特的尸体，Royce的眼神闪烁了一下，手中M4A1战术步枪指着罗昊，问道：“你们杀了他？”
<br/>　　“你都已经看到了，还需要多问什么？”罗昊反唇相讥道：“还是说你的眼睛不好用，需要我给你说明一遍？一颗.45口径的ACP弹打穿了他的脑袋，死得彻彻底底。我知道你们是为了什么而来，是为了这个吧。”
<br/>　　罗昊拿出那瓶“P病毒”，给Royce看了眼又重新收了起来，根本不给他仔细分辨的机会。
<br/>　　“拿来吧！”Royce对罗昊说道。
<br/>　　“你小学没毕业吗？”罗昊用看弱智的眼神看着Royce，说道：“我凭自己本事弄来的东西，凭什么要给你？”
<br/>　　“这本来就是我们的东西！”
<br/>　　Royce身后，一名CiA行动队员上前一步，大有如果罗昊不叫出来，就要动手抢的架势。
<br/>　　“他蠢，你更蠢！”
<br/>　　罗昊看着那名CiA行动队员说道：“这东西到底是谁的，我们都心知肚明，所以就不用说这些低智商的话来侮辱大家的智商。”
<br/>　　罗昊把那瓶“P病毒”拿出来握在手中，对Royce说道：“既然你们这么想要得到它，我可以给你们，我松手的瞬间如果你们能接住，它就是你们的！”
<br/>　　“你这个疯子！”Royce对着罗昊说道：“如果瓶子打碎，我们都会被病毒感染！”
<br/>　　“你错了！”罗昊说道：“是你们会被P病毒感染，但是我们不会！我们是除了‘暗月’以外，唯一拥有抗病毒疫苗的，我们都注射过抗病毒疫苗，所以如果P病毒的瓶子被打碎，P病毒释放，你们会感染这种致命病毒，但是我们不会！”
<br/>　　Royce跟其他CiA行动队员的脸色都是变得有些难看，Royce看到过当年所济斐群岛病毒危机的照片，那些照片带给了Royce很震撼的印象！
