        &lt;h3&gt;第三卷第九十六章  商议重组（感谢李佩云盟主）&lt;/h3&gt;
<br/>
<br/>        寒伯安和郝景文都接到江俞轩的电话。
<br/>
<br/>        江俞轩直奔主题：“大哥，我最近公司出了点问题。”
<br/>
<br/>        “资金链断裂了？”寒伯安正看郝景文给他发的邮件。
<br/>
<br/>        “是的。”
<br/>
<br/>        “我正在看文文发的邮件，你有什么想法？”寒伯安点了一只雪茄。
<br/>
<br/>        “我们现在有两种选择，一种融资，一种银行借贷，但是银行借贷怕是来不及的。”
<br/>
<br/>        “投资你们公司是有风险的，陈子昂这个女人玩的是心跳。”寒伯安手中雪茄那白色的烟雾娉娉袅袅。
<br/>
<br/>        “她是一个积极努力的人，而且也很有经济头脑，目前她看到的项目还没有失败过，只是这一次她忽及了，还有，最近项目拿的有点多，资金回流有些迟缓。”江俞轩看着库房堆积的各种硒鼓叹了口气。
<br/>
<br/>        理想很美好，现实很骨感，陈子昂的联动计划泡汤了，分店是开了很多，前期培训也做得很好，只是名称早期没有确定，怎么投入市场，包装都是简包。
<br/>
<br/>        就比喻一个人，出生几天就要去办出生证，可是非要等得长到好几岁了才去办理那就有点迟了，虽然有个小名，但是小名却不曾叫得出去。
<br/>
<br/>        “应该是工作流程上忽及了吧？”
<br/>
<br/>        “是啊，我这一段时间在西城忙项目，陈子昂也到处跑，她着急的原因是想去甸城，要不然也不会如此匆忙，现在可以说是万事俱备只欠东风了。”
<br/>
<br/>        “文文在北京，我明天也到北京，你让你们财务将公司现状整理一个报告，明天早上一大早我们过去看看。”寒伯安看着郝景文发的资料。
<br/>
<br/>        他们兄弟三人从合伙开公司以来，相互信任，已经拧成了一股绳。
<br/>
<br/>        “大哥，谢谢你。”江俞轩算是吃了定心丸。
<br/>
<br/>        “我列上几点要求，你可以和陈子昂商量一下。”寒伯安知道江俞轩如此着急，肯定是出了大问题，小资金是解决不了问题的，亲兄妹明算账，该提的要求和条件一样也不能落下来。
<br/>
<br/>        事情摆在明面上对他们几个都好。
<br/>
<br/>        “好。”
<br/>
<br/>        江俞轩和陈子昂查看最近的财务报表，其实就是资金回收不到位，如果资金回来快的话，他们是完全有盈利的。
<br/>
<br/>        “本来，我想让五姐拿出一部分，但是五姐说她们也想去甸城投资，我就放弃了。西城的账上也还有些资金。还有别的几个地方也有，但是不多，我不能全部都抽回来。”陈子昂对江俞轩说。
<br/>
<br/>        “目前的财务状况表面上看起来，我们是有利润的，但是就是回款速度有些慢，以后项目不能这么做了，几个分部的钱就不要动了，动了会乱了人心。况且，他们最近都在做着项目。”江俞轩停顿了一会。
<br/>
<br/>        “大哥说等一会他列一下他们的条件，他们明天到，我们商量一下，你要是觉得条件可以的话，我们就请大哥他们出手吧？”
<br/>
<br/>        “好。”
<br/>
<br/>        陈子昂沉默了一下，“耗材现在还没有营销部，我准备成立营销部，专门成立一个部门，才能专注这个事情。”
<br/>
<br/>        “嗯，那就这几天把部门成立了吧，专人负责。有合适的人选吗。”
<br/>
<br/>        “西城的李凌菲，北京的倪小美。我目前只有这两个人选。还有，我要去甸城的话，这边公司就没有人负责了。”
<br/>
<br/>        “你为什么非要去甸城呢？这边的生意不够你做吗？而且你还想将你的耗材推向大的市场，还准备国际化。”
<br/>
<br/>        “我，我是因为谷强。”陈子昂不打算隐瞒了。
<br/>
<br/>        “谷强？谷强是你哥哥？”江俞轩猜测。
<br/>
<br/>        “我怀疑他是失忆了，但是我没有证据，所以我想过去看看。”
<br/>
<br/>        “那也没有必要过去作项目吧？”
<br/>
<br/>        “我已经和祁峥还有谷强都说好了。他们在那边已经开始请专家做预算了。”
<br/>
<br/>        “如果谷强真的是你的哥哥呢？”
<br/>
<br/>        “如果他真的是失忆了，知道他活着就好啊。”陈子昂看着窗外雾蒙蒙的天气，眼里蒙上了一层水雾。
<br/>
<br/>        人生匆匆，如过往云烟，可是有些存在于心里的羁绊是无法抹去的。
<br/>
<br/>        江俞轩是见过谷强的，那年在信息会上见过，俩个人眉眼真的很是相似，如果陈子昂女扮男装的话估计和谷强差不多，奇怪的是，陈家几个老大和陈子昂的相貌只有几分像，最像的当属于他们的眼睛，每个人都是清澈透明的丹凤眼，包括陈子建和陈子乔。
<br/>
<br/>        江俞轩没有经历过亲人离散的痛苦，他了解不了陈子昂的感觉，但是看着陈子昂那雾蒙蒙的眼睛，他心里也有些替陈子昂感到难过。
<br/>
<br/>        “那你准备什么时候去？”
<br/>
<br/>        “明年三月份吧。”
<br/>
<br/>        陈子昂答非所问，公司目前面临的困难是她不曾预料到了，在商场上整整打拼了九年，公司有时候也会碰到资金周转困难，但是她总是会在问题出现之前将缺口堵住，这一次确实是她太操之过急，也不应该将全部精力放到拓展分店上。
<br/>
<br/>        “还有几个月的准备期，子昂，西城的项目是我亲手来抓的，这几年我怕是不能兼顾北京的事情了。”
<br/>
<br/>        “我也在想这个问题，我到哪里去找你这样的人才？”陈子昂忧心的说，她还想去甸城呢。
<br/>
<br/>        江俞轩走了，她也走了，公司这么一大摊子的事情谁管呢？
<br/>
<br/>        “你可以将陈子霖调过来。”江俞轩提议。
<br/>
<br/>        “不行，我哥对北京不熟悉，怕难服众。”陈子昂直接拒绝了。
<br/>
<br/>        “公司的高层都是老员工，他们都有团队意识，我觉得可行，西城那边我可以照看着。”
<br/>
<br/>        “这样也行？”
<br/>
<br/>        “当然可以，我本身就是公司的股东，还有，以后如果大哥他们入股了，公司重大决定都会有大股东开会抉择，以后也会有专业的人员来管理。没有必要你还要亲力亲为的，知道吗？”
<br/>
<br/>        “那一下从西城抽调走三个人会不会不太好？”
<br/>
<br/>        “不会，这几年西城早已经成了一个规模公司了，离开谁都可以运转，你放心吧。”
<br/>
<br/>        “行，那就这样决定了。”陈子昂眉开眼笑。
<br/>
<br/>        “那就趁现在还有时间就召开一个高层紧急会议吧。”江俞轩明天下午还要返回西城，他们的南郊改造项目如火如荼的进行着，杨老总天天都盯着他。
<br/>
<br/>        杨老总要抓业绩，他也要立品牌，这个是西城的重点工程。
<br/>
<br/>        他们在快要下班的时候召开了紧急会议，宣布了江俞轩调到西城，陈子霖和李凌菲调到北京。
<br/>
<br/>        倪小美和李凌菲管理耗材运营，专门负责耗材。
<br/>
<br/>        并且从各地也重新调回来了几个部门经理。
<br/>
<br/>        陈子昂心里算是对人员上的担忧消除了，哥哥这几年已经成长起来了。
<br/>
<br/>        江俞轩虽然回到了江家，正如他所说，他还是公司的股东，西城有他负责，她要李凌菲负责耗材运营，后期可以将海外的市场交给李凌菲和倪小美。
