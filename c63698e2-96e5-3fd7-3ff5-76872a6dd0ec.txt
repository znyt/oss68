　　冷漠的目光望着苏莫，黄袍青年冷冷的道：“你说的不错，猎杀魔物，能者得之，可惜你还不够资格！”
<br/>　　听闻黄袍青年的话，苏莫的目光与对方对视，如同针尖对麦芒。
<br/>　　苏莫清楚，此人便是天虚宗的真传弟子，也是他想要猎杀魔物的障碍。
<br/>　　“你可以试试！”苏莫淡淡的说道。
<br/>　　魔渊周围的众人见此，暗暗惊讶，这个苏莫真是胆大，居然敢在天虚宗真传弟子武仁面前猖狂。
<br/>　　魔渊周围的武者，大约有三、四百人，其中大多数都是第一次见到苏莫。
<br/>　　虽然众人对于苏莫之名，早已如雷贯耳，知道苏莫是妖孽天才，但没有人认为苏莫有资格和天虚宗弟子争。
<br/>　　吕罡等人，见苏莫表现的如此强势、霸道，心中更加无语了。
<br/>　　若是苏莫实力高强，有藐视众天虚宗弟子的资格，如此强势还理所当然，但实力不够还如此强势，那就显得太无脑了。
<br/>　　“你没有让我一试的资格！”
<br/>　　面对苏莫的挑衅，黄袍青年武仁面带不屑之色，随即他看向身边一名高瘦的天虚宗青年，道：“李莽，你去解决他。”
<br/>　　“武师兄放心，交给我了！”
<br/>　　高瘦青年点了点头，随即脚踏虚空，大步向苏莫走去。
<br/>　　天空上，那名天虚宗的黑衣青年见此，急忙退了回来，他知道自己不是苏莫的对手，只有宗门的武尊境师兄出马，才能击败苏莫。
<br/>　　唰！
<br/>　　那名高瘦青年李莽，几步之间，便走到了苏莫前方站定。
<br/>　　“真不知道是谁给你的勇气，敢来和我们争抢！”
<br/>　　李莽望着苏莫，面带调笑之色，继续道：“难道是会武之后，自信心膨胀的结果？”
<br/>　　李莽根本没有将苏莫放在眼里，他可是堂堂武尊境武者，岂会惧怕一个小小的苏莫。
<br/>　　嗷！嗷！嗷！
<br/>　　就在此刻，魔渊中又有三只魔物冲了出来，这三只魔物呈人形，个个身高两丈有余，全身漆黑，面目狰狞恐怖。
<br/>　　这三只魔物的气息并不是很强，大概接近武帝境九重武者的样子。
<br/>　　“死！”
<br/>　　苏莫根本没有理会李莽，见魔物出现，眸中杀机一闪。
<br/>　　下一息，十几道凌厉的灵剑在他周身出现，瞬间洞穿长空，向三只魔物绞杀而去。
<br/>　　嗷嗷！！
<br/>　　三只魔物见此，纷纷嚎叫着出手，锋利的利爪向灵剑拍去。
<br/>　　只不过，这三只魔物的实力，实在是太低了，根本不可能挡住苏莫的灵剑，瞬间就被绞杀，尸体都被斩成了数十块。
<br/>　　旋即，苏莫一挥袖袍，便将三只魔物的尸体收了起来。
<br/>　　“可恶！”
<br/>　　李莽见此，顿时大怒了起来，这苏莫在他面前，居然还敢自顾自的抢夺魔物，完全有恃无恐，分明就是不将他放在眼里。
<br/>　　轰！
<br/>　　李莽瞬间出手了，一拳笔直的轰出，一道强大的拳茫凌空向苏莫击去。
<br/>　　做为武尊境一重武者，李莽这一拳颇为强大，浩瀚的拳劲如同火山沸腾，磅礴的威势震荡八方。
<br/>　　“不知死活！”
<br/>　　苏莫眸中冷光一闪，石剑出现在手中，反手一剑撩出。
<br/>　　咻！
<br/>　　霎时之间，一道犀利无比的三色剑气逆斩而出，风驰电逝，瞬间击中了拳茫。
<br/>　　嗤！
<br/>　　犀利的剑气瞬间便切开了拳茫，去势不减，直击李莽。
<br/>　　“什么？”李莽见此，顿时双眼一瞪，大惊失色，他完全没有想到苏莫如此强悍。
<br/>　　眼见剑气急速临身，李莽毫不犹豫，身上玄力喷涌，瞬间形成了一块厚厚的玄力盾牌。
<br/>　　嗤！
<br/>　　但是，玄力盾牌虽然防御强大，但依旧挡不住三色剑气的锋芒，同样被瞬间撕开。
<br/>　　下一息，三色剑气狠狠的击在了李莽的护体玄力之上。
<br/>　　轰！
<br/>　　一声爆响，玄力炸裂，三色剑气从李莽的身体上飚射而过，带起漫天鲜血。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声响起，只见李莽的左边手臂，连带着大半个肩膀，直接抛飞了出去，鲜血如泉般喷涌。
<br/>　　“什么？”
<br/>　　“这怎么可能？”
<br/>　　“怎么会这样？”
<br/>　　一瞬间，所有人都瞪大了眼眸，不可思议的望着这一幕。
<br/>　　武尊境一重的李莽，居然不是苏莫一合之敌，一招便被斩掉了一臂！
<br/>　　天虚宗的众人震惊了，道宗的吕罡等人震惊了，所有在魔渊附近的人，全部都震惊了。
<br/>　　这一幕出乎了所有人的预料，虽然很多人都知道苏莫强大，但也不至于这么强大吧！
<br/>　　嗖！
<br/>　　受伤的李莽，身形急速暴退，瞬间暴退千里，和苏莫拉开了距离。
<br/>　　唰！
<br/>　　大手一探，李莽将自己的手臂凌空抓起，脸色惨白的望着苏莫，眸中尽是惊恐之色。
<br/>　　“你怎么可能这么强？”李莽不可置信的大吼道，他几乎被苏莫一剑斩懵了。
<br/>　　本以为自己一拳，就能将苏莫击败，没想到事实却是如此的残酷。

<br/>　　武仁是天虚宗的真传弟子，实力是一众天虚宗弟子中最强的存在，是他的大敌。
<br/>　　“有点本事！”武仁的脸色阴沉了下来，一个小小的苏莫，不仅敢和他们争抢魔物，居然还敢打伤他们的人。
<br/>　　最主要的是，苏莫的目光，明显是充满了挑衅，对他的挑衅。
<br/>　　“可恶！”
<br/>　　“太嚣张了！”
<br/>　　“简直是找死！”
<br/>　　一众天虚宗弟子都怒了，个个眸中充满杀机，恨不得立刻将苏莫镇压。

<br/>　　其他人见此，暗暗摇头，这苏莫是彻底和天虚宗弟子干上了。

<br/>　　当然，有的人佩服苏莫，但也有很多认为苏莫是自大，亦或是狂妄。
<br/>　　吕罡等几名道宗的弟子，个个面色怔怔，紧紧的凝视这一切。
<br/>　　苏莫能一剑打败李莽，让他们感到震惊，但现在天虚宗真传弟子武仁要亲自出手了，苏莫必将面临巨大的危机。
<br/>　　七大势力的真传弟子，可都是战力强大之辈，就算是最弱的真传弟子，战力都远超同阶武者。
