　　　　妖兽森林，十界灭生阵内。
<br/>　　萧逸仍旧在吸收地元金丹上的力量。
<br/>　　并不知晓裂天剑派发生的变故。
<br/>　　数天后，他的伤势已然完全恢复。
<br/>　　但，他还无法离开。
<br/>　　地元金丹只要一日还在，大阵就会继续维持。
<br/>　　只有将金丹上的力量吸收殆尽，他才能破阵而出。
<br/>　　恢复肉体，只消耗了金丹五分之一的力量。
<br/>　　不愧是地元境武者力量的标志，蕴含的力量，果真庞大。
<br/>　　金丹，还剩下五分之四的力量。
<br/>　　这些力量，吸收起来会非常快。
<br/>　　之前，恢复肉体伤势，要操控力量治愈体内五脏六腑、经脉以及四肢百骸。
<br/>　　这个过程，需小心翼翼，需细致。
<br/>　　这才耗费了比较多的时间。
<br/>　　足足用了大半个月有余。
<br/>　　现在，只是单纯吸收，速度会快许多。
<br/>　　……
<br/>　　体内两大武魂同时吸收。
<br/>　　地元金丹上的力量，几乎是以肉眼可见的速度在减少着。
<br/>　　金光熠熠的光芒，也在逐渐趋向黯淡。
<br/>　　冰鸾剑武魂的吸收速度，自不必多说。
<br/>　　控火兽武魂，晋升青色后，速度也极快。
<br/>　　金丹上的力量，通过两大武魂，转化为真气。
<br/>　　而后储存于小世界内的气泉。
<br/>　　萧逸甚至可以清晰感觉到，气泉在不断被填充。
<br/>　　这个速度，比以前吞服无杂质丹药时还要快得多。
<br/>　　他本就是洞玄九重修为，气泉填满近三千丈。
<br/>　　一天后。
<br/>　　体内三千三百三十三丈的气泉，已然填满。
<br/>　　再继续修炼下去，便是破玄境了。
<br/>　　洞玄境，突破破玄境的过程。
<br/>　　便是将已经填满的气泉内的真气，继续吸收真气，不断挤，而后强行压缩。
<br/>　　将空气般的真气，压缩为水一般的真液。
<br/>　　或者，可以理解为。
<br/>　　要给气泉填入真正的泉水，而并非飘散的空气。
<br/>　　这是武者一次整体上的飞跃。
<br/>　　单从实力上讲，这就是一次质变。
<br/>　　破玄境武者，之所以强大，也是因为这真液。
<br/>　　打个比方。
<br/>　　用长宽高各一丈体积的空气，去打人。
<br/>　　空气稀疏而无力，只会如棉花糖一般。
<br/>　　用长宽高各一丈体积的水，去打人。
<br/>　　水密集而富有冲击力，爆发力极强。
<br/>　　武者调动真液，无论是普通攻击，还是释放武技。
<br/>　　其威力，都远超以往。
<br/>　　言归正传。
<br/>　　萧逸脑海中回想着破玄境武者的武道知识。
<br/>　　一边思索着该做的步骤。
<br/>　　数分钟后，正式开始突破。
<br/>　　冰鸾剑、控火兽，双武魂齐开。
<br/>　　地元金丹上的力量，再次被快速吸入体内。
<br/>　　通过武魂，转化为真气。
<br/>　　气泉，本身已满。
<br/>　　再强行加入，便是在压缩。
<br/>　　两天后。
<br/>　　萧逸从修炼中醒来，眼中一道精光闪过。
<br/>　　“破玄境，终于成了。”
<br/>　　内视一番。
<br/>　　体内气泉中，真气早已没有。
<br/>　　取而代之的，是一股乳白色的精纯力量。
<br/>　　这便是真液。
<br/>　　真液自动流转，生生不息。
<br/>　　让得气泉，显得更像一个充满生机的水泉。
<br/>　　真液的高度，恰好在三百三十三丈。
<br/>　　代表着，如今只填满了一成。
<br/>　　自己的修为，便是破玄一重。
<br/>　　待得再次将气泉填满，届时便是破玄九重。
<br/>　　“真液，液态的真气。”萧逸自语着，试了试。
<br/>　　紫炎从手中凝聚而出。
<br/>　　旺盛的火焰，散发着骇人的气息，显然比以往强大得多。
<br/>　　顺带一提，气泉内的，仍旧是真气。
<br/>　　只不过成为液态罢了。
<br/>　　萧逸抬起头，看了眼地元金丹。
<br/>　　金丹上，仍旧有许多力量。
<br/>　　萧逸停歇了一会儿，便继续吸收起来。
<br/>　　数天后。
<br/>　　待得金丹光芒散尽，内里力量被吸收殆尽。
<br/>　　萧逸的修为，已达破玄三重。
<br/>　　体内气泉，填满了三成。
<br/>　　没了地元金丹的支撑，十界灭生阵，渐渐开始消散。
<br/>　　这时，萧逸也准备停下修炼。
<br/>　　体内控火兽武魂，已然停止吸收。
<br/>　　但，他却惊骇地发现，冰鸾剑，并没有停下。
<br/>　　它，似乎还在吸收着什么。
<br/>　　半晌，萧逸终于知道了。
<br/>　　十界灭生阵上，那即将熄灭的十界灭生火，竟化作一丝丝青色光芒，陡然被吸入他的体内。
<br/>　　“这…”萧逸一惊。
<br/>　　同一时间，他内视一番。
<br/>　　原本已经停止吸收，呆头呆脑的控火兽武魂，此刻竟活跃了起来。
<br/>　　那一丝丝青色光芒，被冰鸾剑吸收后，转而进入控火兽武魂内。
<br/>　　“难道…”萧逸似是想到了什么，脸色一阵惊喜。
<br/>　　数分钟后，待得十界灭生阵彻底消散。
<br/>　　冰鸾剑才停下吸收。
<br/>　　而这时，萧逸明显能感觉到，自己通过控火兽武魂，掌控了另外一种火焰。
<br/>　　“嘭。”萧逸左手虚脱。
<br/>　　一股青色火焰，猛地被凝聚出来。
<br/>　　正是十界灭生火。
<br/>　　“去。”萧逸的控火本事，早已炉火纯青。
<br/>　　十界灭生火，被他挥手打向不远处的一棵参天巨树上。
<br/>　　青色火焰，瞬间包裹了参天巨树。
<br/>　　巨树，并没有燃烧起来。
<br/>　　但，它的树木气息，却在快速消散。
<br/>　　不过数秒，萧逸收回了青色火焰。
<br/>　　一阵微风吹来。
<br/>　　刷的一声，刚才那棵参天巨树，被吹散成一地碎屑。
<br/>　　“好诡异的火焰。”萧逸瞳孔一缩。
<br/>　　如果说，紫晶灵炎，是狂暴，是焚毁万物。
<br/>　　那么，十界灭生火，便是诡异，焚烧生机。
<br/>　　“呵呵。”萧逸喜悦一笑。
<br/>　　果然是富贵险中求。
<br/>　　自己遇到黑袍老者，如在钢丝上走路，危险无比。
<br/>　　但其收获，便是现在的破玄三重修为。
<br/>　　以及掌握了操控十界灭生火的能力。
<br/>　　无论哪一样，自己都不枉涉险一番。
<br/>　　更别说两样皆得。
<br/>　　笑罢，萧逸走到黑袍老者的尸体旁。
<br/>　　摸索了一番，从中找出一个乾坤袋。
<br/>　　萧逸打开来看了看，这是个高级乾坤袋。
<br/>　　没有武技，没有功法，没有武器。
<br/>　　只有一堆黑魔殿成员的服饰，一些尚算珍贵的天材地宝。
<br/>　　百多粒五级妖兽内丹，百多瓶五级妖兽精血。
<br/>　　两颗星芒石。
<br/>　　最后是一些银票，加起来大概在数百万两左右。
<br/>　　想来，黑袍老者在这妖兽森林中，斩杀过不少妖兽。
<br/>　　而这些战利品，现在都归了萧逸。
<br/>　　倒是那两颗星芒石，也不见这黑袍老者有武器，也不知他用来做什么。
<br/>　　萧逸也懒得多想，直接将这些东西都放到自己的乾坤戒中。
<br/>　　乾坤戒的空间，比乾坤袋大太多了。
<br/>　　而后，拿出一颗星芒石，将烈焰手套提升为上品灵器。
<br/>　　烈焰手套，本只是下品灵器。
<br/>　　之前对他的实力增幅已经很少了，便几乎很少用。
<br/>　　现在，晋升为上品灵器。
<br/>　　将是他的又一大助力。
<br/>　　“呼。”萧逸深呼吸一口气，准备离开了。
<br/>　　临走前，将黑袍老者的尸体，放入了乾坤袋。
<br/>　　乾坤袋，不能装活物，但尸体是能装的。
<br/>　　这黑魔殿长老，可是猎妖殿通缉榜之首，奖励极丰富。
<br/>　　萧逸自然不可能放过。
<br/>　　换了一身衣服，戴上幽魂面具后，萧逸身影一闪，离开了妖兽森林。
