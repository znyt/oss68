　　离开后堂，萧逸前往任务处，交接了任务。
<br/>　　再次接了大量任务后，离开衡元主殿。
<br/>　　紫炎火翼一振，御空飞离。
<br/>　　他来衡元主殿之前，已经将这个地域的邪修清剿殆尽。
<br/>　　接下来，自然是赶往下一个地域。
<br/>　　……
<br/>　　数天后。
<br/>　　萧逸连跨近10个地域。
<br/>　　“怎么回事？”萧逸的身影，停在高空中，眉头紧皱。
<br/>　　这数天时间，他每过一个地域，都会凝聚太阴太阳之眼，以天地之力的形式感知地域内的邪修位置。
<br/>　　可在他的感知中，这些天路过的地域，竟皆是邪修稀少。
<br/>　　甚至于他特地寻到的邪修分部，只有寥寥数人。
<br/>　　“怎么可能。”萧逸眉头紧皱着。
<br/>　　之前一月时间，他横扫近百地域，每过一个地域，暗中必然都藏着为数不少的邪修。
<br/>　　一个邪修分部内，起码有过百人。
<br/>　　可现今这几天，所过地域的邪修分部，却人迹稀少？
<br/>　　难不成是他这些天杀怕了这些邪修？
<br/>　　这些邪修也猜错出他的行走路线，故特遣撤离了？
<br/>　　不，萧逸相信，绝不会是这个原因。
<br/>　　这些邪修，猖獗惯了，行踪虽诡秘低调，但行事却高调得很。
<br/>　　每每那些屠城之事，操控妖兽爆发兽潮之事等等，哪一件不是闹得整个地域人人皆知？
<br/>　　这些邪修，怎可能因此而被杀怕。
<br/>　　萧逸特地落到一个地域之内的主殿查过。
<br/>　　前几天，还不时有地域内的强者和天骄神秘失踪之事。
<br/>　　“怎么回事？”萧逸眉头紧皱，直觉告诉他，事情不寻常。
<br/>　　嗖…
<br/>　　紫炎火翼一振，再次疾速飞行。
<br/>　　一天后，萧逸横跨数个地域，随后停了下来。
<br/>　　思索了一阵，直视前方。
<br/>　　“前方，便是火牙地域的范围吧？”萧逸皱了皱眉。
<br/>　　火牙地域，可不是个普通地域。
<br/>　　在东面范围这边，可是一等一的大型地域。
<br/>　　虽然同样不是闻名地域，但也是个极强的地域。
<br/>　　论武者势力，自然比不上剑域、天藏地域那等闻名地域。
<br/>　　但也比黑铁地域、衡元地域这等大型地域要强得多。
<br/>　　嗖…身影一闪，御空飞行。
<br/>　　进入火牙地域，萧逸习惯性地放出感知。
<br/>　　以天地之力的形式，感知整个地域的邪修位置。
<br/>　　这一感知之下，萧逸脸色大变。
<br/>　　“嘶。”萧逸不禁倒吸一口凉气。
<br/>　　在他的感知中，整个火牙地域的黑气，密密麻麻。
<br/>　　在火牙地域的最中心部分，黑气几乎囊括了大片。
<br/>　　这证明，整个火牙地域的邪修数量，多到了一个恐怖的地步。
<br/>　　中心部分，必然是邪修分部，里头的邪修，起码达万之数。
<br/>　　里头邪修所修之邪道力量加起来，竟然汇聚成一股惊人的邪道之力。
<br/>　　这股邪道之力，在萧逸的感知中，便成了一大片的黑气范围。
<br/>　　“该死，火牙地域被邪修攻陷了不成？”萧逸面具之下的脸色，陡然一变。
<br/>　　单单邪修分部里头，已有过万邪修。
<br/>　　整个火牙地域分部四周的邪修，绝对在数万之数以上。
<br/>　　而且，在他的感知中，邪修分部之内，有不少的黑气气息，达到相当惊人的地步。
<br/>　　也就是说，里头有着不少修为极强的强者。
<br/>　　“嘶。”萧逸再次吸了口凉气。
<br/>　　这等数量和实力的邪修，怕是去攻陷一个霸主势力都足够了。
<br/>　　这般想着，萧逸手中乾坤戒光芒一闪，取出了一套黑袍。
<br/>　　黑袍覆盖在身，遮挡了身躯。
<br/>　　他不知道这片地域的邪修，有没有特意去注意他。
<br/>　　若有的话，他穿个黑袍，大概伪装一番，倒是可保意外。
<br/>　　若是直接去杀人，萧逸倒不必这般乔装。
<br/>　　但他打算先进入火牙地域的中心部分查探一番。
<br/>　　……
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸收起了紫炎火翼，闪身前行。
<br/>　　途中，也路过地域内的普通大城。
<br/>　　这些大城之内，看起来倒是正常，并无什么异况。
<br/>　　萧逸皱眉，继续前行。
<br/>　　一个多时辰后，萧逸在火牙地域的中心大城，火牙城停下身上。
<br/>　　在他之前的感知中，便是火牙城这里，聚集了数量惊人的邪修。
<br/>　　再次祭出太阴太阳之眼，以天地之力的形式感知，确定下邪修分部的位置。
<br/>　　嗖…萧逸闪身前行，数分钟后，在一座高山之前停下。
<br/>　　“火牙宗？”萧逸皱了皱眉。
<br/>　　火牙宗，可是火牙地域的武道圣地，第一大宗门。
<br/>　　其宗门，坐落在火牙城边缘的一座巍峨高山之上。
<br/>　　这时，萧逸明显注意到，有不少行色匆匆的武者，前往火牙宗。
<br/>　　而同时，火牙宗内弟子，则忙碌着接待着这些武者进入宗门大山。
<br/>　　不过，这些武者，大多着装正常。
<br/>　　反倒是萧逸的一身黑袍，引来了不少目光。
<br/>　　萧逸刚打算闪身到一边，脱下黑袍。
<br/>　　但想了想，还是不脱。
<br/>　　同样的，在中域行走的武者，一身黑袍的，多了去了。
<br/>　　判定是否邪修或者正常武者，又不是因为服饰。
<br/>　　总有武者有难言之隐，喜欢身穿黑袍。
<br/>　　萧逸继续前行，刚没走几步，一个身穿火牙宗弟子服饰的武者，一把拉住了萧逸。
<br/>　　“该死，你怎地穿这等服饰前来？”这个火牙宗弟子急声道。
<br/>　　“你们分部呢？不，你的令牌呢？”
<br/>　　“什么令牌？”萧逸皱眉问道。
<br/>　　宽大的黑袍，遮盖了他的面容，只露出眼睛。
<br/>　　火牙宗弟子闻言，显示眉头一皱，随后脸色一变，笑道，“哦，原来是来参加火牙盛事的武者吗？”
<br/>　　“刚才在下失言，胡说八道，还请不要放在心上。”
<br/>　　“莫名其妙。”萧逸冷哼了一声，越过这个火牙宗弟子，自顾前行。
<br/>　　身后，火牙宗弟子松了口气，随后看向萧逸的背影，面露阴蛰之色，“一个死人罢了，老子懒得和你计较。”
<br/>　　火牙宗弟子只是低声说了一句。
<br/>　　却不知晓，他的一举一动，一直在萧逸的感知注视之下。
<br/>　　如果没有猜错的话，刚才这火牙宗弟子，必然是误将他当成了邪修分部的人。
<br/>　　火牙宗这里，必然就是隐藏在火牙地域的邪修分部。
<br/>　　“火牙盛事？”萧逸皱了皱眉。
<br/>　　这火牙盛事里头，也必有猫腻。
<br/>　　萧逸看了眼周遭，这时，仍旧陆续有不少武者前往火牙宗山门。
<br/>　　可这里头，有不少武者都只是正常武者。
<br/>　　火牙宗内，聚集了大量且实力惊人的邪修。
<br/>　　火牙宗外，则有大量普通武者前往。
<br/>　　这个邪修分部，到底要做什么？
