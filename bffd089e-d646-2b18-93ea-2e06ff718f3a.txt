叶翎去了洗手间，用冷水拍了拍小脸，她走出门。
<br/>
<br/>        走在回廊里，迎面就来了一个富二代，富二代看见叶翎双眼都直了，“叶美人，你竟然回海城了？”
<br/>
<br/>        富二代目光肆意的看着叶翎，叶翎走的那一年18岁，还没有来得及绽放，但是这两年她却如玫瑰花般绽放了，明媚精致的五官上镶嵌了一双勾人的水眸，下面是娇艳欲滴的红唇，对男人散发着致命的吸引力。
<br/>
<br/>        富二代又看了看叶翎这连外媒都评价成天生尤物的身段，复古酒红的宫廷小衣衬的她肤如凝脂，茶色的卷发慵懒散落肩头，平添几分旖旎和香艳。
<br/>
<br/>        “叶美人，这两年你都成为大明星了，我们想见你都只能在电视上看到，今天既然这么巧碰到了，那我请你喝一杯。”富二代邪笑道。
<br/>
<br/>        叶翎停在原地，她挑着勾人的水眸看着那个富二代，“想请我喝一杯的男人多了去了，你，算是哪根葱？”
<br/>
<br/>        富二代脸一冷，“叶翎，不要给你脸，你不要脸，不就是一个戏子么，还不是谁出的价高就陪谁喝酒！”
<br/>
<br/>        叶翎慵懒的勾唇，“既然知道我是有市场价的，那就去打听一下我的行情，想让我陪你喝酒，先看看自己兜里的钱够不够。”
<br/>
<br/>        “你！”
<br/>
<br/>        富二代觉得自己被羞辱到了，以前被护在顾夜瑾羽翼里的叶翎乖巧软糯的像花骨朵，而现在的叶翎高冷，倨傲，妩媚，冷艳，就像是一朵带刺的红色玫瑰。
<br/>
<br/>        这样的女人很容易激起男人骨子里的征服欲。
<br/>
<br/>        富二代露出了势在必得的猥琐光芒，“叶翎，你不过就是顾家的养女，这两年你出道都是有顾夜瑾护着，一路给你保驾护航，你最好确保自己能够永远获得顾夜瑾的宠爱，不被他抛弃，一旦你被他抛弃，拥有了这么一张脸蛋和身段的你，一定会沦为男人的玩物！”
<br/>
<br/>        叶翎看着那个富二代，“作为女人，怎么获得顾夜瑾的宠爱是我的事，而作为男人，你应该做的是打败顾夜瑾，让你成为我的金主，而不是等在顾夜瑾身后捡漏，我很看不起你。”
<br/>
<br/>        “…”
<br/>
<br/>        富二代没想到叶翎这么的伶牙俐齿，真是句句戳他心窝子，反正这里没人，如果他对她做些什么，也没人发现。
<br/>
<br/>        这样的邪念一旦产生了，就遏制不住，富二代迅速上前，伸手就去抓叶翎，“叶翎，今天我就要尝一尝海城第一美人究竟是怎样销魂的滋味！”
<br/>
<br/>        叶翎冷笑，咸猪手伸过来的时候，她就站在原地，动都没有动。
<br/>
<br/>        下一秒身后就传来了一道清寒凛冽的嗓音，“你敢动她一下试试看！”
