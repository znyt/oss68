　　一月时间，转瞬便过。
<br/>　　这段时间，萧逸都奔走于到商行购买药材，而后炼药，与其他猎妖队交易。
<br/>　　炼药师之所以赚钱快，就是因为炼制出的丹药，比所用材料价值高10倍乃至20倍甚至更多。
<br/>　　特别萧逸炼制的还是完美级别的丹药，能作为硬通货。
<br/>　　不过是到商行买了二百万左右的药材，便炼制出了两千多颗四品级别的完美丹药。
<br/>　　东荒十八城，武者众多。
<br/>　　相对的，商行也有许多。
<br/>　　疾风猎妖队总部。
<br/>　　萧逸看着那一堆堆乾坤袋，心中一阵激动。
<br/>　　“易霄大师。”宁浩说道，“这便是十五支猎妖队的商物。”
<br/>　　“每支猎妖队的商物都差不多，四级妖兽精血200，内丹200；三级妖兽精血2000左右，内丹2000左右。”
<br/>　　萧逸点点头，道，“没闹出什么不愉快吧？”
<br/>　　“没有。”宁浩笑道，“他们知道易霄大师是拿完美级别的丹药和他们交易，而且价格更高，可乐坏了。”
<br/>　　“那就好。”萧逸点点头。
<br/>　　萧逸大手一挥，将这堆乾坤袋一一收起。
<br/>　　“宁浩队长，易某还有要事，便不多留了。此次相助，易某记于心中，日后有易某能帮得上的地方，尽管说。”
<br/>　　“易霄大师客气了。”宁浩拱了拱手。
<br/>　　……
<br/>　　乾坤袋之间，是无法重复放入的。
<br/>　　这么大堆乾坤袋，萧逸也难拿，只能用灵气包裹住，而后化出紫炎双翼，振翅离去。
<br/>　　他没有去妖兽森林。
<br/>　　而是去了城外的荒野之地。
<br/>　　东荒十八城，呈弧度分布。
<br/>　　前面是辽阔的妖兽森林，后面则是接连其他大城。
<br/>　　城与城之间，除了官道便是荒郊野岭。
<br/>　　大概飞了数百里左右，萧逸便停了下来。
<br/>　　随便寻了座小山，一拳轰出个山洞，做好掩饰，便走了进去。
<br/>　　他打算炼制无杂质打断，彻底破丹化脉。
<br/>　　这些荒郊野岭，虽不如妖兽森林般危险，但始终有些弱小的妖兽生活在其中。
<br/>　　而突破之时，最忌打扰，萧逸自然要做好万全准备。
<br/>　　盘膝坐下，萧逸拿出了离火金炉，各种天材地宝，以及妖兽内丹，彻底开始炼药。
<br/>　　天材地宝，他本身就极多，不愁不够。
<br/>　　妖兽内丹，算上之前自己拥有的，这些月来自己猎杀所得的，还有16支猎妖队交易得来的。
<br/>　　四级内丹，3800颗。
<br/>　　三级内丹，近40000颗。
<br/>　　接下来的日子，他都在没日没夜的炼药。
<br/>　　足足十天过后，才停了下来。
<br/>　　过度的疲劳，让他脸色苍白，身体有些虚弱。
<br/>　　但其成果，也是相当惊人。
<br/>　　他每炼制一堆丹药，便会放入乾坤袋。
<br/>　　如今，十数个乾坤袋，被装得满满的。
<br/>　　他相信，若是将里头的丹药全部倒出来，必定堆积如山。
<br/>　　无杂质洞玄丹，1900颗。
<br/>　　无杂质先天丹，20000颗。
<br/>　　每炼制一颗无杂质丹药，便要耗费两颗妖兽内丹。
<br/>　　“若非我会炼药，而且炼制速度够快的话，十年都不一定能存够这么多丹药。”
<br/>　　看着面前的乾坤袋，萧逸一阵感慨。
<br/>　　这两万多颗丹药，若真论价值，那简直一笔天文数字。
<br/>　　单单材料，天材地宝、妖兽内丹，加起来就逾数千万两。
<br/>　　这还是妖兽内丹直接通过猎妖队交易，价格更便宜。
<br/>　　若是按正常途径，到各大商行买，这个价格会飙升个数倍。
<br/>　　“离开剑派，近8个月，如今才完全做好准备。”
<br/>　　“我倒要看看，凝聚出来的，是何等气泉，竟让我如此费尽心机。”
<br/>　　萧逸嘴角扬过一丝期待的笑意。
<br/>　　一颗颗丹药入腹，快速化为庞大药力。
<br/>　　正常来说，就算是无杂质丹药，也要半个时辰左右，药力才会被完全吸收。
<br/>　　但萧逸有冰鸾剑和控火兽，两大武魂同时吸收，药力瞬间就能吸收掉。
<br/>　　当丹药服用到五千颗时，冰火气丹和九大真脉，渐渐出现了融化的迹象。
<br/>　　顺带一提，上次在裂天剑派吞服的三百里无杂质洞玄丹，并非没有效果。
<br/>　　只是效果看起来不明显，大概在一成左右。
<br/>　　也就是说，其实冰火气丹和九大真脉，之前就已经融化了些许。
<br/>　　现在的五千颗丹药，是在上次的基础上，继续融化着气丹和真脉。
<br/>　　待得20000颗先天丹完全吞服。
<br/>　　冰火气丹和九大真脉，已经融化了大半。
<br/>　　与此同时，萧逸脸上的喜意更为浓郁。
<br/>　　他知道，自己对丹药所需的数量，并没有预估错误。
<br/>　　剩下的1900颗无杂质洞玄丹，足以将气丹和真脉完全融化。
<br/>　　果然，当剩下的丹药完全吞服，药力被吸收。
<br/>　　破丹化脉这个过程，已然完成。
<br/>　　气丹为基，真脉为材，方成气泉与小世界。
<br/>　　体内，融化后的气丹与真脉，化作了一团奇怪的能量。
<br/>　　能量互相交融，互相碰撞。
<br/>　　接下来，便是凝气泉的步骤了。
<br/>　　这个步骤，以前易老已经演示过，萧逸实施起来，如行云流水般，毫无阻碍。
<br/>　　不多时，当着个步骤结束后。萧逸身上，猛地爆发出一股惊天气势。
<br/>　　轰的一声，周遭灵气瞬间暴走。
<br/>　　这是修为境界突破的征兆。
<br/>　　此刻，萧逸已经踏入洞玄境。
<br/>　　顾不上欣喜，他马上内视一番。
<br/>　　体内，冰火气丹、九条真脉，早已消失。
<br/>　　取而代之的是，原本气丹的位置，出现了一团稳定的能量。
<br/>　　透过能量，萧逸赫然看到一个巨大无比的世界。
<br/>　　那就是小世界，也是属于他体内的一方天地。
<br/>　　除了他之外，任何人休想窥探半分。
<br/>　　这片天地，便是小世界，长宽高，各六千六百六十六丈。
<br/>　　不多一分，不差一毫。
<br/>　　而在小世界的最中央，便是一个巨大的干涸气泉。
<br/>　　气泉，长宽高，各三千三百三十三丈。
<br/>　　同样，不多一分，不差一毫。
<br/>　　“三千三百三十三丈，嘶。”萧逸倒吸一口凉气。
<br/>　　在这巨大气泉之上，横亘着两个武魂。
<br/>　　正是冰鸾剑和控火兽。
<br/>　　成就小世界后，武者体内的武魂，也会随之进入小世界。
<br/>　　小世界的大小和强度，必须能够承载自身武魂进入。
<br/>　　这也是为什么小世界的大小，会和武魂品阶有关。
<br/>　　萧逸蓦然想起了易老之前的话。
<br/>　　裂天剑派外门最出色的天才，气泉长宽高各一百五十丈。
<br/>　　内门天才，达二百丈，最出色的，达三百丈。
<br/>　　普通天才与超级天才之间，也不过是差距百来丈。
<br/>　　自己一下子就跳到三千多丈。
<br/>　　而且，以易老的见识，在他口中，五百丈，已是传说中的绝世天才，甚至他根本没有见过。
<br/>　　可想而知，自己这个三千多丈的气泉，变态到什么地步。
<br/>　　不过萧逸转头一想，自己的气泉，是要承载冰鸾剑的。
<br/>　　也许，也只有这种恐怖程度的气泉，才配得起承载冰鸾剑。
<br/>　　“喝。”萧逸只觉浑身有劲，大喝了一声。
<br/>　　一拳震碎山洞的石头，飞跃而出。
<br/>　　现在，他没有凝聚紫炎火翼，而是单纯靠自己的本事，御空飞行。
<br/>　　达到洞玄境后，他明显感觉到，通过体内小世界的联系，自己对这个真正世界，有着一定的掌控力。
<br/>　　天地间的灵气，宛如一个个精灵，在他眼中飞舞，且受他控制。
<br/>　　“凝。”萧逸大喝一声。
<br/>　　霎那间，方圆万米之内的天地灵气澎湃朝他涌来。
<br/>　　犹记得，当初慕容墨禁锢他时，最多也不过能操控方圆数百米的天地灵气。
<br/>(本章完)
