　　玄天成为混元大罗金仙，顿时在洪荒之中掀起无尽风暴。对此，每个人的反应都各有不同。
<br/>　　对于人族来说，这自然是最值得高兴的，玄天为人族，如今成了混元强者，人族的地位自然也就顺势涨了许多，洪荒中其他种族想要对付人族，至少要考虑考虑玄天这位强者的态度。
<br/>　　其中种族对于人族自然是又敬又怕，敬畏的自然是人族的强大，怕的是玄天这位混元强者若是有一天想要对付他们，他们完全没有一点抵抗力。
<br/>　　洪荒诸神们，对于玄天成为混元大罗金仙一事，自然是心里极为复杂，他们既希望玄天成功，那样像他们这样没有鸿蒙紫气的准圣，便有机会成为混元了，同时又不希望他成功，若是他真的突破了，洪荒便会又多出一位站在他们头顶，主宰着他们命运的强者。
<br/>　　不过，如今玄天已经成功突破了，这让他们松了一口气的同时，又有些郁闷，怎么别人都能突破，就自己不能突破，难道自己真的比别人差吗？
<br/>　　诸圣的态度就有些微妙了，站在圣人的角度上来看，自然是不希望有人突破，与他们齐肩，那样圣人的威压便会不断被削弱。其实他们很想在玄天突破之时，便去阻止他，但却被元始阻止了，谁也不知道那是为什么。
<br/>　　玄天的突破，身为本尊的元始自然是极为高兴的，毕竟，‘逝我’的突破，对于他来说，也是帮助极大的，‘逝我’越强，将来归一之时，他的实力增长的也会越强。
<br/>　　玄天突破后，并没有高调的去做别的事，而是如万劫魔尊一般，隐藏了起来，潜心悟道修炼，不遇到大事，一般不会出现在人族之中。
<br/>　　人族之中，在伏羲的带领之下，越来越强盛了，人族的数量也在急剧增加。人口的增长，随之而来的便是各种问题的出现。住房问题，食物问题，灾病问题等，颇为严重。
<br/>　　其中，最严重的便是食物问题。人口的剧增，人类的一般的猎食、采摘野果等方式已经不能满足人类的需求了，人族中，一年之中，便会有大量的人数是死在食物紧缺之上的。
<br/>　　对此，如今已经长大的姜姓烈山氏神农，颇为在意这件事。为了解决人族食物问题，他日思夜想，可谓是绞尽脑汁，但就是没有想到很好的办法，来解决人族食物问题。
<br/>　　对于武者来说，还好，他们已经有能力能够自己出去猎食，而对于普通人来说，捕猎是一件十分危险的事，野外随时会伴随着各种危险出现，一个不小心，便会丧失生命。而人族之中，普通人恰恰又是最多了，他们的食物问题，便成了最大的麻烦。
<br/>　　这日，神农来到野外，希望能够找到一种方法，解决人族的食物问题。
<br/>　　就在此时，他看到一只彩色神鸟从他面前飞过，嘴中衔着一株五彩九穗谷，掠过神农时，它嘴中的九穗谷掉落在地上。
<br/>　　神农见了，脑中顿时灵光一闪，像是什么被打开了。他上前几步，抓起地上的九穗谷，，仔细打量一翻，瞬间便想到了许多：
<br/>　　“鸟既然可以以穗谷为食，那人是否也可以呢？还有，不仅穗谷，其它的植物，别的动物能吃，为何人类不能吃呢？”
<br/>　　神农想了一会儿，他脑海之中的智慧之门仿佛是被打开了，思维一下子扩散开来，他一下子便想到了很远。
<br/>　　神农再次搜寻一些九橞谷，连忙回到人族部落，寻得一块空地，将九橞谷埋在了土里，以水浇灌，每日精心照料。后来，此地长成了一片九橞谷，神农见此，将九橞谷收了起来，放到嘴中尝了尝，感觉可以吃，他心中顿时大喜。
<br/>　　于是，神农连忙将其他动物吃过的植物寻来，按照一些植物的习性，种植了起来。种植过程中，自然也遇到了许多问题，但后来都被神农一一解决了。
<br/>　　此过程中，他发现，有的植物能够吃，而有的却不能吃。为此，神农还经常误食有毒之物，若不是他跟随玄都学习修行之道，更是学习了炼丹之术，精通丹药，他恐怕早已身死了。
<br/>　　不过，经过多次尝试，最终还是让他找到了一些有用的食物，在众多食物之中，神农最终筛选出稻、黍、稷、麦、菽五谷，最为人族的主要食物。
<br/>　　之后，神农在部落之中，大肆传授播种五谷的技巧。起初，还有许多人不信，但有人抱着试一试的态度，依照神农之言，种植了起来，最终果然得到了许多食物，而且人类完全可以吃。
<br/>　　人族大喜！
<br/>　　后来，在神农的带领下，越来越多的人种植起五谷来，他们砍伐树木，收割野草，用石头做成各种斧头、锄头等工具，开垦土地，种植起了五谷之物。
<br/>　　五谷解决了人类的很大一部分粮食问题，族人的生活开始好转，其他部落听闻此事，也开始向神农部落学习种植五谷之术，渐渐地，这种五谷粮食，在人族之中得到了广泛的传播，人族之中的食物问题，一下子被解决了一大半，烈山氏神农的威望，一时之间，风头两无，后人尊称神农为“农皇爷”！
<br/>　　姜氏部落也在这段时间内，壮大了起来，渐渐堪比伏羲的风氏部落。
<br/>　　身为人族共主的伏羲，此时也听闻了神农的事情，不由对神农大感兴趣。
<br/>　　此时的伏羲，已经在位四百九十年了，再过十年，便该退位了。人族之中，有过规定，共主最多只能在位五百年，便要退位让贤。
<br/>　　况且，自从伏羲创出先天八卦后，便能够推算出自己的未来，经过近五百年的人族共主之位，人族气运的汇聚，天地主角气运的汇聚，让伏羲的修为达到了转轮圆满境，他知道，自己已经功德圆满了，是时候证道天皇了。
<br/>　　这段时间，伏羲也一直在找能够继承自己，成为人族共主的人，如今神农的出现，顿时让伏羲为之一喜。他一阵推算，发现自己竟然推算不出神农的命格，只能知道，神农乃是圣人门下，人族大贤，在他的命格之中，隐隐带着一股煌煌大日的威严。
<br/>　　伏羲顿时便知道了，神农不简单，他可能就是自己要找的那个人。
<br/>　　于是，伏羲决定拜访一下这位神农。伏羲来到姜氏部落，不管其他人的激动之情，直接召见了神农氏。
<br/>　　伏羲将神农带到一个屋子中，两人纵天谈地，讨论人族事宜，大多数是伏羲在问，神农氏回答，伏羲一边问，一边观察神农。一番回答下来，他对这位神农氏很满意，他在治理人族的一些见解上，并不在自己之下，有些地方，甚至还犹有过之，特别是农业方面。
<br/>　　知己啊！
<br/>　　伏羲当下便要求与神农拜把子，称兄弟。神农自无不可，一番结拜之后，两人你望望我，我看看你，只觉得相见恨晚！
