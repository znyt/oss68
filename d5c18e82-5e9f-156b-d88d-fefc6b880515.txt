        听到这儿，明司寒忍俊不禁地打趣道：“傅氏集团的网络，是国际排行第一的白客替他们做的安防系统。不是外人随随便便想黑就能黑的。再说了，那个洛羽说不定是傅夜沉的私生子呢？”
<br/>
<br/>        “好吧！那我就只能认倒霉了！”明希哲扶着额头，又躺回了沙发上。
<br/>
<br/>        不知道为什么，此时此刻，明司寒又想起了那个女孩。
<br/>
<br/>        从他得知，她就是自己收的那个徒儿“Star大小姐”起，他对她的感觉便有了些变化。
<br/>
<br/>        其实，以前在游戏里聊天的时候，他就对她的印象还不错，她给他的感觉就像是个软萌软萌可爱又有趣的小女生。
<br/>
<br/>        而且，她的真实面貌还跟他曾幻想过的样子差不多。
<br/>
<br/>        长得很漂亮，声音也很甜，只是那脾气……
<br/>
<br/>        明司寒暗自汗颜。
<br/>
<br/>        还真不敢恭维。
<br/>
<br/>        “喂！”明希哲抬起手来，在明司寒的眼前晃了晃，“你发什么呆啊？”
<br/>
<br/>        “没什么。”明司寒回过神来后，咧嘴笑了笑。
<br/>
<br/>        “你这又是开网咖，又是搞竞技的，什么时候能回本给我啊？”明希哲接着说道，亲兄弟也得明算账。
<br/>
<br/>        明司寒从父亲那儿拿了八千万，又从他这个二哥这里借了两千万，加起来刚好一个亿去创业。
<br/>
<br/>        明希哲知道自己的这个弟弟是搞IT这块的，但真不知道他搞出了什么名堂出来。
<br/>
<br/>        开个网咖一年也赚不了多少钱吧？
<br/>
<br/>        至于竞技，感觉就像是在赔钱。
<br/>
<br/>        明司寒却得意地笑了笑：“明年年底，连本带利地还给你。”
<br/>
<br/>        “你打算怎么还？嗯？”明希哲接着问道。
<br/>
<br/>        明司寒抬起手来拍了拍明希哲的肩膀，会心一笑：“你们拍电影所用到的特效，我也在做。还有游戏和动漫这块。”
<br/>
<br/>        “你一个人？”明希哲顿时难以置信地看着明司寒。
<br/>
<br/>        明司寒笑道：“我有团队的！”
<br/>
<br/>        “那你一个亿够吗？”明希哲倒是关心起这个重点来。
<br/>
<br/>        明司寒摇了摇头：“当然不够，所以，我正打算找傅氏集团的傅大Boss谈投资的事情。”
<br/>
<br/>        “你一个在校大学生，人家大老板会愿意跟你谈投资？我看还是算了吧！”明希哲忍不住给明司寒泼冷水，顿了顿后，又接着说道，“我倒是有个主意，就是，你让老爸替你出马，一定马到功成。”
<br/>
<br/>        “要是老爸出马，以后就真的都没我什么事了。”明司寒一脸不悦地皱起了眉头。
<br/>
<br/>        明希哲懂明司寒的意思，于是端起酒杯，喝了一口：“那你就自己想办法吧！”
<br/>
<br/>        “我会的。”明司寒也跟着端起酒杯，碰了一下明希哲手中的杯子，随之将杯中的酒一饮而尽。
<br/>
<br/>        虽然说是说不醉不归，但是明希哲也不敢多贪杯，喝得差不多就停了，然后躺在沙发上休息。
<br/>
<br/>        明司寒在茶几前席地而坐，面前还架着一台笔记本电脑。
<br/>
<br/>        虽然他不知道她为什么也会在云城，但是他却能感觉到，下午的这个时间点，她一定在线。
<br/>
<br/>        果不其然，他登录游戏后，她果然在打游戏。
<br/>
<br/>        于是，他给她留了言。
<br/>
<br/>        徒儿，打完这局后，进师父开的房间。师父带你飞。——余寒犹厉。
<br/>
<br/>        好哒！师父！徒儿打完，马上来！——Star大小姐。
<br/>
<br/>        瞧吧！
<br/>
<br/>        网络世界里，他的女徒儿真的是个软萌的小女生。
<br/>
<br/>        明司寒下意识地望着自己前方的那堵墙。
<br/>
<br/>        而墙的另一边，同样是在客厅的茶几前，连星儿盘腿坐在垫子上，一手握着鼠标，一手在键盘上飞速地操作着。
<br/>
<br/>        结果，这局还是输了，队友太不给力了。

<br/>        连星儿退出当局后，立马进了师父所在的房间，然后和师父一起双排。

<br/>        而在这期间，他们从来没有在线下聊过，也没有在线上开过语音。

<br/>        但是，他们双排打游戏的时候，配合却十分默契，完全不需要语音交流。
<br/>
<br/>        就这样，她和这位师父一直保持着这种联系方式。

<br/>        所以，连星儿对她的这个师父可谓是崇拜得不得了。

<br/>        不过了……

<br/>        因为要了真实联系方式的话，就会涉及到要见面。
