　　林峰他刚解决掉了变异熊，就听到了这边的声音。
<br/>　　然后一路冲杀过来就看到了寒鸦拿着炸药，要去和两个变异的响尾蛇同归于尽，他当机立断就拿出了自己的主战坦克，然而，七辆坦克的齐射，炮弹都射完了，居然都无法奈何这些畜生。
<br/>　　所以林峰不再迟疑，选择自己亲自出手。
<br/>　　“这么硬？”
<br/>　　林峰不给这头变异响尾蛇任何反抗的机会，二话不说拔出斩灵剑就是站在蛇脑袋上一刀对着蛇的眼睛刺了下去。
<br/>　　噗嗤。
<br/>　　这一击。
<br/>　　直接把这头变异响尾蛇彻底击毙。
<br/>　　林峰拔出斩灵剑，然后就朝另外一个变异响尾蛇杀去。
<br/>　　嘶嘶。
<br/>　　这头变异响尾蛇彻底的发怒了。显然他看到自己老婆被人干掉了非常的生气。
<br/>　　天罡符！
<br/>　　噼里啪啦！
<br/>　　林峰拿出天罡符直接把这蛇劈的身上冒烟，身体也出现了痉挛。
<br/>　　刷刷刷就是几枚飞镖朝对方的蛇眼睛射去。
<br/>　　噗嗤噗嗤！
<br/>　　林峰的飞镖是百发百中，直接把另外一个变异响尾蛇的眼珠子也给射瞎了。
<br/>　　但蛇这种东西，他本来实力就不好，所以眼睛在不在对他来说没有多大的影响力。不过剧烈的疼痛却是确确实实存在的。疼的他就是张开血盆大嘴朝林峰撕咬而来。
<br/>　　来得好！
<br/>　　林峰拿出一堆汽油弹就是朝对方嘴巴里丢了过去。
<br/>　　然而林峰也被对方一嘴给生吞了。
<br/>　　“林峰！”
<br/>　　寒鸦等人回过神来纷纷面色大变。林峰战神居然被蛇生吞了！！！
<br/>　　然而下一刻他们就愣住了。
<br/>　　因为又一个林峰出现在了这个变异响尾蛇头顶上，一刀刺进了对方的眼珠。
<br/>　　随后林峰跳了下来，这个变异响尾蛇也是嘴巴里喷着火，直接被击毙。
<br/>　　“怎，怎么两个林峰。”
<br/>　　寒鸦他们都傻了眼了，自然不知道之前那个拿着汽油弹的林峰实际上就是个诱饵罢了，只是林峰的一个分身。
<br/>　　林峰以迅雷不及掩耳之势，搞定了两个九阶的变异响尾蛇，然后继续朝其余的八阶变异蛇斩杀而去，这几个变异蛇虽然也厉害，但是比九阶的要好对付多了，林峰直接给他们降龙十八掌全部拍黄瓜一样的拍死。
<br/>　　“总算是全部搞定了。”
<br/>　　林峰把这里的怪物全部斩杀殆尽，也是累的一身汗。
<br/>　　不过九阶八阶的怪物死了，但是依然还有上千只丧尸和数百只怪物在这个黑龙镇之中。
<br/>　　他们的威胁性依然无比的巨大。
<br/>　　但是林峰倒不是非常在意了，因为他就算无力再战，还有孙悟空和妲己外加千年僵尸，足够应付这些怪物的存在了。
<br/>　　最主要的是，战士会还有大概三百多人没死，他们原本绝望的眼神之中也开始充满战意和希望。更何况还有罗敏这个八阶韩风这个七阶。
<br/>　　足足五个小时候。
<br/>　　天色渐渐的暗沉下来。
<br/>　　肉眼看得见的丧尸全部被消灭，但不保证一些屋子里还有丧尸躲着。
<br/>　　不过到了晚上已经不利于行动了，倒不是因为晚上怪物大规模出没，这只是其一，最主要的是，他们晚上什么都看不清，黑黢黢一片，若是打手电，且不说奢侈，而是相当于直接给怪物当靶子了。
<br/>　　众人在一家镇子里最大的超市内安营扎寨。
<br/>　　每个人都大口的吃着罐头喝着农夫山泉。
<br/>　　这些吃的喝的他们来之前就已经准备好了。
<br/>　　因为他们来黑龙镇不是来旅游的，而是抱着把这里当做据点来发展的目标来的，所以除了武器弹药，食物辎重全部都备齐了。
<br/>　　吃着吃的居然有人就哭起来了。
<br/>　　来时上千人，可是到现在只剩下了三百人。
<br/>　　哪怕是十二个干事也死的只剩下了五个。
<br/>　　会长寒鸦也是丢了一条胳膊，这条丢掉的胳膊也不知道被那个变异狗给吃了，反正是肯定找不回来了。
<br/>　　寒鸦倒是看得很开，也不怎么在意。
<br/>　　罗敏倒是急的要死。
<br/>　　“林峰小兄弟，这一次多谢你了，要不是你，我们这群人可都要交代在这里了。”
<br/>　　“是啊林峰你可真厉害了，还真是英雄出少年。”
<br/>　　“牛逼！林峰，老子活了三十多岁，如今还是第一次对一个00后佩服的五体投地。”
<br/>　　几个没死的干事此时此刻那是由衷的佩服林峰了。
<br/>　　“是啊林峰，这一次真的不知道该说什么了，若不是你，我们战士会……”
<br/>　　寒鸦还要说什么，实际上他一肚子苦涩，不是林峰，他们就全完蛋了，他心里很愧疚。
<br/>　　林峰打断了，看了大家一眼，笑道：“干嘛这么煽情，感谢我什么？我也是战士会的，我也只是做了我该做的事情，尽人事听天命罢了。”
<br/>　　寒鸦闻言愣了愣，几个三十多岁的人似乎都没有想的林峰一个未成年，居然会把事情给看的如此的通透。
<br/>　　“佩服！”
<br/>　　每个人都竖起了大拇指。
<br/>　　众人一天下来也是各种疲倦，很快就响起了一片呼噜声。
<br/>　　翌日。
<br/>　　每个人都继续开始歼灭，这一次，是一间一间屋开始仔细的地毯式搜查。
<br/>　　到了下午。
<br/>　　所有威胁都被清理干净，然后每个人开始打扫战场。
<br/>　　那些还算完整的尸体一个个的摆在广场上。
<br/>　　足足几百具之多，到最后，甚至还找出了一些多的断胳膊断腿，就好像是一个玩具上多出来的零件。
<br/>　　这些都是被怪物或者被丧尸啃完剩下的残肢罢了。
<br/>　　每个人都沉默不语。
<br/>　　这些尸体是要运回去的。
<br/>　　剩下的工作就是挖晶石。
<br/>　　光是挖晶石就挖了足足一天。
<br/>　　一大堆的晶石全部摆在了一起。堆成了一座座的小山。
<br/>　　“林峰，这两个九阶的晶石归你。”
<br/>　　寒鸦想都没想就把其中最大的也是最珍贵的九阶晶石丢给了林峰。
<br/>　　这两个晶石每个都有人的脑袋那么大，人的脑袋，六斤半，这个晶石比这还要重的多。
<br/>　　其余人虽然都羡慕嫉妒，但是却没有任何人不认为同。
<br/>　　因为没有林峰，他们早就死了。
<br/>　　林峰自然也不矫情欣然接受。这的确是他应得的。
<br/>　　反正活着的人每个人都得到了许多晶石，林峰也分了一大堆，他也懒得去数具体多少，反正他只在意那些高级别的晶石。
<br/>　　到了晚上。
<br/>　　寒鸦带着一部分人回去了。
<br/>　　他们要把剩下的战士会重心全部转移到这个黑龙镇他们好不容易打下来的据点上。
<br/>　　然而林峰也回了次曙光基地。
<br/>　　几个出去抢夺据点的大势力。
<br/>　　军部成功了，他们拿到的是基地以东的两个大镇。
<br/>　　天神帮也成功了他们拿到的是基地以北的一个大镇。
<br/>　　骷髅会也成功了，他们拿到的是基地以南的一个镇。
<br/>　　而火神帮失败了，全军覆没，甚至帮主和副帮主也死了。出去的几百人无一活口。只是因为他们的目标小镇老龙口内有一个超出预料的A级怪物存在。一个A级怪物，轻而易举覆灭他们整个团队。
