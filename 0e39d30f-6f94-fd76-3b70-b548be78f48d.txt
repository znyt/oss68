　　不周神山，第九重天幕！
<br/>　　黑暗侵袭，空间不着点滴光彩，漫天黑暗，神识的作用被压制到最低。
<br/>　　此层空间，本该寂静无声，但此时，却因此处发生大战而变得汹涌澎湃，热闹非凡。
<br/>　　漆黑不见五指的空间，出现九个炙热夺目的太阳，照亮了黑暗空间，驱散黑暗，点亮光明，向着一个长着八只触角的怪物撞去。
<br/>　　“可恶！”那怪物见到九个太阳横空，本能的有些畏惧，黑暗对光明，本就存在属性克制，要是平时，他才不会将之放在眼里，但如今，他身受重伤，实力十不存一二，面对九日齐飞，他不得不慎重。
<br/>　　“黑暗泯灭！”
<br/>　　顿时，空间之中的黑暗，变成一道巨大的黑暗空洞，缓缓旋转，将一切吸入其中，越来越大，越来越强，气势宏大，泯灭一切，陡然之间，便将元始用天火凝聚的太阳吞没！
<br/>　　“嗯？有些麻烦。”元始看着朝自己卷来的巨大黑暗空洞，他能够感受到其中的巨大威力，要是一个不小心被卷入其中，就会被瞬间分解，化为飞灰湮灭。
<br/>　　“阴阳印！”
<br/>　　元始想了想，还是用阴阳印来应付此招，阴阳印进可攻，退可守，而且威力也都不弱，元始比较喜欢用它来对敌。
<br/>　　元始双手迅速结印，一阵玄妙而复杂的动作在一瞬间飞快完成，动作行云流水，充满道韵奥妙。顿时，一个巨大的阴阳太极图出现在虚空，卷向黑暗空洞。
<br/>　　当黑暗空洞撞向太极阴阳图后，它便不再前进，因为它就算想前进也不能前进，有阴阳印抵挡，它就不能前进。
<br/>　　而且，伴随着阴阳鱼的缓缓旋转，这漫天的黑暗之气却正在被阴阳鱼吸收，黑暗属阴，阴极阳化，阴阳相生，造化天地。
<br/>　　随着更多的黑暗之气被阴阳印所吸收，阴阳印也变得越来越强，吸收便变得越来越快，黑暗变得薄弱，空洞瞬间被瓦解，崩溃消失。
<br/>　　“混蛋！黑暗风暴！”
<br/>　　黑暗魔神见一招没有见效，甚至还被对方所利用，异常暴怒，本座堂堂混沌魔神，现在竟然没有拿下一个小辈，这怎么能行？故而，紧接着使出一招黑暗风暴。
<br/>　　黑暗席卷天地，变成一个巨大的黑暗龙卷风，迅速向元始卷来。声威浩大，气势磅礴，黑暗纵横，道韵压迫，这个空间在动荡。
<br/>　　“这便是黑暗法则么？果然强大！不过，声威竟然盖过了本座，这怎么行？”元始见到无尽黑暗朝自己卷来，揶揄了一声，周身大道道则翻滚，气势瞬间放大，散发着无量仙光，漫天黑暗之气根本不得近身。
<br/>　　黑暗魔神那一双恐怖而充满怨毒之色的眼睛冷冷的盯着元始，恨声道：“可恶的小辈，本座今天一定要将你……”
<br/>　　“开天印！”
<br/>　　元始哪会无聊、愚蠢到听黑暗魔神把话说完，陡然间，化作万丈之高。全身道韵笼罩，如同一尊神魔，身高上触苍穹，下踏白云，抬手可捞日月星辰，气势恐怖到极点。
<br/>　　他所有道则复苏，在轰鸣声中，他的手中凝聚出一柄神斧！
<br/>　　以他如今对大道的领悟，吸收盘古开天印记及传承，再加上他自己观看了盘古看天的过程，融入种种大道，悟出了这威力惊人的开天印！
<br/>　　这是他所用大道的合力一击！
<br/>　　咔擦——
<br/>　　他一斧劈出，开天辟地，黑暗风暴及黑暗魔神所在之地，鸿蒙切开。紫气分裂，化作玄黄二气。玄者升为天，演化出日月星辰，星河璀璨，化作长河围绕在黑暗魔神身边，奔腾不息。
<br/>　　黄者为地。大地浮现，万千山峦一一拔地而起，沟壑纵横，大江大河奔腾，涌入大海，滋养万物，出现在这天地之间！
<br/>　　“怎么可能？！”黑暗魔神立刻就感觉到不妙，这开天辟地不是盘古的招式吗，怎么现在会出现在一个小辈手中？来不及多想，他心念一动，天地黑暗潮涌，化作一道巨大天幕，乱天动地，包围在身边，化作一副铠甲。
<br/>　　在黑暗涌动之中，一副黑暗铠甲穿在了黑暗魔神身上，他化作魔神真身，高大雄伟，身高数万丈。
<br/>　　他浑身的甲胄都是黑色的，闪烁着摄人的光芒，全身除了恐怖的黑瞳与巨大的秃顶之外，都被笼罩在内。
<br/>　　哗啦！
<br/>　　一道惊天的黑暗瀑布瞬间出现，之后化作辽阔天幕，遮蔽了整个空间，使光明不得靠近。
<br/>　　这是黑暗魔神的战兵——黑暗天幕！
<br/>　　黑暗魔神旋转天幕，黑暗侵袭，暗潮涌动，惊涛拍岸，暴风涌起，黑洞狂暴，伴随着鬼哭狼嚎，腥风血雨，天地化为一片黑暗地狱。
<br/>　　九天之上，日月星辰都在摇动。初生的太阳都彻底暗淡了下来，远不能与这黑暗幽冥相比。
<br/>　　“灭世印！”
<br/>　　元始无悲无喜，又是一印轰出！
<br/>　　刚刚开辟形成的天地陡然间剧烈震动，化为一个巨大磨盘，将天地一切笼罩。
<br/>　　磨盘搅动，将天地吞没，一切具皆绞碎，巨大的灭世磨盘流露出恐怖的混沌鸿蒙之气，玄黄、阴阳、天地三才、四象、五行、六合、八荒等等刚刚新生的大道悉数破灭！
<br/>　　顷刻之间，刚刚诞生的天地便被这天地磨盘吞噬，复归混沌鸿蒙！
<br/>　　轰！
<br/>　　混沌魔神吐血倒飞，魔神真身之上出现无数道裂痕，鲜血汩汩而流。
<br/>　　这是一种惊人的景象，他身上的黑暗铠甲何其坚固，此刻却具皆被这灭世磨盘磨灭。
<br/>　　黑暗魔神是混沌魔神，其修为之强大无需置疑，但此刻，却如此被动，几乎被灭杀。
<br/>　　“你是什么人？”他真的被惊住了，不是刚开天不久吗？在当世怎还会有如此强大的人物？还有他那开天印怎会如此像盘古的开天式？难道是盘古的传人？
<br/>　　“诛神印！”
<br/>　　元始不言不语，周身道则萦绕，阴阳印、五行印、四象印、不周印、元心印，雷电法则、因果法则、寂灭法则、封印法则等众多大道法则，围绕在元始周身，如众星拱月一般。
<br/>　　元始衣衫飘摇，仙光喷涌，周身大道化作一柄巨大的紫金色长枪，对准黑暗魔神，以一种无法阻挠的气势，轰碎一切，轰然穿透黑暗，种种道光围绕着这柄长枪飞舞，如同伐天之矛！
<br/>　　霸气凌天，轰碎苍穹！
