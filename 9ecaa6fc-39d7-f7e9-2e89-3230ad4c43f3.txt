　　本来老郭一个外门弟子是没有资格看这些卷宗的，不过老郭身份特殊，再加上这是为了掌教叶少阳的事，苏钦章当然全力配合。
<br/>　　“正好，郭师兄，我有件事还想找你商量，你可知道，掌教失踪的事，在法术界已经传开了？”
<br/>　　“有这种事！”老郭很是吃惊。
<br/>　　“对，我派人打听了，大概是因为几大宗门，在空界都有关系，而且空界之战，消息不可能瞒得住，有些人打听到了事情经过，据说还想找掌教兴师问罪，不过掌教走了，倒是好了，生了许多麻烦。”
<br/>　　老郭道：“这话怎么说？”
<br/>　　“他们说掌教包庇转世鬼童，与阴司作对，要掌教给个交代，现在掌教失踪了，他们也没法子，这件事茅山没有参与，总不能找到我们头上来。”
<br/>　　“哦，这个且不管他们怎么折腾。”
<br/>　　苏钦章道：“我要说的不是这个，最近龙虎山张掌教下来法帖，邀请我去龙虎山议事，说要重建法术公会，推选出一个首领出来，调遣整个法术界，支援空界联盟。”
<br/>　　“什么鬼！”老郭脑海中蹦出“武林盟主”四个字，随即又奔放地联想到了岳不群，急忙打住，听苏钦章往下说。
<br/>　　苏钦章道：“我打电话给张掌教问了，听他说，他们龙虎山有个卷宗，写着‘鬼童出，天下乱’，一旦转世鬼童现身，说明整个法术界进入了末法时代，天劫降临，必须一起对抗，杀了转世鬼童，因此，法术界必须团结起来，一起应对，只是法术界现今一盘散沙，缺乏这样有号召力的人物，所以几大门派主张一起选出这么一个人，大家在应对天劫这件事上，完全服从他的管理……郭师兄，我这么说你可明白？”
<br/>　　老郭想了一会，说道：“末法时代不就是一个说法么，怎么又跟天劫捆绑在一起了？”
<br/>　　苏钦章道：“道渊真人占乩得知，两件事本为一件，正好转世鬼童现身，一切都对得上，现在法术界都支持这个观点。”
<br/>　　“哦，这么说，推举的人一定是道渊了？他年纪最大，辈分最长，实力也差不多是最强的了。”
<br/>　　苏钦章道：“并非如此，听张掌教的意思，是要选一个二代弟子，这样能够带领法术界走的更远，毕竟这次天劫，是应在二代弟子的身上。悬空观一战，和这次星宿海一战，掌教得罪了很多人，张掌教也明说，这次选举，本来是不想拉上茅山参加的，结果因为掌教失踪了，他们觉得茅山应该脱离掌教的管辖，重新跟法术界融合……”
<br/>　　老郭冷笑，突然想到什么，问苏钦章：“你个人怎么看？”
<br/>　　“我是坚决支持掌教的。”苏钦章不容置疑，“只是现在掌教不在，有些事我一个人不敢决断，郭师兄你见多识广，只能找你商量了。”
<br/>　　老郭说道：“苏老板，我问你一个问题，你对少阳包庇转世鬼童这件事，怎么看？”
<br/>　　电话那头沉默了一下，说道：“我不知道什么鬼童，我只知道，于公，那是我掌教夫人，于私，那是我嫂子。”
<br/>　　老郭对苏钦章这回答十分满意，接着问道：“那我问你，如果少阳为了你这嫂子，得罪天下，甚至与整个法术界为敌，你……何去何从？”
<br/>　　苏钦章又沉默了一会，道：“自古臣不言君过，掌教就是掌教，我服从指挥就行，有死而已！”
<br/>　　“哈哈哈哈，痛快！”老郭大笑起来，“苏老板你看着文弱弱的，倒也有骨气，不过，师父没选错人，我茅山一脉，都是这么不讲理！爱咋咋地！”
<br/>　　苏钦章没有老郭这么豪情，叹道：“可惜掌教不在，不然这次推选，法术界二代弟子中，没一个有资格跟他争的。”
<br/>　　老郭道：“我觉得张无生这厮突然提出这种事，里面必然大有文章，我们见面说吧。我明天就去找你。”
<br/>　　挂上电话，老郭陷入深思之中，扁头这时候就在他身边，也听到了他们的对话，说道：“他们为什么要趁小师叔不在，选什么法术界的首领。”
<br/>　　老郭冷笑说道：“还不是想架空他道门第一弟子的身份，谁都知道，二代弟子之中，没人有小师弟的实力，杀凌宇轩，悬空观一战，星宿海一战，小师弟出尽风头，但也把人得罪光了。现在他为了冷玉妹子……”
<br/>　　说到这，老郭叹了口气，“真没想到，冷玉真的是转世鬼童……现在想来，师父真的是有先见之明，没有让少阳负责山门的事情，否则这次真的会连累山门。”
<br/>　　扁头道：“大叔你真的怕连累山门吗？”
<br/>　　“我怕！”老郭严肃的说道，“茅山千年传承，绝不可以毁在我们这一代人手上，好在少阳虽然看似叛逆，道理还是懂的，所以故意不怎么回山，一切战斗也不想让山门插手。你记住，山门不能没落，这不光是我，就算是少阳和道风也是这么想的。”
<br/>　　说到这，深吸一口气，道：“但是为了少阳，我也不惜得罪天下，粉身碎骨！”
<br/>　　扁头点了点头。
<br/>　　零界，与紧挨着人间的空界不同，这里是虚无中的一片陆地，有山，也有水，但是没有阳光雨露，只有漂浮在四周的黑云，和无尽的海水，如果有人能从远处看过去，这里就像是一片云彩包围之中的岛屿。
<br/>　　在这个巨大的岛屿上，僵尸并不是唯一的生灵，在丛林里、海水中，还有着数不清的邪恶生物，但多数是一些低等的，没有智慧的，因此尸族作为入侵者，很快就统治了整个零界，成为他们的大本营，但是大大小小的斗争，从来都没有结束过，在这片被放逐的土地上不断上演。
<br/>　　鲜血，杀戮，冰冷。
<br/>　　在这片土地的最中间，有一座山，高耸入云。黑云。是这块大陆上最高的一座山峰，尸族管他叫天弃山，名字的由来，跟空界的神弃平原一样，表明这里是一座被上天放弃的地方。
