<br/>				冷华一边听着陌尘的话，一边在一旁帮吞云身上的伤上药，他微顿了一下，问：“既然不是上游大陆的人，那又会是什么人？难道会是海外地域？可是，主子并未去过那些地方，又如何与人结怨？让那些人想致她于死地？”<br/>　　<br/>　　关习凛沉思着，目光不由的落在一旁摇篮中的小灏儿身上，他略带迟疑的道：“小九曾说小灏儿身世神秘来历不简单，会不会，那些人是冲着小灏儿来的？”<br/>　　<br/>　　这话一出，众人不由沉默着。也是，按理说，凤九到了这上游大陆后就一直在这边建立着势力，并没与其他地域的人结下什么仇怨，又怎么会让人出动十几名仙圣强者，以及一名仙帝来取她性命？莫不是，真的是她收养的那个孩子引来的祸事？<br/>　　<br/>　　陌尘没有说话，只是将目光落在那摇篮中的孩子身上。是不是如他们猜测一般，眼下还不知道，但，他觉得他们就算是不知道，凤九也应该是会知道的。<br/>　　<br/>　　在房中，冷霜端了清水进去后，在轩辕墨泽的示意下退出了房间。<br/>　　<br/>　　“我帮你先将衣服脱下来。”轩辕墨泽说着，看着她红衣上暗红的颜色，眉头不由的微拧起来，而双手却是小心翼翼的帮她将身上衣裳脱下，尽量的不扯到她身上的伤口。<br/>　　<br/>　　凤九应了一声，配合着将衣裳脱下，只剩下最里面的一件肚兜遮着身上的春光，只是，饶是此时眼前春光无限，轩辕墨泽也无心思欣赏，他的目光落在她身上那一道道的伤口上面，看着有的伤口皮肉外翻，似乎伤得很深时，深邃的黑瞳中划过一抹森寒而冷冽的寒意。<br/>　　<br/>　　他珍之惜着的人儿，竟在他的眼皮底下被人伤成这样，那些人，就是死一百次也不足惜！<br/>　　<br/>　　“你体内的青莲不是可以疗伤吗？先试着用青莲之力修炼身上的伤口。”他压下心中的杀意，缓着声音对她说着。<br/>　　<br/>　　凤九正想着事情，听到他的话后应了一声：“好。”<br/>　　<br/>　　她闭上了眼睛，双手微抬置于身前丹田处，调动体内的青莲气息，以青莲的复苏之力来修复身上的伤口。<br/>　　<br/>　　然，随着时间过去，她额头上渗出汗水，脸色越发的苍白，而身上的伤口却依旧如初，并没有半点修复的迹象。<br/>　　<br/>　　一旁看着的轩辕墨泽心头一沉，隐隐察觉出什么来。<br/>　　<br/>　　这时，凤九睁开眼睛，散去一身的青莲之气，缓了一口气后，才抬头看向他，清幽的眸子对上他深邃的黑瞳：“修复不了，青莲之息与我身上那每一处伤口上的气息仿佛磁石相斥，互相排斥，无法修复。”<br/>　　<br/>　　说着这话，她心里也缓了过来，原本的猜测在这时得到肯定。<br/>　　<br/>　　“那些人是冲着你体内的混沌青莲来的。”<br/>　　<br/>　　轩辕墨泽沉着声音说着，一边取出药来为她清理伤口后，再洒上她特制的刀伤药，又取出布条，将每一洒上药的伤口仔细的包扎好。<br/>　　<br/>　　凤九眸光微沉，清幽的眸子似乎在想着什么一般……<br/>
