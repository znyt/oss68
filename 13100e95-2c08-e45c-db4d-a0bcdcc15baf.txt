<br/>				李瑞将鱼拉回去后，积极更新菜单，并且定制价格，在几个自己所控制的餐厅开始宣传。<br/>　　<br/>　　由于他所控制的几个餐厅，已经因为蔬菜的原因，将声誉打开，现在可以说非常的火爆。每天晚上用餐的人，可以说是已经预定到了两个星期后。<br/>　　<br/>　　没有想到的是，今天晚上来吃饭的人，拿到餐厅特制菜单后，才发现精品菜单最前面几页有着非常大的图片，每一页都是一个鱼菜。<br/>　　<br/>　　有好几种非常著名的鱼菜，比如清蒸鲈鱼、炭烧鳗鱼，还有鲜嫩的虹鳟鱼刺身，松鼠桂鱼、红烧鲤鱼等等。<br/>　　<br/>　　菜式多图片漂亮，看着就有食欲，而且旁边还大大的将价格标示的非常清楚，一盘清蒸鲈鱼，888元。炭烧鳗鱼，998元！等等，最最便宜的酸菜鱼，竟然也要688元。<br/>　　<br/>　　好么，图片漂亮！这价格也更加的漂亮。<br/>　　<br/>　　对于吃饭的顾客看到价格后，倒是没有觉得惊讶，因为上面价格标示的非常清楚，觉得贵可以不点。<br/>　　<br/>　　来这里吃饭，没有几个人会因为价格贵而走人的，都是喜欢满足口腹之欲的人，怎么可能会因为这点价格就闪人？<br/>　　<br/>　　而且经历了这里的蔬菜的味道冲击后，对于其他精品菜色也就有了信心，直接按照自己喜欢的点菜。不过也有些新顾客是没有吃过这里菜的，看到这么贵的菜单，心中也充满了疑问。<br/>　　<br/>　　王海，一名大公司的销售经理，今天他特意在李瑞负责的李氏餐厅，宴请客户马总。<br/>　　<br/>　　他知道只有这里才能吃到上次好吃的蔬菜，虽然贵，但是有贵的原因。他不会忘记上次来吃的拿到韭菜炒鸡蛋，简简单单的一个菜式，却让他吃出了小时候的味道。还有，就是晚上的冲动，真的让他非常的怀念。<br/>　　<br/>　　至此后，他只要有时间有条件，就会来这里吃饭，品尝别的地方没有的味道。<br/>　　<br/>　　虽然贵，但是按照他的工资还是吃的起的。不过，作为一名肉食动物，在这里竟然没有精品肉类菜式。<br/>　　<br/>　　有的也是利用精品蔬菜制作的普通肉类菜式，虽然口味有所提升，但是却没有精品蔬菜的味道好吃，总得来说是有些遗憾的。<br/>　　<br/>　　餐厅对此倒是做的比较好，精品就是精品，如果是精品配菜的话，那么就会标注清楚，价格也比以前贵不了多少。<br/>　　<br/>　　没有想到，今天晚上宴请马总，竟然看到了更新的精品菜单，而且是他喜欢的鱼肉，顿时开心不已。<br/>　　<br/>　　他点了两个自己喜欢的鱼肉后，就将菜单递给了马总，当然菜单是精品菜单，不是普通菜单。<br/>　　<br/>　　马总并没有在李氏餐厅吃过饭，等他看到菜单上的价格，顿时就眉头皱到了一起。<br/>　　<br/>　　“老王啊，这里的菜单没有问题？”<br/>　　<br/>　　王海一看他就知道是价格惹的祸，于是笑着说道：“马总，这个菜单没有问题！在这里吃饭，要是不点精品蔬菜可就浪费了一次机会！”<br/>　　<br/>　　“哦？浪费什么机会，为什么这个菜单上的价格这么贵？”<br/>　　<br/>　　“哈哈，马总！你是不常来武市，所以不知道。这个李氏餐厅可是非常火的，预定都要排到两个星期以后了。我也是两个星期前预定的。”<br/>　　<br/>　　“什么？这个餐厅这么火？是什么原因？”<br/>　　<br/>　　“马总，你先点餐！等菜上来后你尝过了，就知道这个餐厅为什么火了！”<br/>　　<br/>　　“哦？那好，我就尝尝这里的菜，看看究竟！”于是也就点了一个鱼菜和几盘蔬菜。<br/>　　<br/>　　等马总将菜单放在一边的时候，才发现有两个菜单。不仅制作的不同，而且从档次上来看，也有着非常大的区别。<br/>　　<br/>　　“咦？老王，这个是……”他倒是没有怀疑王海骗请自己，装大头，合作有个几年了，他相信王海没有这个必要。<br/>　　<br/>　　“哦！马总，你看看！”说着将普通菜单递给马总。<br/>　　<br/>　　马总一看，有些惊讶的问道：“老王，这里的蔬菜有些菜品是一样的，怎么价格有这么大的区别？”<br/>　　<br/>　　“马总，这就是精品菜单和普通菜单的区别，用料不同，虽然是一样的菜，但是东西可就区别大了，要不然也不会预定到两个星期后去了！”<br/>　　<br/>　　两个人说话间，这菜也就上来了。<br/>　　<br/>　　“来来来！马总，尝尝这个鲈鱼！这个是最新出的菜，我以前可是也没有吃过呢！”<br/>　　<br/>　　“好，一起尝尝！”<br/>　　<br/>　　两个人也就同时夹了一块鱼肉，吃进嘴里。顿时，也让两个人惊讶了，这鱼肉进入口中后，就极大的刺激味蕾。<br/>　　<br/>　　鲜、嫩、滑、香！既有鱼肉的鲜美，又有鱼肉的嫩滑，可以说没有一点的腥味。这么好吃的鱼，也是王海和马总所没有吃过的。<br/>　　<br/>　　两个人有尝了其他几个鱼菜，一时之间，都有些不好形容。<br/>　　<br/>　　“真是好吃啊！”虽然有千言万语，但是却无法用词来形容，只能用好吃两个字概括了。<br/>　　<br/>　　虽然两个人不是美食家，但是却是标准吃货，对于如此好吃的菜品，当然是满意之极。<br/>　　<br/>　　王海也是暗自高兴不已，他还怕鱼肉没有心中所想的那么好吃。没有想到这一吃，真的是惊讶之极，真的是好吃啊。<br/>　　<br/>　　共同的爱好，也让两人之间的关系更加的融洽。做销售就是做朋友，只有关系到位了，才能出业绩。<br/>　　<br/>　　王海用心的招待马总，也让他看到了王海的用心，所以这一顿美食，也让马总满意之极。<br/>　　<br/>　　业务么，大家都是相同的条件，那么为什么不把合同给用心的人呢？只有用心的人，才能在以后用心的完成合同。<br/>　　<br/>　　“唉！这一吃，筷子都停不下来啊！”<br/>　　<br/>　　等菜吃的差不多的时候，马总才突然发现，自己好像吃撑了。没有想到今天晚上吃了平时的好几倍的饭菜，真的是过量了。<br/>　　<br/>　　“哈！我也是，这每次来这里吃饭啊，总是吃撑！”王海也是笑着说道。<br/>　　<br/>　　“真没有想到，这一次来武市竟然品尝到如此美食，真是幸运！”<br/>　　<br/>　　“马总，你要是喜欢，可以常来！”<br/>　　<br/>　　“哈哈！好！一定会的！”马总也是高兴的说道。<br/>　　<br/>　　“那可是说好了！等你来了，我还请你来这里吃饭！”<br/>　　<br/>　　两个人都是大笑着不止。王海也知道自己的美食攻略没有白费，合同已经差不多就是自己的了。<br/>　　<br/>　　同时也感叹不已，这李氏餐厅有了这些菜品，真的可以用日进斗金来形容了。<br/>　　<br/>　　今天晚上凡是来餐厅吃饭的人，都是高兴而来，满意而归！可以说这次鱼类的上市，取得了满意的效果。<br/>　　<br/>　　港区酒店餐厅，也是同样如此。而且由于是星级酒店，所以鱼类菜品的价格更加高，有的因为厨师不同，价格都定制到了千元以上，甚至是两千元以上。<br/>　　<br/>　　就是这样的价格，却让所有的人趋之若鹜，良好的口碑和超好的菜品，让他们赚的盆满钵满的。<br/>　　<br/>　　这也让高恒更加的重视陈默哪里的渠道，甚至通知凌经理，已经要保持好与陈默的关系，想尽一切办法维护好这样的优质进货渠道。<br/>　　<br/>　　嗯当然，从陈默哪里的鸡子，也要多多益善。这些时日，从陈默哪里拿过来的鸡子，都没有机会到达酒店的餐桌上，而是被高恒这帮港区高层给瓜分了，这也是数量太少的过，每次拉货就是几十只而已。<br/>　　<br/>　　实在是鸡子太好吃了，无论是怎么做，都让人难以忘记那个味道。<br/>　　<br/>　　现在，就要加上鱼肉了！尤其是今天晚上高恒和一大帮子港区高层，一起来了个全鱼宴！最后，所有的人都吃撑了。<br/>　　<br/>　　对于陈默这里的东西，好是好，但是却总是吃多，控制不体重啊！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
