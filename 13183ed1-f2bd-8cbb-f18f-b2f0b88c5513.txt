　　种族技能……那是什么东西？
<br/>　　被系统弄的有些发懵，刘哔顺从的点开技能面板，只见和先前不同的是，技能面板和之前相比有了细微的差别。
<br/>　　先前的时候，面板内只有‘不死’一个被动技能，而此时的被动技能栏内，又多出了一个名为‘吞噬’的技能，这让刘哔心中一喜，连忙查看起了关于该技能的介绍。
<br/>　　名称：吞噬
<br/>　　类别：种族技能（被动）
<br/>　　功能：宿主所使用的任何食材，都有一定几率使宿主获取食材的一种或多种特性，获取率受食材新鲜度，完好度影响。
<br/>　　虽然介绍看起来没头没脑的，但是向系统询问之后，他也算是搞明白了这项被动技能的具体功效。
<br/>　　通俗一点说，就是他可以通过吃东西，来获得食物的属性，越新鲜的，越完整的成功率就越大，当然，烹饪也算破坏食材的一种。
<br/>　　举个栗子，他如果有机会，吃到一只皮卡丘，那没准也会觉醒十万伏特，当然，如果吃的是烤皮卡丘，那成功率就会比吃生皮卡丘片要低一些了。
<br/>　　理论上来说，这个技能的强大是毋庸置疑的，但眼下的情况，却起不到什么用处。
<br/>　　不要说皮卡丘了，这个世界有异能的，全都是一些僵尸鬼怪之类的，除此之外，也只有一些能人异士，这让刘哔心中不禁开始犯难了。
<br/>　　鬼魂没有实体，无从下口，僵尸满身腐肉，更是下不去口，吃人的话，他心中又有些不能接受，这样看来，唯一能吃的也就只有一些修炼出法力的小妖了。
<br/>　　但是这个世界有没有妖还不一定，在某些事情还未证实的情况下，他也不敢妄下结论，况且，就算真的有妖，以他现在的实力，谁吃谁还真不一定。
<br/>　　苦笑一声，刘哔从系统界面退了出来，虽然暂时用不上，但他对于这项技能还是非常满意的，至少以后回到他原来世界的时候，应该会给他带来很大的帮助。
<br/>　　末世里，最不缺的就是变异兽，这个技能的发挥空间还是非常大的。
<br/>　　“对了老板，向你打听个事呗。”回过了神，刘哔想起自己原本的计划，看了正在忙碌的老板一眼，继续说道：“你知道任老爷任发住在哪里吗？”
<br/>　　这并不是什么大事，老板也没有隐瞒的心思，给刘哔报了一个地址之后，还十分热心的告诉他应该怎么去走。
<br/>　　道了一声谢，从桌子上拿了一根牙签，刘哔剔着牙离开了摊位，向老板指明的方向走去。
<br/>　　此时大概是早上七点多钟了，街上的行人也逐渐多了起来，道路两旁的店面，也都基本开张，不时传来的商贩吆喝声，让这个早晨变的有些嘈杂。
<br/>　　对于这种吵闹，刘哔微微有些抵触，习惯了末世那种清冷荒寂的环境，他却对于眼前的这幅光景有一种恍惚的陌生感。
<br/>　　还记得大概十年前，他也是喜欢热闹的人，平日无聊的时候，也总是拉着几个要好的朋友一起去大街上溜达。
<br/>　　但是自从末世爆发，身边的家人朋友一个个死去，这让他的性格变的越来越孤僻，如果不是弟弟一直在陪伴着他，估计他此时都不知道该如何和别人交流。
<br/>　　孤单，并不能说它是一件坏事，但是孤单的时间久了，这事就会变得有些糟糕了。
<br/>　　约莫半个钟的功夫，刘哔来到了任府，他此时才发现，这个任家的势力，至少是财力要比他想象中的大很多。
<br/>　　虽然对任家镇不很熟悉，但通过他仔细的观察，这里几乎是镇子上最为繁华的街道了，而任府，就坐落在这条街道的中央位置。
<br/>　　没有具体查看过，刘哔并不知道任府内部具体有多大，但粗略看去，占地面积也是十分可观，能在闹市建起这么一栋大宅，任家的恐怖可想而知。
<br/>　　此时，依靠在任府门前的石狮子上，刘哔静静盯着任家朱红色的大门，上方两个凶神恶煞的铺首，将他的注意力全部吸引。
<br/>　　铺首并不稀奇，这种东西在一些老宅中也十分常见，常被人称之为门环，可寻常人家，通常用狮、虎、龟、蛇等猛兽用以装饰，有一些特殊喜好的，也只会用一些毒虫的头像。
<br/>　　像任家这种门环，却不太多见，恰好，在末世发生之前，刘哔回老家探亲的时候，就见过这种门环。
<br/>　　因为这种门环的长相凶恶，给他留下了很大的印象，事后他查询过一些相关资料，得知了这种门环叫做螭头环。
<br/>　　而那面目狰狞的异兽，就是龙生九子中的‘螭’，这种异兽在山海经上曾有记载，形似螺蚌，性好闭，最反感别人进入它的巢穴。
<br/>　　而使用什么样的门环，都是有所讲究的，作为九子之一，用‘螭’来做门环也能将一些凶神恶鬼挡在门外，起到镇宅驱煞的作用。
<br/>　　但是，凡事都有好坏，螭头环可不是寻常人家可以装配起的，这种门环的规格相当高，在古代的时候也只有一方要员才可以使用。
<br/>　　因为身为要员，往往身上都会有浓厚的气运，自然能将其镇压住，如果身份地位不相匹的人贸然使用，身上的气运不足以将其镇压，就会引发一些不好的事情。
<br/>　　当时接触到这些资料的时候，刘哔心中还有些不以为然，因为这只是古人流传下来的一个说法，并没有确切的科学依据。
<br/>　　但经历过末世爆发，系统出现，穿越到电影世界等一系列的事情，对于这样的传闻，他不免认真严肃了起来。
<br/>　　有很多事情，是根本无法用科学来进行解释的，老祖宗留下来的东西，也自然有它存在的道理。
<br/>　　恐怕，这个任家还真的没有那么简单，之所以出事，也不是单单只是因为坟墓的原因。
<br/>　　沉吟片刻，刘哔走上前去，叩了叩任府的大门，却没有得到一丝回应。
<br/>　　重复几次之后依旧没有人来开门，刘哔不禁皱起眉头，按照常理来说，不应该这么久都没有回应才对。
<br/>　　看着眼前朱红色的大门，他的心中顿时升起一种不好的预感，怕是任家真的出了什么事情。
