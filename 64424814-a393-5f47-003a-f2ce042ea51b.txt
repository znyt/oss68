　　萧逸的声音并不高，却清晰地传入在场武者的耳中。
<br/>　　他的语气淡淡的，却让得近三千武者同时脸色一变，不自觉地后退起来。
<br/>　　近一百，对接近三千，看起来毫无胜算。
<br/>　　可这近百人中，先天九重好几个，且都是北山榜前二十之人；先天八重十数，先天七重数十，最弱的都是先天六重。
<br/>　　反观人数占优的近三千武者，唯一的两个先天九重赵不群和顾长风已然重伤，毫无战力。
<br/>　　唯一能上台面的，只有那十个大型团伙的‘首领’。
<br/>　　其余的，先天一重、二重占了大部分，甚至最弱的，只有后天九重。
<br/>　　这近三千武者，就算用脚趾头都能想到，最后败的必然会是他们。
<br/>　　当然，他们仗着人数优势，最后就算败，萧逸等人也会损伤惨重。
<br/>　　总的来说，这会是个两败俱伤的结果。
<br/>　　“诸位。”萧逸再次站前一步。
<br/>　　近三千武者再次齐刷刷地后退一步。
<br/>　　萧逸笑笑，道，“若不想战，便自动散开，萧某当刚才之事没有发生过。”
<br/>　　事实上，两败俱伤的结果，是不算萧逸在内。
<br/>　　若算上萧逸，那只会是一面倒的屠杀场面。
<br/>　　凭着洞玄四重的肉体力量，单他一人，就可轻松杀尽这数千人。
<br/>　　洞玄境与先天境的差距极大，远不是人数能够弥补的。
<br/>　　这个世界的武者，飞天遁地、翻江倒海，无所不能。
<br/>　　那些境界极高的武道强者，每一个都拥有毁天灭地般的修为。
<br/>　　伏尸百万，轻而易举。
<br/>　　在真正的实力面前，人数只是个笑话。
<br/>　　虽说如今的萧逸并没有那等高深的境界，但对付面前的武者，足够了。
<br/>　　言归正传。
<br/>　　萧逸的话音刚刚落下，不出十秒，一众武者便全部散去了，恢复到之前的混战。
<br/>　　既然明知抢不了萧逸的宝物，他们只能回到之前争夺裂天剑派弟子名额的战斗上。
<br/>　　前一秒还是联手准备抢萧逸的东西，下一秒便互相战斗起来。
<br/>　　世界就是这样，人性也便是如此，没有永远的敌人，也没有永远的朋友，只有永远的利益。
<br/>　　花剑等人，也重新恢复到之前各抢各分数的战斗上。
<br/>　　原地，只剩下萧逸等五人。
<br/>　　“你们的分数抢够了吗？”萧逸问道。
<br/>　　林劲等人点点头，道，“我抢了其中一个‘首领’的分数，足够通过考核了。”
<br/>　　铁牛也道，“俺也够了，再抢下去也没多少意义。”
<br/>　　秦飞扬和柳嫣然也点点头，表示够了。
<br/>　　一行五人，没有再参与之后的战斗，洒然离去。
<br/>　　为期一个月的考核，所有人都必须在最后一天回到剑派山门。
<br/>　　那里早有等待着的执事，负责记录分数，并在最后给出排名。
<br/>　　.....
<br/>　　裂天剑派，山门外，十个执事负手而立。
<br/>　　在一旁的是，萧逸五人。
<br/>　　他们，算是最早回来记录分数的人了。
<br/>　　萧逸淡淡地笑着，回想起这一个月，他的嘴角莫名的扬起了一抹满足。
<br/>　　一月前，他刚来这剑派山门，子然一身，排队报名。
<br/>　　一月后，同样的地方，他却收获了四个好友，四份真挚的友谊。
<br/>　　对于他来说，这四份友谊，这一个月的出生入死，比很多东西都重要得多。
<br/>　　上辈子，他虽是人人敬怕的杀手之外，但一生都是独来独往，没几个朋友。
<br/>　　这辈子，穿越而来，遇到的却都是勾心斗角和阴谋诡计。
<br/>　　唯一让他欣慰和看重的，也只有萧家那几个疼爱他的老人。
<br/>　　如今，这几份友谊，给了他极大的满足。
<br/>　　他也格外珍惜这几份友谊。
<br/>　　不能说他矫情，只能说，这是一种心境。
<br/>　　这时，一旁的林劲等人有些无聊，闲聊起来。
<br/>　　萧逸从乾坤袋中拿出了风吟剑，对柳嫣然道，“这个送你。”
<br/>　　“嗯？”柳嫣然愣了愣。
<br/>　　萧逸轻笑道，“上次，你说我修为最低，金翼丹理所当然归我。如今，似乎是你实力最弱吧，风吟剑便给你了。”
<br/>　　上次的金翼丹，按道理，应该是给柳嫣然的。这次的风吟剑，便算是还她的人情。
<br/>　　柳嫣然美眸闪烁，但没有接，而是问道，“这可是珍贵的灵器喔，你舍得给我？”
<br/>　　“刚才，你还以为我要抢你的地阶武技，哼。”
<br/>　　柳嫣然一副嗔怒的样子。
<br/>　　萧逸苦笑一声，女人果然都是小气的生物。
<br/>　　“我没有地阶武技，如果有，我会给你。”萧逸收起了笑意，认真地说道。
<br/>　　这是真话，其他武者梦寐以求的地阶武技，在他眼中，并非什么珍贵的东西。
<br/>　　他的底牌，无论是紫炎还是修罗战体，随便一样都远比地阶武技珍贵百倍。
<br/>　　只不过，紫炎自不必说，那是冰鸾剑‘掠夺’来，并附加到控火兽上的能力，无法给予，其他人也不可能得到。
<br/>　　修罗战体，则是外人几乎不可能修炼的功法。
<br/>　　妖兽精血上的残暴气息，连当年的狂血玄君都难以压制，更别说其他人了。
<br/>　　先不说其他人不知道修罗战体的存在，不知道当年的狂血玄君因修罗战体而强大无比。
<br/>　　就算知道了，修炼了，那也是非常危险的事，不亚于将一颗定时炸弹放到自己体内。
<br/>　　也只有萧逸，拥有冰鸾剑，才莫名其妙地将残暴气息给消散掉，修炼起来毫无危险。
<br/>　　“哼。”这时，柳嫣然哼了一声，撅嘴道，“算你会说话。”
<br/>　　“这风吟剑，我就不要了。”柳嫣然眉宇间透漏着喜悦，但始终没有要风吟剑。
<br/>　　“不要？”萧逸愣了愣。
<br/>　　“嗯。”柳嫣然道，“如果可以的话，下次送我别的东西好吗？”
<br/>　　“你想要什么？”萧逸问道。
<br/>　　柳嫣然笑笑，道，“还没想好，下次如果看到喜欢的东西，再找你要。”
<br/>　　并非是她不喜欢风吟剑，而是她不想收太贵重的东西。而且，接下来的考核中，还要战斗，在她看来，萧逸更需要风吟剑。
<br/>　　萧逸笑笑，猜出了她心中所想，也没有拆穿，而是点头应允下来，“好。”
<br/>　　萧逸想了想，风吟剑毕竟是从顾长风手上抢来的，要是他日后找麻烦，反而对柳嫣然不好，还不如自己先留着。
<br/>　　.....
<br/>　　不多时，陆续有人从森林内回来，并到执事处记录分数。
<br/>　　一直到入夜，森林内的所有武者才全部回来。
<br/>　　十个执事做好所有统计后，开始公布名次。
<br/>　　十个执事中，为首的正是一个月前给萧逸令牌的‘易老’。
<br/>　　“第一名，萧逸，2535分。”
<br/>　　易老说着，有些意外地看了萧逸一眼。
<br/>　　其他武者，见识过萧逸打败顾长风和赵不群，倒没有太大的意外之色。
<br/>　　人群中，一个年轻人忽然皱起了眉头，目光紧紧地盯着萧逸。
<br/>　　“第二名，雷战，2530分。”
<br/>　　易老说着，看向了那年轻人。
<br/>　　不错，那年轻人正是雷战。
<br/>　　萧逸也看向了雷战，今日的混战中，雷战并没有出现，也不知是何原因。
<br/>　　不过，雷战一直都是独来独往，单凭一人，竟拿到了2530分，着实恐怖。
<br/>　　要知道，萧逸今日连抢了顾长风和赵不群两人的分数，加上自己这一个月来打败其他小型队伍得到的分数，也不过才2535分。
<br/>　　仅仅是比雷战高出5分，以微弱之差，得了第一名。
<br/>　　“第三名，林劲，830分。”
<br/>　　参加考核的武者，原本有一万多名，也就是说，总分是一万多分。
<br/>　　那些大型团伙仗着人数优势抢得最多，而且全都上交给首领。
<br/>　　导致，顾长风和赵不群两人原本有一千一百分左右。而其他的‘首领’，则每个都有500分左右。
<br/>　　萧逸五人，原本每人只有300分左右，在抢了他们后，分数自然大大飙升。
<br/>　　“第四名，铁牛，820分。”
<br/>　　“第五名，柳嫣然，800分。”
<br/>　　“第六名，秦飞扬，650分。”
<br/>　　“第七名，何健，550分。”
<br/>　　“.....”
<br/>　　易老快速地宣布着名次。
<br/>　　这时，萧逸和林劲满脸疑惑之色地看向秦飞扬。
<br/>　　“怎么回事，你的分数不应该这么低。”萧逸问道。
<br/>　　“我记得，你的分数和我一样是830的。”林劲也问道。
<br/>　　一旁的铁牛憨厚地指了指秦飞扬，道，“今日战斗时，我看到他和好几个女子眉来眼去。”
<br/>　　萧逸脸色怪异，道，“你这家伙，该不会是将分数送人了吧。”
<br/>　　秦飞扬撇撇嘴，道，“美女有所求，我自然不能拒绝。”
<br/>　　“这个色胚。”萧逸无奈地摇摇头。
