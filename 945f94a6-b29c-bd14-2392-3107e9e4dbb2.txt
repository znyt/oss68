        &lt;h3&gt;第三卷第二百七十九章  江俞琪的担心不攻自破&lt;/h3&gt;
<br/>
<br/>        江俞琪在新闻里看到了江俞轩辞职的新闻发布会，看到新闻的时候她手中的杯子一不小心掉到了地上。
<br/>
<br/>        江俞琪怎么也想不到江俞轩退出了陈子昂的公司。她不知道弟弟的心里是怎么想的，这么多年了江俞轩从来没有说是要从陈子昂的公司离开，是什么原因让他离开的呢？
<br/>
<br/>        是两个人之间出现了矛盾吗？江俞琪看着新闻里的弟弟，今年二十八九的人了，和张倩楠的婚姻既没有公开也没有在一起。但是两个人的关系让人觉得很奇怪，别人看不到的地方她这个做姐姐的是能看得到的。
<br/>
<br/>        江俞琪这几年已经是商界的大佬了，在房地产业里面也是数一数二的人物。她随着爸爸一直在房地产界南征北战，西城的建设其实也有她一大部分的功劳，前期她已经将工作做好，然后才去的别的地方。
<br/>
<br/>        弟弟江俞轩在陈子昂公司已经好多年，对房产界也不是非常的熟悉，她带了一段时间之后，便放手让江俞轩自己干。这几年的江俞轩，已经是房产界的新秀，可以说是最璀璨的一颗星。
<br/>
<br/>        江俞轩这两年乐此不惫的在陈子昂的公司跑来跑去。有多少股份暂且不说，能赚多少钱也不说因为他喜欢。她和爸爸都看在眼里，从来没有责怪过他。反而觉得弟弟是重情重义之人，人生活在这个世界上，是需要有朋友相互帮助相互扶持的。
<br/>
<br/>        曾经他们都以为，江俞轩会和陈子昂在一起，但是事情让他们感觉出乎意料，中间出现了一个张倩楠，弟弟有自己的选择，他们也不可能多说什么。
<br/>
<br/>        只要他觉得幸福就好，他做什么决定他们都会支持他，不过江俞琪还是很好奇，为什么江俞轩会从陈子昂的公司离职？
<br/>
<br/>        江俞轩从陈子昂公司回来，江俞琪好整以暇的看着他。
<br/>
<br/>        “说？怎么回事？”
<br/>
<br/>        “我就是觉得精力不济，不能兼顾，再说了高新的项目方案已经落地，要实施了，我得专注这个项目，后期要去别的地方发展。”江俞轩并没有感觉到难过，无论陈子昂误会不误会，他已经决定了，陪着她一起走，余生漫长，他们都还有大把的时光。
<br/>
<br/>        暂时的磕磕碰碰又能算得了什么？他们都已经同甘共苦了这么久，只要一个眼神彼此都知道在想什么。
<br/>
<br/>        陈子昂就是没戴过笼头的驴---嘴硬，心里想着一套行动上是一套，然后嘴里什么都不说，他一定要敲开陈子昂的嘴，让她亲口对他说“我爱你”。
<br/>
<br/>        “你打算后期想去哪里发展？”江俞琪问弟弟。
<br/>
<br/>        “新年的时候我给爸爸说过，他没有给你说吗？”江俞轩反问江俞琪，爸爸不可能不给江俞琪说的。
<br/>
<br/>        “我要听你说。”江俞琪盯着江俞轩，恨不得在江俞轩的脸上盯出个所以然来。
<br/>
<br/>        “爸爸都给你说了，你还要问。”
<br/>
<br/>        “但是爸爸说你和张倩楠也过去，我不相信你会跟着张倩楠过去，是因为陈子昂吧？”
<br/>
<br/>        “他们都过去啊，我当然也想过去，反正哪里有生意就往哪里去。”江俞轩现在的感情可以说是一塌糊涂，张倩楠没有解决，陈子昂误会他了，陈天天背了个锅。
<br/>
<br/>        “不是，你去甸城为什么就要从陈子昂的公司离开呢？”江俞琪不明白。
<br/>
<br/>        “我专心我们江家事业你不高兴吗？”江俞轩并没有给姐姐解释那么多，有些事情只可意会不可言传，他怎么给姐姐说那么多的事情？
<br/>
<br/>        “当真？”江俞琪紧追不舍。
<br/>
<br/>        “当然是真的，姐，我过两天带杨总去体检，我估计杨总不太好，带他去确认一下。”江俞轩转变了话题。
<br/>
<br/>        “好，很严重？”
<br/>
<br/>        “我找了西医的教授看了，人家建议在仔细全面的作一次深层次的体检。”江俞轩语气沉重。
<br/>
<br/>        “唉，如果真有问题，他怕是要带病工作了。”江俞琪叹了口气。
<br/>
<br/>        “是啊，高新的方案落地了，项目要启动了，他肯定是要盯着的。”江俞轩成功的将话题带过了。
<br/>
<br/>        “话说，高新的项目你有几成的把握？”
<br/>
<br/>        “高新的项目很瞩目，但是还有一个项目比这个更瞩目，有了那个项目托底，我们的这个项目就是小ks了。”江俞轩胸有成竹的一笑。
<br/>
<br/>        “你是说西岭的别墅群？”
<br/>
<br/>        “不愧是我的姐姐，就是聪明。”
<br/>
<br/>        “为什么我们就不能参加？”江俞琪问，西岭的别墅群项目很大，设计豪华而高端，利润点也会很高。
<br/>
<br/>        “这个我们拿不到，已经内定了，西城现任领导和他的好搭档已经预定了好几套。”江俞轩微微一笑。
<br/>
<br/>        “你怎么知道？”
<br/>
<br/>        “我们算是他们最大的竞争对手，他们又对我实行了一把威逼利诱。不过，说实话，我的重心不在西岭，高新的建设比西岭重要得多，高新建好了是一个城市的坐标，而西岭只会是鸡肋。”江俞轩哈哈大笑。
<br/>
<br/>        他江俞轩虽然年轻，但眼光还是很准的，要不然也不会和陈子昂一起在短短的几年将一个几个人的公司做成现在的规模。
<br/>
<br/>        拭目以待谁才是笑到最后的英雄！
<br/>
<br/>        老江家的人很聪明，但是也很倔强，他们觉得不能做的事情是坚决不做的，同流合污这种事情对他们来说就是绝缘体。
<br/>
<br/>        当然，他们有骄傲的资本，也有那种自信。
<br/>
<br/>        “现在的风气不太好，我们需要处处小心，不要触雷，不给别人机会，也不给自己机会，让一切劣根和劣势远离我们，清清白白做事，明明白白做人。”江俞琪像是给自己说也像是对弟弟说。
<br/>
<br/>        “江傻妞，我知道了，别在婆婆妈妈的了。”江俞轩笑眯眯的盯着自己的姐姐，她认真起来的样子好可爱啊。
<br/>
<br/>        江俞琪看着弟弟那一如既往调皮的样子，松了一口气，看来一起都很正常。
<br/>
<br/>        “我就要婆婆妈妈的，并且是一辈子。”
<br/>
<br/>        “得了，你婆婆妈妈什么？还一辈子？”
<br/>
<br/>        “不乐意？那你别叫江俞轩，你去叫陈俞轩吧。”
<br/>
<br/>        “也不是不可以，到时候你可别说我不思进取啊。”江俞轩突然就想起了安林曦教授对陈子昂父亲说的话，微微一笑。
<br/>
<br/>        “哼，下辈子吧，这辈子除非时空逆转，你们都能回到从前。”江俞琪知道弟弟的小心思，他以为他隐藏得很好，真是笑死人了。
<br/>
<br/>        “你还别说，我就能将时空逆转！看你老弟如何扫清一切障碍，走上人生巅峰！”江俞轩的脸上写满了笃定与自信。
