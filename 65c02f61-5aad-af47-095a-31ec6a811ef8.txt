　　　　说明一下吧，最近被有些人弄的不胜其烦，这个作者号是我的小号，我上一个作者号叫不过是个小丑。
<br/>　　年初了时候在写一本最强生物之九头蛇，可惜好不容易上架了，我却生了一场大病，照成了一个月多不能更新，书也就太监了，后来我开了这本书，归根结底是在那本书的基础上写的。
<br/>　　不过故事剧情其实是完全不一样的，我只不过沿用了几条蛇，跟一个绝世蛇种的故事。
<br/>　　因为绝世绝种的故事在那本书里就写了一个开头，所以我想把这个故事完整的写完。
<br/>　　很多人说我抄袭九头蛇，以为自己发现了什么天大的秘密一样，不断的发帖，不断的说抄袭，弄的我就很无语。
<br/>　　删帖你怪我，不删又没完没了。
<br/>　　就不想想，如果我要抄袭，为什么不抄袭那些火的书，我抄袭一本太监的书干嘛，我又不傻。
<br/>　　大家觉得很相似，因为本来就是一个人写的，书的味道自然是一样的。
<br/>　　而且我早在那本书上发了通告了，我只所以不在这本书里说明，是不想影响大家的观看体验，因为那本书没什么好介绍的了，是太监的一本书，就算大家去看，看完之后，没了，也就没了，不可能有新的章节。
<br/>　　，我可以推荐大家去看我第一本完结的作品，神级蚊子，那本书是完本的。
<br/>　　另外在说一点，不用讨论什么龙强还是蛇厉害，这完全没有任何的意义，在小说的世界上，只看谁是主角，主角的光环胜过一切。
<br/>　　我希望这个帖子发出去之后，能让大家理解，喜欢的话，请多支持一下，要是不喜欢看，也没有关系，小说成千上万，我相信总有一本是你喜欢的。
<br/>
