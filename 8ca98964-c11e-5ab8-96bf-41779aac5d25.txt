　　“两位觉得如何？”
<br/>　　修罗剑王见阿鼻鬼王和冰雪狮王沉默不语，低声开口问道。
<br/>　　他在等待两人的答复，只要两人同意，他们便可立刻前往天王宫，面见帝一魂。
<br/>　　“既然要出手，我们为何不帮助拓跋青云呢？虽然在天赋之上，拓跋青云可能略逊帝一魂，但是并不差多少？”冰雪狮王凝重的道，相比于帮助帝一魂，他更加倾向于帮助拓跋青云。
<br/>　　毕竟，若是杀了帝一魂，不至于和天王宫不死不休，但是，若是杀了拓跋青云那就不一定了。
<br/>　　“此言有理！”阿鼻鬼王点了点头。
<br/>　　修罗剑王闻言，忍不住眉头一皱，这两个家伙，真是妇人之仁。
<br/>　　“看来，你们并不想现在就插手！”修罗剑王叹了口气道。
<br/>　　“不是不想，而是未来不可测！”
<br/>　　阿鼻鬼王摇了摇头，继续道：“首先，若是六魂融合，天尊能否复生，无法肯定；其次，苏莫是个不确定因素。”
<br/>　　“是啊，苏莫和龙族新的祖龙关系极深，祖龙在龙族的地位崇高，未来难以预料！”冰雪狮王沉声说道。
<br/>　　主要是，龙族的实力太强，即便那新的祖龙如今不能掌控整个龙族，但是，只要能动用一部分力量，也非同小可。
<br/>　　“那你们的意思是？”修罗剑王面色凝重的看着两人。
<br/>　　“继续等，看清局势！”阿鼻鬼王郑重的说道，这是目前最稳妥的方法。
<br/>　　修罗剑王闻言，皱眉沉思，事情发展到这个地步，也是他没有想到的。
<br/>　　事实上，天尊大人能不能复生？这三具轮回之身谁才具有让天尊再进一步的可能？他完全不清楚。
<br/>　　若是三人公平竞争，一切自然而然最好，但现在，天王宫、地王宫、人王宫，甚至是龙族都参与其中，事情就复杂了。
<br/>　　“好吧，再等一段时间！”少倾之后，修罗剑王叹了口气，再等一段时间也好。
<br/>　　若是帝一魂或者拓跋青云，有人能率先突破，那也不需要他们再出手了，若是苏莫能表现出一点优势，他们也能更好的决定帮谁。
<br/>　　“嗯！”阿鼻鬼王和冰雪狮王均是点了点头，这样再好不过了。
<br/>　　……
<br/>　　天王宫。
<br/>　　大殿之中，天君河端坐在主座之上。
<br/>　　下方，帝一魂、天尘宇、大长老天罡雷以及数名神王级别的强者就坐。
<br/>　　“诸位，现在苏莫的所在，已经众所周知了，你们觉得还如何应对？”天君河环视众人，朗声问道。在坐之人，均是天王宫的核心，他没有任何的忌讳。
<br/>　　“宫主，我们需要立刻派人，将龙都城完全监视起来，只要苏莫敢踏出龙都城，就是他的末日。”一位黄衣神王强者道。
<br/>　　“这有些不妥，监视龙都城，定然无法瞒过龙族之人，一旦被发现，没有活路！”体型壮硕的大长老天罡雷摇头道。
<br/>　　“无需靠近龙都城，只要小心谨慎，不会有失！”黄衣神王又道。
<br/>　　天君河闻言，面带沉吟之色，随即看向帝一魂，道：“一魂，你的意思呢？”
<br/>　　“三管齐下！”帝一魂一字一顿的说道。
<br/>　　“如何三管齐下？”天君河疑惑的问道，众人均是盯着帝一魂，等待对方讲解。
<br/>　　“第一，派人分散在龙都城附近的城池之中，监视龙都城外出之人，无需太多人手，三五人即可，修为不要太高，免得引起注意；第二，收买龙族的普通族人，或者是龙族的奴仆；第三，安排我见吞虚龙帝。”帝一魂面色冷漠，讲述了自己的安排。
<br/>　　众人闻言，沉默了一番，均是暗暗的点了点头。
<br/>　　这样倒是比较妥当，目前也没有更好的办法了。
<br/>　　“你是想要拉拢吞虚龙帝？”天尘宇向帝一魂问道。
<br/>　　“不错，据你所言，吞虚龙帝并没有保护苏莫之心，看来，那就说明吞虚龙帝和龙腾并未一条心，这是一个机会。”帝一魂沉声道。
<br/>　　“机会怕是很渺茫，苏莫和龙腾关系匪浅，吞虚龙帝不至于为了你，而和龙腾撕破脸皮！”天君河摇了摇头。
<br/>　　“事在人为！”帝一魂道，他也知道机会不大，但是，总要去争取一番。
<br/>　　“嗯，老夫来安排！”天君河点了点头，此事还需要背着地王宫，若是被地王宫发现，则会再生事端。
<br/>　　……
<br/>　　在天王宫商议如何对付苏莫之时，地王宫也在进行着同样的一幕，而商议出的结果，基本相差不多。
<br/>　　不仅是地王宫，终极神王殿，也同样在商议。
<br/>　　只不过，终极神王殿，并不是要擒拿苏莫，他们的目的，是取回终极神剑和终极剑道的秘密。
<br/>　　“岂有此理！”大殿之中，终极神王怒发冲冠，眸中留露出浓烈的杀机，这苏莫居然想不归还终极神剑和终极剑道的秘密。
<br/>　　“殿主，苏莫分明是要将终极神剑和终极剑道据为己有！”紫霄沉声说道。
<br/>　　“看来，本座要亲自和他见一面了！”
<br/>　　少倾，终极神王深深的吸了口气，压下心中的愤怒，他需要和苏莫当面相谈。
<br/>　　因为，现在苏莫有龙族庇护，他们终极神王殿惹不起，只能尽量和解。
<br/>　　相比于终极神剑，他更在乎的是终极剑道，而终极剑道的秘密，到底是什么，没有人清楚。
<br/>　　若是，终极剑道可以共享，他还是有希望从苏莫身上，取回终极剑道的。
<br/>　　只要能得到终极剑道，终极神剑不要也罢。
<br/>　　“殿主，见他也无用，我们应该联合天王宫、雷焱神王殿等势力，想办法擒拿苏莫，才能夺回一切！”紫霄沉声道，他并不希望终极神王和苏莫相见，不然，他以前做的那些事，很可能都要暴露了。
<br/>　　“殿主，紫霄之言有理！”
<br/>　　一旁的毁灭剑王附和道：“苏莫得到了终极神剑和终极剑道，就逃离了终极剑域，明显就是要独占终极剑道和终极神剑，您去见他也无济于事。”
<br/>　　“不管怎样，龙都城本座要亲自去一趟！”终极神王叹了口气，若是无法让苏莫交出终极剑道，他只能从吞虚龙帝着手了。
<br/>　　言罢，终极神王略一沉吟，道：“事不宜迟，你们坐镇终极神王城，本座区区就来！”
<br/>　　话音一落，终极神王闪身而出，瞬间离开了大殿。
<br/>　　紫霄见此，面色一变，他的事情怕是要彻底败露了，终极神王殿是不能再待了。
