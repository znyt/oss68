        脑海里一想到是这个男人杀了傅夜沉，童筱颖下意识地闭上了眼睛，扣动了手枪上的扳机。
<br/>
<br/>        结果，什么事也没发生。
<br/>
<br/>        四爷淡定地绕到童筱颖的身后，伸手将童筱颖的身子圈在自己胸前，大手则握住了她手里的手枪，指向了已经放满了水的浴缸：“宝贝，还是我来教你怎么开枪吧！”
<br/>
<br/>        “嘭”的一声闷响。
<br/>
<br/>        童筱颖完全没反应过来，下意识地睁开双眼之际，只见一发子弹便直接打进了浴缸的水中。
<br/>
<br/>        “怎么样？”男人温柔的气息，扑在了她的耳垂上，“学会了吗？”
<br/>
<br/>        童筱颖顿时吓得脸色铁青，浑身不停地微微发颤。
<br/>
<br/>        想必下一秒，他便会开枪把她给杀了吧！
<br/>
<br/>        一想到这里，童筱颖抱着视死如归的心态，咬着下唇，再次闭上了眼睛。
<br/>
<br/>        却没料到，唇上突然传来温柔的触感，令她惊慌失措地睁开了双眼。
<br/>
<br/>        瞳孔里，傅夜沉的俊脸，清晰得不能再清晰。唇齿间，是他那令她熟悉得不能再熟悉的气息。
<br/>
<br/>        童筱颖只觉鼻子一酸，豆大的眼泪在此刻犹如断了线的珍珠，一颗颗“啪嗒啪嗒”地从眼角滚落。
<br/>
<br/>        当他放开她的时候，他随手摘去了她脸上的那副白羽假面眼镜，并轻轻地替她擦拭着糊了一脸的泪水。
<br/>
<br/>        “大哥，我以为你……”童筱颖欲言又止，声音有些哽咽得说不出话来。
<br/>
<br/>        傅夜沉轻轻地刮了一下她的鼻梁，薄唇微扬：“你还真是大胆又犯傻，每一次都差点坏了我们的计划。”
<br/>
<br/>        “每一次？！”童筱颖难以置信地吸了吸鼻子。
<br/>
<br/>        “你就不能老老实实、安安分分地给我待在家里吗？非得跑出来瞎折腾。”傅夜沉无奈地笑了笑。
<br/>
<br/>        童筱颖眨巴着眼睛，不明所以然地看着傅夜沉。
<br/>
<br/>        从他突然找裴金澈替他接管傅氏集团，他带着团队去法国开始，她就感觉到他的不对劲了，只是当时她还理所当然地误以为他是在跟她置气。
<br/>
<br/>        “现在，你是不是应该把你所知道的，都告诉我了？”傅夜沉接着说道。
<br/>
<br/>        童筱颖却答非所问：“为什么我开枪没反应？”
<br/>
<br/>        傅夜沉被她这句反问的话，问得差点哑然失笑。
<br/>
<br/>        她以为这真枪是玩具手枪吗？
<br/>
<br/>        只要扣动扳机就能发射子弹？
<br/>
<br/>        保险栓都没落，还想对人开枪！
<br/>
<br/>        他的这小丫头，真的是蠢萌蠢萌的。
<br/>
<br/>        “你应该感到庆幸，不然你会落得个谋杀亲夫的罪名。”傅夜沉笑道。
<br/>
<br/>        童筱颖见他不肯告诉自己，只好转开话题问道：“你先回答我的问题，我再告诉你，我所知道的那些事情。”
<br/>
<br/>        “好。”傅夜沉点了点头。
<br/>
<br/>        童筱颖接着问：“我想知道，你‘诈死’的前因后果。还有，你怎么就成了夜惑里的四爷？真正的四爷呢？你会不会有危险？你们到底在查什么？”
<br/>
<br/>        “我以后再告诉你这些‘为什么’的答案。只是现在，你得给我老老实实地回家去待着，好吗？”傅夜沉轻轻地抚摸着童筱颖的脸颊，温柔地哄道，“筱颖，你是我最大的软肋。我一直在想尽办法，掩盖你对我的重要性。他们不知道你的存在，所以，你现在是安全的。但是，我没法一直保证你的安全。所以……”
<br/>
<br/>        “所以，你才会突然对我下狠心，又突然对我很心软，是吗？”童筱颖哽咽道。
<br/>
<br/>        是他将她拉到他的世界里来的，结果当她真正爱上他的时候，他却一次又一次地推开她。
<br/>
<br/>        “因为，我放不下你，害怕失去你，心里一直在左右徘徊。”他的语气忽然深沉。
<br/>
<br/>        最初，她再次出现在他的世界里时，他们的计划还未正式实施。
<br/>
<br/>        他以为，他可以做到一边跟她结婚生子，一边把自己的任务完成，直到后来，连华生提醒他，找来夏玄炘当幌子的时候，他才真正的意识到，筱颖的存在是对他最致命的威胁。
<br/>
<br/>        他每一次借机跟她大吵大闹，将她从自己身边赶走，却没料到她总是做出一些令他措手不及的反应，让他心软且舍不得对她再放手。
<br/>
<br/>        或许，他有些自私。
<br/>
<br/>        明明自己正处于危险期，却不想因为再次错过她，而不顾一切地又强行将她绑在自己身边。
<br/>
<br/>        “所以，大哥是打算让我当你圈养在笼中的金丝雀吗？”童筱颖一脸委屈地质问。
<br/>
<br/>        一开始，他就把她“抓”来，“圈养”在他为她准备的“金笼”里，拿最贵的食物“喂养”她，让她在他的金笼里，只为他一人欢呼雀跃。
