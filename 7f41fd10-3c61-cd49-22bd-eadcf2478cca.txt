        卡片上是手写的钢笔字，字迹翩若惊鸿，很漂亮，看着特舒心。
<br/>
<br/>        童筱颖匍匐在大床上，下巴枕在左手背上，右手拿起卡片，看着卡片上写的内容，嘴角情不自禁地微微上扬。
<br/>
<br/>        送给我最美丽的小天鹅。
<br/>
<br/>        这行字的右下角，还有一只可爱呆萌，咧着嘴，露着笑脸的卡通青蛙简笔画。
<br/>
<br/>        而这只小青蛙画得不是很流畅，应该是第一次画时，行笔有些生疏，却不影响它的憨态。
<br/>
<br/>        一睁开眼睛就能遇上这个小小的惊喜，让童筱颖的心情十分愉悦。
<br/>
<br/>        她看得出，大哥是有心了。
<br/>
<br/>        而且，他是非常用心地在哄她开心。
<br/>
<br/>        这天早上，他又迟到了！
<br/>
<br/>        其实，他的迟到，是于她落以柔而言的。
<br/>
<br/>        平日里，傅夜沉都会在七点五十五分准时进办公室，但今天，他没有。
<br/>
<br/>        傅夜沉到办公室里的时候，已经九点半了。
<br/>
<br/>        落以柔不得不重新给他泡了一杯红茶过来。
<br/>
<br/>        茶杯触碰桌面的那一刻，她一双柔情似水的眼睛，不禁偷偷地瞅着略低头看资料的傅夜沉。
<br/>
<br/>        他的侧面，脸部轮廓棱角分明，鼻梁英挺，唇型特别好看。一袭西装笔挺，成熟俊朗，散发一股温文儒雅的男人韵味。
<br/>
<br/>        每天，她都被他的帅气，撩得心花怒放。
<br/>
<br/>        此刻，他就坐在办公桌前，于她而言，明明近在咫尺，却莫名其妙地遥不可及。
<br/>
<br/>        落以柔到现在都不敢置信，她心里偷偷爱慕的他，居然已经结婚了！
<br/>
<br/>        不，他一定还没结婚，瞧，他的手上都还未戴结婚戒指，不是吗？
<br/>
<br/>        就算他真的要结婚了，他的婚礼，也一定会像上次那样，全程交由她去办，不是吗？
<br/>
<br/>        落以柔努力自我安慰着。
<br/>
<br/>        一定是昨天那个女人耍了什么手段，逼迫大哥娶她为妻！
<br/>
<br/>        傅夜沉那修长的指间，握着白金钢笔，笔尖朝她刚整理给他的资料上划下了两三行线，低沉的嗓音，缓缓地说道：“这些再拿回去改改。”
<br/>
<br/>        落以柔猛然回过神来，弱弱地应了声：“好。”
<br/>
<br/>        接着，她从他手中拿回了资料，转身离开了总裁办。
<br/>
<br/>        这段时间，郑忠威替他请来了第三方财务审计公司，来查财务部的账，一查就查出了许多大问题。
<br/>
<br/>        昨晚，傅夜沉赶回公司时，得知以财务部部长为首，部门内三个组长让员工私下替他们做阴阳账本，私吞公款，他一气之下，将这批人，全部开除并交由律师处理。
<br/>
<br/>        幸好，郑忠威替他提前做好了准备，财务部虽然被开除了一半的人员，但余下的工作任务由郑忠威请来的第三方会计公司替他们临时填补空缺。
<br/>
<br/>        郑忠威还在夸，这些都是落以柔的功劳，若不是她心思细腻，看账本看得仔细，还真发现不了这些看似很小却是很大的问题。
<br/>
<br/>        这次，财务部人员大变动，连部长都被抓了，倒是让其他部门的几个高层大快人心，毕竟，他们每次找财务部申请经费的时候，财务部那部长没少给他们什么好脸色。
<br/>
<br/>        除了郑忠威“除奸”有功以外，落以柔的名声在公司里也传得沸沸扬扬。
<br/>
<br/>        一个刚毕业的大学生，能有这般优秀的作为，属实大家学习的榜样。
<br/>
<br/>        从此，落以柔在公司里，另有了一个新的称呼，被大家亲昵地唤做“落姐”。
<br/>
<br/>        这一上午，傅夜沉在公司里忙得不可开交，同样，童筱颖在家里也忙得不可开交。
<br/>
<br/>        昨天傅夜沉让高档内衣店，一次性给送来了一百款不同样的内衣内裤供她一一挑选，她看得眼花缭乱，只留下其中的十款。
<br/>
<br/>        这不刚把衣服整理进衣帽间，又有服装设计师上门来，说是要给她量身定制婚纱、新娘凤袍和敬酒礼服。
<br/>
<br/>        不过，值得庆幸的是，刘婶在得知她真的是童筱颖后，差点喜极而泣。
<br/>
<br/>        刘婶安慰她说：“二少爷不懂得好好珍惜你，是二少爷没有这个福气！”
<br/>
<br/>        有刘婶在家，童筱颖在这个家里并不感到孤单。
<br/>
<br/>        送走服装设计师后，童筱颖去了厨房，见刘婶正在给傅夜沉炖汤，于是主动上前去搭把手。
<br/>
<br/>        对于她童筱颖先嫁给了二少爷，离了婚，又嫁给大少爷这事，刘婶倒是一点想法也没有，纯粹地只是替大少爷傅夜沉感到欣慰。
<br/>
<br/>        “大少爷说，他今天中午要加班，不能回来吃饭了。我捉摸着他，肯定又不会吃午饭了。他经常是这样，每次公司里一忙，中午就没空去吃午饭，他就索性不吃了。”刘婶念叨道。
<br/>
<br/>        童筱颖忍不住好奇地问：“那您是怎么知道他中午忙起来就不会去吃午饭？”
