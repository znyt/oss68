　　第344章
<br/>　　姓名：和珅（初出茅庐的绝世文官）
<br/>　　年龄：32
<br/>　　技能：商业精通，相人，过目不忘
<br/>　　隐藏属性：贪徒
<br/>　　啧啧……程大雷在心底惊叹，过不愧一出场就是满级的人物，自带三个技能，这还是第一次见到。
<br/>　　这是程大雷第一次抽到绝世人物，看来随着山寨升级，自己能抽到高级人物概率也越来越高了。
<br/>　　据和珅自己所说，他幼时家里薄有产业，后来家道中落，考过科举只是没考上，晃悠着就到而立之年。听说徐神机现在本事很大，所以特来投奔。
<br/>　　这可不是因为混不下去来投奔，而是因为混得不好，想要混得更好才来的，果不愧是个贪徒呐。他贪的是什么，名，利，女人，或者是天下。
<br/>　　要是能用的话肯定是个人才，恰恰是程大雷目前最缺的那种人才，可这样一个人真的能用么？
<br/>　　坐在大厅里，双方大眼瞪小眼，互相打量着彼此。程大雷是因为有系统加持所以看得穿和珅，可和珅有相人之术，未必看不穿程大雷。
<br/>　　不得不说，和珅当真好相貌，不过分显眼，却也不丑，只是让人看到后感觉很舒服。也没有一上来就给程大雷一阵乱拍……当然，人人都能看得出来的奸臣，还算奸臣么。
<br/>　　程大雷沉默，他同时也沉默着，只有徐神机还在一旁不停说着和珅的好话。
<br/>　　“和兄弟可是不远万里从幽州来的，嘿，没有这老交情，他能受这罪。”
<br/>　　“他的本事十里八乡都是服的，不信大当家可以去打听打听，那可是我们那里有名的明白人。”
<br/>　　“知道啦，知道啦，再明白能有你明白么。”程大雷。
<br/>　　“这么说倒也是。”徐神机。
<br/>　　“好啦，军师你先出去转转，让我与和先生单独谈谈。”
<br/>　　徐神机离去后，大厅内只剩下程大雷与和珅二人。程大雷越看越觉得对方可靠，越看越舍不得杀掉。
<br/>　　程大雷忙提醒自己，可是不能被对方外表蛊惑，这都是表象呐。
<br/>　　“和先生读圣贤书，行仁义事，讲究学得文武艺，货与帝王家，可我们做的是山贼行当，死后连祖坟都入不了的。这样怕是会玷污先生的理想。”
<br/>　　“大当家这话，和某无法认同。”
<br/>　　呦，还颇有点直言不讳的意思。程大雷心中一动，那就让你尽情说，说完我随便找个理由说你说得不对，提起斧头把你杀了。
<br/>　　“有话请讲？”
<br/>　　“上有道，民奉公守法，上无道，民揭竿而起，大当家觉得，如今的帝国有道么？”
<br/>　　“这……”
<br/>　　“若说贼，无非是成者王侯败者寇而已，大武立国之初何尝不是天下之贼。如今他们失了公道，天下英雄当共逐之。依和某看来，习得文武艺，不该卖于帝王，而是该货于天下英雄。”和珅大手一指：“以我观之，大当家便是天下英雄。”
<br/>　　“喔，何解？”
<br/>　　“和某粗通相人之术，斗胆于大当家观一观面相。”
<br/>　　“有话直说？”
<br/>　　“人生异相，必怀异能，大当家天生一副好相貌，如今虽隐于草莽，但已露麟角之相。”和珅道：“独战戎族为武，斧劈杨龙停为义，千里出关为情，刺杀草原王为勇，如此一有情有意，有武有勇之人，大当家不是英雄，天下便再无英雄。”
<br/>　　“呃……这个……哈哈。”
<br/>　　程大雷像大夏天吃了碗冰淇淋一样痛快，忍不住开怀大笑。看人家这马屁拍得，让你完全感觉不出来对方在谄媚，还以为对方说得句句是真理。
<br/>　　不过话说回来，自己做的事和珅是怎么知道的。啧啧，果然是人才呐，求职之前都知道做背景调查。没有傻乎乎愣头青一样拿着份简历就去求职。
<br/>　　程大雷越看越顺眼，都有些不舍得杀了。
<br/>　　“嗯呐，你拍马屁的本事我见识到了，可你其他方面的本事呢，我这里可是不留庸才。”
<br/>　　“和某句句实话，不敢有一字诓骗大当家。”
<br/>　　呵，装，你继续给我装。
<br/>　　“我需要考一考你，如果你能通过，就可以留下。”
<br/>　　“怎么考？”
<br/>　　“你随我来。”
<br/>　　程大雷把和珅引到外厅，又请司马星、刘悲、吴用过来。
<br/>　　这三人负责蛤蟆寨的后勤，每天都会遇到琐事，这些事情自然就递到程大雷面前。有些程大雷也不知该如何完美解决，只能选择一个折中的法子。
<br/>　　今天程大雷坐在帅案后，和珅垂手立在另一侧，当真是没有一点不规矩的地方，充分让程大雷感受到了被尊重的感觉。
<br/>　　“来来来，今天本寨主处理公务，你们有什么麻烦事，尽管禀报。”
<br/>　　“大当家，西颜部订一批高粱过冬，用兽皮结账，可现在冬天粮价太贵，兽皮又一时不好出手，这单生意咱们还要不要做？”司徒星。
<br/>　　“粮价太贵，可以去抢啊……”
<br/>　　程大雷说了一句，忽然想起身边的和珅，想了想道：“和先生，这件事你看给如何处理，西颜部是我们的老客户，他们的生意可不能丢？”
<br/>　　“戎族要粮无非是过冬而已，一些大户家里都会有积年的陈粮，这些粮食价格便宜，但填饱肚子是不成问题的。而兽皮是越往东走，价格越贵，靠近长安的地方是有人愿意以高价收购的。”
<br/>　　“啧……”程大雷不得不赞一声。
<br/>　　刘悲，吴用，司徒星将遇到的麻烦一件件说了，不管怎样杂乱的事情，和珅都能一语中的。而有些特别麻烦，牵扯关系特别多的事情，他也需要看一看才能给出解决办法。
<br/>　　和珅一个下午处理的事情，比程大雷忙三天做的都多。程大雷当真有些舍不得眼前这贪官了，这人用人也行，管理也行，除了不会领兵打仗以外，几乎没有弱点。
<br/>　　蛤蟆寨的后勤如果交给他处理，程大雷就可以无事一身轻了，节省下时间，与苏樱逗逗闷子，吃吃柳芷小桃的胭脂不好么。
<br/>　　“大当家，我算过关了么？”和珅问。
<br/>　　“嗯，还差不多吧，和我比是差点，你今天做的事情，我一个时辰就可以办完。”
<br/>　　“我自然是不能和大当家相提并论。”
<br/>　　
<br/>(本章完)
