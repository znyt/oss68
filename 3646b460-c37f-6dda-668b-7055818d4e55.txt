　　当——
<br/>　　正在此时，一声巨大的钟响，将正沉浸在悟道之中的元始惊醒，之前的种种，顿时烟消云散，仿佛从来没有出现过。
<br/>　　“混蛋！竟敢打断我大哥悟道！”
<br/>　　这时，旁边的通天直脾气又上来了，直接拔出青萍剑，想要上去做上一场。
<br/>　　“三弟，此地是紫霄宫，不可鲁莽。”这时老子连忙劝道。
<br/>　　“混蛋，扁毛雀，这里要不是紫霄宫，看我不弄死你。”通天恨声道。
<br/>　　“来啊，谁怕谁！”一手持着混沌大钟，太一嚣张道。之前看见有人在紫霄宫前悟道，他顿时不爽，用混沌钟将之震醒。
<br/>　　“二弟，不要乱说话。”在他身前，一个身穿锦袍的男子威严地骂道，之后又对着元始三人道：
<br/>　　“三位，之前是我家二弟不小心没有控制好钟声，若是有所损害，我在这里替我家二弟向你们道歉。抱歉了。”
<br/>　　“大哥，我们怕他们干什么？你——”太一大怒，吼叫道。
<br/>　　可帝俊却在赔礼之后，拉着太一，走向另一个地方坐下，太一满脸愤怒之色，却被帝俊拉着，很是不甘的坐下。
<br/>　　“哼！”通天冷哼一声，毫不理会。
<br/>　　这时，周围已经来了好多人，有女娲与伏羲，后土与玄冥，镇元子与红云老祖，冥河与鲲鹏老祖，太一与帝俊等几人，周围人此刻都在静静地看笑话。
<br/>　　有人做出头鸟，打断他人悟道，他们此时乐见其成，人是自私的，谁也不希望有人能够走在他们前面。
<br/>　　就在此时，元始突然睁开双眼，眼中两道黑白玄光毫不掩饰地直直射向帝俊与太一。玄光速度之快，两人都还没有反应过来，眼见就快要被射中，两人急忙闪身，想要躲避。
<br/>　　可是，玄光速度太快了，使得两人都没有完全躲过去。
<br/>　　黑白玄光带有浓烈的死亡气息与毁灭气息，直直地穿透两人的肩膀，浓郁的死亡法则在他们身体之中破坏冲撞，毁灭他们的肉身。
<br/>　　“噗嗤——”
<br/>　　两人顿时倒飞出去，口中直直喷出一口黑血，变得虚弱不堪，模样极为狼狈。
<br/>　　“混蛋！元始，你欺人太甚！”
<br/>　　两人虚脱的走进宫门前，对着元始怒声道，就想要动手。
<br/>　　“不好意思，本座之前参悟大神通没有控制好，若是两位有所损害，那就不好意思，本座也不会道歉。二位想要如何，本座接了！”
<br/>　　元始淡淡的声音在众人耳边响起。
<br/>　　“嘶——”
<br/>　　周围几人都倒吸一口气，心中不仅在为元始的修为而心惊，更是在为元始的大胆出手的忌惮。这人无法无天，行事霸道，之前没有人敢相信有人在紫霄宫前争斗，但眼前这人就敢，这不是肆无忌惮又是如何？
<br/>　　紫霄宫是什么地方？那可是圣人的到场啊！此人竟敢在此动手，真是大胆！
<br/>　　帝俊太一两人之所以会被元始击中，除了有修为不如元始的原因之外，还有就是之前两人根本不相信他敢在紫霄宫前出手。但事实证明，他们的想法是错的，元始不仅出手了，还狠狠地出手，将他们重创。
<br/>　　“我们走着瞧，此事不算完。”帝俊大声道。
<br/>　　“本座随时恭候。到时候想找本座麻烦可别忘了带上一副棺材，本座可不会替你收尸。”
<br/>　　“哼！”
<br/>　　之后，各自找了一个地方坐下，帝俊与太一需要疗伤，元始眯着眼睛，淡淡地看了一眼周围众人，众人只感觉自己被一只猛兽盯上了，心中发寒。
<br/>　　元始只是随意一瞟，便不再多管，而是随意找了个地方，盘腿坐下，之后闭目养神，静等紫霄宫开。
<br/>　　悟道被打断，要说不生气，那是不可能的，更何况当时元始已经到了最后的阶段，若是再给他一点时间，说不定他能够一次明悟自己的混元大道，就此突破混元。此次悟道被打断，他当时简直有一种想要当场将二人灭杀的冲动。
<br/>　　阻人道者，乃不共戴天之仇！
<br/>　　不过，此时停下来仔细想想，这一切仿佛冥冥之中早有注定，罢了，罢了，既然自己此时机缘不到，亦不可强求，大道方向已经找到，他就不相信今后不能突破。
<br/>　　“后土、玄冥见过三位道友。敢问三位道友可是昆仑三清？”
<br/>　　这时，黄色衣裙女子与黑色衣裙女子不理其他人，来到元始这边，微微见礼，声音轻柔的开口问道。
<br/>　　元始睁开眼睛，淡淡地打量了对方一眼，两人都十分漂亮，端庄大方，姿**人，乃洪荒顶级的美女，让人看了赏心悦目。看过一眼后，他开口说道：
<br/>　　“不错，吾三兄弟便是三清，本座玉清元始，这两位是吾二弟太清老子与三弟上清通天。若是本座没有猜错的话，两位想必就是十二祖巫中的后土与玄冥道友了。”
<br/>　　“不错，我就是后土，她是玄冥。”黄色衣裙女子，也就是后土指着黑色衣裙女子对元始介绍道。
<br/>　　“嗯。”元始淡淡点了点头。
<br/>　　场面一下子寂静下来，良久，后土略带犹豫之色地向元始问道：
<br/>　　“不知道兄有没有察觉到，我们之间存在某种联系，为何我当初一看到道兄，就会给我一种熟悉感？”
<br/>　　“我也有这种感觉。”这时，通天开口插话道。
<br/>　　“你们俩身上，盘古血脉的气息很浓。”这时，老子惊讶地看着两女。
<br/>　　“嗯，吾等三人乃盘古元神清气所化，你们乃盘古精血融合先天地脉煞气所化，两者本同源而不同形，若是有联系，那就只可能是盘古了。”元始淡淡道，这事早晚会天下皆知，此时说出来又有何妨。
<br/>　　“原来如此。”
<br/>　　后土点了点头，表示自己知道。
<br/>　　之后，两者都不知道要说些什么，众人心思各异，不知道想些什么。想要拉近关系吧，这样有显得太过刻意了，想要就此走开吧，这样又不太好。就这样，后土与玄冥在元始三人身边坐下，还时不时的与他们交谈起来。
<br/>　　元始知道，此事不会那么简单，就算后土与玄冥不说，日后其他祖巫也会来找他们分一分，谁为盘古正宗，到时候，场面可就不会这么和谐了。
<br/>　　但元始并没有多想，十二祖巫来了又如何，大不了干上一场。
<br/>　　眨眼间，九十九年时间转瞬即逝，距离鸿钧讲道却是只剩下一年时间了。
<br/>　　“哐啷——！”
<br/>　　紫霄宫大门开了，所有人神情一动，扭头望去时，却看见一对金童玉女，走了出来，显然，他们扮演的是类似看门童子的角色。但这两个童子的修为，却让在场不少人眼皮狂跳，一个个瞳孔收缩，露出一丝惊骇，大罗金仙初期？
<br/>　　紫霄宫内，两个看门的童子，居然也有大罗金仙初期的修为吗？那么他们的主人又有多厉害？
<br/>　　这一刻，没人敢小瞧紫霄宫的鸿钧道人。
<br/>　　元始睁开双眼，站了起来，看向两个童子，眼神淡漠，心中却暗暗道：这就是日后的天帝与王母吗？果然不简单。
<br/>　　这时候，只见那童子望着宫门前的众人，大声开口道：
<br/>　　“诸位道友，百年时间将至，老师即将开讲大道，紫霄宫内，不得喧哗，尔等速速进来，不可怠慢！”
