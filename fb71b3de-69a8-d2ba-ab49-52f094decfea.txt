　　　　在众人努力之下，镜面前端汇集出一道粗壮的灵力波，将帅旗包裹起来，在灵力网中艰难地前进。
<br/>　　起初旗杆穿行的速度还是很快的，然而越往灵力网的深处去，越是感到重重的压力，就算有钟馗主导，所有人一起努力，还是愈发感到吃力。
<br/>　　毕竟，这是集合了七座山峰吸收了至少几百年的灵力形成的阵法。
<br/>　　“怎么可能！”
<br/>　　摇光仙子站在悬崖边上，看到下方的一幕，喃喃自语了一声，作为星宿海的宗主，她对自己这北斗七星阵的强度再熟悉不过，就算是自己这样的阐教金仙进去，也只有粉身碎骨一个下场。钟馗居然能将法器送进去……他的修为，果然已经进入化境。
<br/>　　“掌教，可发信号，让所有弟子一起上，阻止他们！”一个观主凑在摇光仙子耳边说道。
<br/>　　摇光仙子抬起玉手，一道灵光在掌心飞出，落在空中，如同烟花一般炸开，幻化出一道符文，是星宿海七峰集结令，落在山谷之中。
<br/>　　没用一时片刻，从其余六座山峰上不断涌现出人影，至少有数百个，从四面八方朝山谷中涌去。
<br/>　　这些都是星宿海最普通的弟子。星宿海在空界有千年基业，门下弟子无数，之所以没让这些普通弟子参与守山，是因为他们实力太弱，更加容易丧命。眼下看到钟馗强行闯阵，要救叶少阳出去，无奈之下派出了他们。
<br/>　　她心里也是有盘算的，钟馗跟道风不一样，钟馗虽然一向率性狂悖率，毕竟是身居高位的阴神，在没有阴司正式命令之下，他应该不至于会屠杀青冥界的法师，假如他真这么做了，那更是落人口实。摇光仙子甚至期待他杀几个自己门下的弟子……
<br/>　　这时候山顶上已经陷入混战，摇光仙子抽身而退，看了一眼黎山老母和道风，这两人是混战的核心，战斗已经到了疯狂的地步，这种程度的战斗，连她看上去都有点触目惊心的感觉。
<br/>　　不愧是转世鬼童，摇光仙子心想，师姐……多少年没遇到过这么强大的对手了？
<br/>　　望着四面八方用来的星宿海弟子，萧逸云等人不禁皱起眉头，倒不是怕，大致扫了一眼之后，就大致确定了这些道士的实力，就算他们一起上，也没什么大不了。
<br/>　　他们只是担心分心，毕竟对方人太多了，现在所有人都集中精力推动帅旗，哪怕一丝工夫都抽不出来。
<br/>　　“你们继续，什么都不要管！这群杂鱼！”钟馗深吸一口气，身体猛然暴涨了几十倍，看上去有半座山那么高，对着最先一批靠近的道士，直接用袖子扇了过去，带起了一阵阴风，将这些道士直接吹到了山那边去。
<br/>　　“杂鱼。”钟馗拿出酒葫芦，喝了一口，对着另一边山上冲下来的一群道士喷过去，一口酒液如同雨雾一般，将这些道士扫飞。
<br/>　　面对之后赶到的几波道士，钟馗直接把手探过去，随手捞了一把，每一把都捞起一堆，像是捡了一堆石子在手里，直接朝对面山坡扔过去，至于落下来之后怎么样，看也不看一眼。
<br/>　　“哈哈哈。”钟馗一边喝酒，一边狂笑起来，笑声中带着几分邪气，和一种睥睨天下的气势，巨大的身躯和狰狞的五官，使他看上去像是一尊魔神。
<br/>　　钟馗的恶名，在整个鬼域都是一等一的，在战场上抓到太阴山那边的恶鬼厉妖，一般都是抠眼挖心，生吞活剥，残暴至极。为此阴司几番追究，钟馗都懒得管，依旧我行我素，不过他也明白，眼前这些道士是杀不得的，把他们赶走便是，至于中间是不是少数几个捏死的撞死的，那就管不着了。
<br/>　　“钟天师，顶不住了……”萧逸云咬紧牙关说道。众人一起作法，旗杆已经完全没入灵力网中，不过在离阵心还有一段距离的时候，被阵法强大的压力挡住了，众人咬牙坚持，几乎用尽全力，也只能维持旗杆不被搅碎，悬浮在空中，剧烈地抖动着。
<br/>　　“你们天子殿的，都是一群废物！”钟馗身体缩回到正常大小，双手作法，加强着镜面符文的灵力，旗杆再度艰难挺进。
<br/>　　在几股灵力之中，钟馗感受到了一股妖气，转头看到了在一旁默默出力的扁头，喝道：“哪来的孽畜！”
<br/>　　扁头低头不敢作声。
<br/>　　老郭急忙上前，哆哆嗦嗦地解释了扁头的来历。
<br/>　　钟馗听说是叶少阳电话他渡的劫，也不说什么了，继续带领大伙推动旗杆。旗杆在钟馗强大的法力驱动下，继续往阵心挤压，但是前进的速度越来越慢。
<br/>　　“这阵法……”钟馗深吸一口气，毕竟这北斗七星阵汇聚的七座山峰吸收了千年的灵力，就算是他，也感到有些吃力。
<br/>　　形成推动帅旗前进的灵力波，钟馗一个人独占了七分，其余人合力贡献了三分，尽管如此，除钟馗外，所有人都已经累到几乎虚脱，帅旗缓缓前进，在距离叶少阳足够近的地方，最终还是停止了。
<br/>　　“他娘的！”
<br/>　　钟馗深吸一口气，帅旗距离阵心，只剩下最后一道灵力网，但就像漩涡一样，越是靠中心水流旋转的速度就越快，阵法也是一样。最后的一张灵力网，并非不可摧毁，只是缺口刚产生，周围的切割符立刻聚拢过来，堵住缺口，让帅旗很难更进一步。
<br/>　　“他娘的！”钟馗又骂了一声，灌了一口酒，使出浑身法力，用力推动着帅旗，生生顶入了最后一层灵力网，不断扩大缺口。
<br/>　　山顶上，摇光仙子突然看见这一幕，立刻发生示警：“阻止他们！”
<br/>　　摇光仙子的想法很单纯，只要叶少阳还在阵中，道风就不会走，一旦叶少阳获救，谁也拦不住道风离开。
<br/>　　她身后的丛林里，突然飞出数道人影，是一直潜伏着的苏沫、张云，还有那个黄脸长手的男子。
<br/>　　三人出现之后，立刻飞越众人，朝山谷方向飞去。
<br/>　　道风等人完全陷入混战，即使发现三人出现，也来不及对他们进行阻击。
